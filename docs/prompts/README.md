# 再利用プロンプト（`docs/prompts/`）

人間・AI がそのまま貼り付けて使う指示文です。作業手順の正本は `docs/ai/` にあり、ここは**役割分担と起動方法**に特化しています。

| ファイル | 役割 | 使い方 |
|----------|------|--------|
| [`過去問ノート-orchestrator.md`](過去問ノート-orchestrator.md) | **親エージェント**（オーケストレーション専念） | チャットに貼る、または「このプロンプトに従って」と依頼 |
| [`過去問ノート-worker.md`](過去問ノート-worker.md) | **サブエージェント**（1問作成） | 親が Task 起動時に中身を埋めて渡す（単体では使わない） |

## 依頼の例

```
@docs/prompts/過去問ノート-orchestrator.md

R3春期 午前 問7〜10 を作成して
```
