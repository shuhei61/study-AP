<p class="ap-question"><strong>問61</strong>　経済産業省が取りまとめた"デジタル経営改革のための評価指標(DX推進指標)"によれば，DXを実現する上で基盤となるITシステムの構築に関する指標において，"ITシステムに求められる要素"について経営者が確認すべき事項はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ITシステムの全体設計や協働できるベンダーの選定などを行える人材を育成・確保できているか。</div>
<div class="ap-choice-note">ベンダーに丸投げせず、ITシステムの全体設計や協創できるベンダー選別ができる人材の確保は，"ITシステムに求められる要素"ではなく，"ガバナンス・体制について求められる要素"（<a href="用語/ITガバナンス" class="internal-link" data-href="用語/ITガバナンス">ITガバナンス</a>の観点）の確認事項です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　環境変化に迅速に対応し，求められるデリバリースピードに対応できるITシステムとなっているか</div>
<div class="ap-choice-note">正しい。環境変化への迅速な対応と，求められるデリバリースピードへの対応は，"ITシステムに求められる要素"のひとつ（スピード・アジリティ）として<a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>に規定されています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　データ処理において，リアルタイム性よりも，ビッグデータの蓄積と事後の分析が重視されているか</div>
<div class="ap-choice-note"><a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>では，データをリアルタイム等、使いたい形で使えるITシステムかどうかが問われます。<a href="用語/ビッグデータ" class="internal-link" data-href="用語/ビッグデータ">ビッグデータ</a>の蓄積と事後分析を重視するという記述は，データ活用の趣旨と逆です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　データを迅速に活用するために，全体最適よりも，個別最適を志向したITシステムとなっているか。</div>
<div class="ap-choice-note"><a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>では，部門を超えてデータを活用し全社最適を踏まえたITシステムかどうかが問われます。個別最適を志向するという記述は指標の趣旨と逆です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>DX推進指標（<a href="用語/デジタル経営改革のための評価指標" class="internal-link" data-href="用語/デジタル経営改革のための評価指標">デジタル経営改革のための評価指標</a>）は、<a href="用語/デジタルトランスフォーメーション" class="internal-link" data-href="用語/デジタルトランスフォーメーション">デジタルトランスフォーメーション</a>(DX)の変革を後押しし、ビジネスのアクションに繋がる気付きの機会を提供するものとして経済産業省が策定したものです。項目ごとにレベル0～5の成熟度が設定され、各企業が簡易な自己診断を行えます。</p>
<p>本問の対象である"ITシステムに求められる要素"として、指標では次の3つを規定しています。（1）データ活用 … データをリアルタイム等、使いたい形で使えるITシステムか。（2）スピード・アジリティ … 環境変化に迅速に対応し、求められるデリバリースピードに対応できるITシステムか。（3）全社最適 … 部門を超えてデータを活用し、バリューチェーンワイドで<a href="用語/顧客" class="internal-link" data-href="用語/顧客">顧客</a>視点の価値創出ができるよう、システム間連携などにより全社最適を踏まえたITシステムか。</p>
<p>アはガバナンス・体制の要素、ウはデータ活用の趣旨と逆、エは全社最適の趣旨と逆の記述です。したがって正解はイです。</p>
</div>
