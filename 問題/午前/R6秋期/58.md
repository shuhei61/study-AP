# [令和6年秋期 午前 問58](https://www.ap-siken.com/kakomon/06_aki/q58.html)

#問題 #マネジメント #システム監査

<details class="ap-reveal">

<p class="ap-question"><strong>問58</strong>　システム監査において，監査手続の適用に際して用いられる技法のうち，データの生成から入力，処理，出力，活用までのプロセス，及び組み込まれているコントロールを，書面上で，又は実際に追跡するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　ウォークスルー法</div>
<div class="ap-choice-note">正しい。<a href="用語/ウォークスルー法" class="internal-link" data-href="用語/ウォークスルー法">ウォークスルー法</a>は、データの生成から入力、処理、出力、活用までの<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>、及び組み込まれている<a href="用語/コントロール" class="internal-link" data-href="用語/コントロール">コントロール</a>を、書面上で、または実際に追跡する技法です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　チェックリスト法</div>
<div class="ap-choice-note"><a href="用語/チェックリスト法" class="internal-link" data-href="用語/チェックリスト法">チェックリスト法</a>は、監査対象に応じて調整したチェックリスト（質問書）に対して関係者から回答を求める技法であり、<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>全体を追跡するものではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　突合・照合法</div>
<div class="ap-choice-note"><a href="用語/突合・照合法" class="internal-link" data-href="用語/突合・照合法">突合・照合法</a>は、関連する複数の証拠資料間を突き合わせる、または記録された最終結果を原始資料まで遡って突き合わせる技法であり、<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>追跡の技法ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ドキュメントレビュー法</div>
<div class="ap-choice-note"><a href="用語/ドキュメントレビュー法" class="internal-link" data-href="用語/ドキュメントレビュー法">ドキュメントレビュー法</a>は、監査対象の状況に関する<a href="用語/監査証拠" class="internal-link" data-href="用語/監査証拠">監査証拠</a>を入手するために、関連する資料および文書類を入手し内容を点検する技法であり、<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>追跡の技法ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準(平成30年)では、選択肢の4つの監査技法の他に、<a href="用語/インタビュー法" class="internal-link" data-href="用語/インタビュー法">インタビュー法</a>、<a href="用語/現地調査法" class="internal-link" data-href="用語/現地調査法">現地調査法</a>、<a href="用語/突合・照合法" class="internal-link" data-href="用語/突合・照合法">突合・照合法</a>、コンピュータ支援監査技法、<a href="用語/テストデータ法" class="internal-link" data-href="用語/テストデータ法">テストデータ法</a>などを定義しています。</p>
<p>本問は「データの生成から入力、処理、出力、活用までの<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>、及び組み込まれている<a href="用語/コントロール" class="internal-link" data-href="用語/コントロール">コントロール</a>を、書面上で、又は実際に追跡する」技法を問うており、これに該当するのは<a href="用語/ウォークスルー法" class="internal-link" data-href="用語/ウォークスルー法">ウォークスルー法</a>です。したがって正解は「ア」です。</p>
</div>
</div>
