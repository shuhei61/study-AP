<p class="ap-question"><strong>問56</strong>　JIS Q 20000-1:2020(サービスマネジメントシステム要求事項)によれば，継続的改善に関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　改善活動では，品質と費用の両方について改善の目標を設定しなければならない。</div>
<div class="ap-choice-note">JIS Q 20000-1:2020では、品質、価値、能力、費用、生産性、資源の活用およびリスク低減のうち、1つ以上について改善を設定すると明記されており、品質と費用の両方を必ず設定する必要はありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　改善活動は，提供するサービスを改善の対象にして，サービスマネジメントシステムを改善の対象にしない。</div>
<div class="ap-choice-note">提供するサービスだけでなく、<a href="用語/サービスマネジメントシステム" class="internal-link" data-href="用語/サービスマネジメントシステム">サービスマネジメントシステム</a>（SMS）そのものも改善対象に含まれます。SMSおよびサービスについて、適切性、妥当性、<a href="用語/有効性" class="internal-link" data-href="用語/有効性">有効性</a>を継続的に改善しなければならないと明記されています。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　改善の機会に対して適用する評価基準には，改善とサービスマネジメントの目的との整合性が含まれなければならない。</div>
<div class="ap-choice-note">正しい。組織は改善の機会に対して評価する評価基準を定めるものとされ、その評価基準には、改善と<a href="用語/サービスマネジメント" class="internal-link" data-href="用語/サービスマネジメント">サービスマネジメント</a>の目的との整合性が含まれなければならないと明記されています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　実施された改善は，設定した目標に照らして測定するのではなく，改善前と比較した改善の効果を測定する。</div>
<div class="ap-choice-note">改善の効果は、設定した目標と比較して判断します。設定した目標に照らして実施された改善を測定し、目標が達成されていない場合は、必要な処置をとると明記されています。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/継続的改善" class="internal-link" data-href="用語/継続的改善">継続的改善</a>は、常に変化するビジネスニーズに対応するために、またはサービスの適切性、妥当性、<a href="用語/有効性" class="internal-link" data-href="用語/有効性">有効性</a>などの向上を目的として、ITサービス、<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>、ITインフラストラクチャを常に改善していく<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>です。JIS Q 20000-1:2020では「<a href="用語/パフォーマンス" class="internal-link" data-href="用語/パフォーマンス">パフォーマンス</a>を向上するために繰り返し行われる活動」と定義されています。他の<a href="用語/マネジメントシステム" class="internal-link" data-href="用語/マネジメントシステム">マネジメントシステム</a>と同じく、<a href="用語/PDCA" class="internal-link" data-href="用語/PDCA">PDCA</a>サイクルを通じてサービス等の<a href="用語/パフォーマンス" class="internal-link" data-href="用語/パフォーマンス">パフォーマンス</a>を継続的に向上させます。</p>
<p>JIS Q 20000-1:2020における、<a href="用語/継続的改善" class="internal-link" data-href="用語/継続的改善">継続的改善</a>に対する要求事項は次のとおりです。</p>
<ul>
<li>品質、価値、能力、費用、生産性、資源の活用及びリスク低減のうち、一つ以上について改善の目標を設定する</li>
<li>改善が優先付けされ、計画され、実施されることを確実にする必要な場合には、SMSを変更する</li>
<li>設定した目標に照らして実施された改善を測定し、目標が達成されていない場合は、必要な処置をとる</li>
<li>実施した改善を報告する</li>
</ul>
<p>したがって正解は「ウ」です。</p>
</div>
