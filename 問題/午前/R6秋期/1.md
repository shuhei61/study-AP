<p class="ap-question"><strong>問1</strong>　M/M/1の待ち行列モデルにおいて，窓口の利用率が25%から40%に増えると，平均待ち時間は何倍になるか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　1.25</div>
<div class="ap-choice-note">利用率の増加率（40÷25＝1.6）と平均待ち時間の倍率は一致しません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　1.60</div>
<div class="ap-choice-note">利用率そのものの比（1.6倍）であり、M/M/1の平均待ち時間の式から求まる倍率ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　2.00</div>
<div class="ap-choice-note">正しい。利用率25%と40%で平均待ち時間の比をとると2倍になります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　3.00</div>
<div class="ap-choice-note">式に当てはめた結果の倍率ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>M/M/1の<a href="用語/待ち行列モデル" class="internal-link" data-href="用語/待ち行列モデル">待ち行列モデル</a>で平均待ち時間を計算する式は以下のとおりです。<br>平均待ち時間＝利用率÷(1－利用率)×平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a></p>
<p>利用率が25%のときの平均待ち時間は、<br>0.25÷(1－0.25)×平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a>＝(1/3)×平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a></p>
<p>利用率が40%のときの平均待ち時間は、<br>0.4÷(1－0.4)×平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a>＝(2/3)×平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a></p>
<p>2つを比べると平均待ち時間は2倍になることがわかります。したがって「ウ」が正解です。</p>
