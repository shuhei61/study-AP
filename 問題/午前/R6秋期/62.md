<p class="ap-question"><strong>問62</strong>　官民データ活用推進基本法などに基づいて進められているオープンデータバイデザインに関して，行政機関における取組についての記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　行政機関が保有する個人情報を産業振興などの目的でオープン化する場合は，データ公開に先立ち，個人情報保護委員会への届出が義務化されている。</div>
<div class="ap-choice-note">個人情報は原則として不開示情報なので，届出の有無にかかわらず<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>として公開できません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　行政機関において収集蓄積された既存のデータが公開される場合，営利目的の利用は許されておらず，非営利の用途に限って利用が認められている。</div>
<div class="ap-choice-note"><a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>は営利目的・非営利目的を問わず二次利用可能なルールが適用され，営利目的での利用も認められています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　行政機関における情報システムの設計において，情報セキュリティを確保する観点から，公開するデータの用途を行政機関同士の相互利用に限定している。</div>
<div class="ap-choice-note"><a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>は国民参加・官民協働の推進や経済活性化などを目的としており，利用規約に従えば誰もが様々な用途で使用できます。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　対象となる行政データを，二次利用や機械判読に適した形態で無償公開することを前提に，情報システムや業務プロセスの企画，整備及び運用を行っている。</div>
<div class="ap-choice-note">正しい。<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>バイデザインは，公共データについて<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>を前提として情報システムや業務プロセス全体の企画，整備及び運用を行うことを指します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>バイデザインは，公共データについて，<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>を前提として情報システムや業務プロセス全体の企画，整備及び運用を行うことを指します。<a href="用語/官民データ" class="internal-link" data-href="用語/官民データ">官民データ</a>活用推進基本法において，国，地方公共団体が保有する<a href="用語/官民データ" class="internal-link" data-href="用語/官民データ">官民データ</a>について国民がインターネット等を通じて容易に利用できるよう措置を講じることが義務付けられたため，情報システムを<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>バイデザインにすることが求められています。</p>
<p><a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>とは，①営利目的・非営利目的を問わず二次利用可能なルールが適用されたもの，②機械判読に適したもの，③無償で利用できるものという3項目すべてを満たす形で公開されたデータのことを言います。<a href="用語/情報公開法" class="internal-link" data-href="用語/情報公開法">情報公開法</a>に不開示情報として定められているものはオープン化の対象になりません。個人情報は原則として不開示情報なので，届出の有無にかかわらず<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>として公開はできません。</p>
</div>
