<p class="ap-question"><strong>問59</strong>　システム監査基準(令和5年)における予備調査についての記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　監査対象先の事務手続やマニュアルなどを通じて，業務内容，業務分掌，体制などを把握する。</div>
<div class="ap-choice-note">正しい。予備調査は本調査に先だって監査対象の実態を把握するプロセスです。事務手続やマニュアル等による業務内容・業務分掌、組織図等による体制などを把握します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　監査対象の実態を把握するために，必ず現地に赴いて実施する。</div>
<div class="ap-choice-note">現地に赴いて確認する<a href="用語/現地調査法" class="internal-link" data-href="用語/現地調査法">現地調査法</a>は監査技法のひとつにすぎません。文書・資料の閲覧やインタビューなどで予備調査を行うこともあります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　監査の結論を裏付けるために，十分な監査証拠を入手する。</div>
<div class="ap-choice-note"><a href="用語/監査証拠" class="internal-link" data-href="用語/監査証拠">監査証拠</a>を実際に入手するプロセスは本調査です。予備調査は本調査のために必要な情報を事前に入手するために行います。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　調査の範囲は，監査対象先だけに限定する。</div>
<div class="ap-choice-note">調査範囲は監査対象部門に限られません。<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の目的とその必要に応じて関連部門に対して行うこともあります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>予備調査は，本調査に先だって監査対象の実態を把握するプロセスです。把握すべき事項には，情報システムや業務等の詳細，事務手続やマニュアル等による業務内容や業務分掌，組織図等による体制などがあります。「ア」が適切です。</p>
<p>監査人が現地に赴いて確認する<a href="用語/現地調査法" class="internal-link" data-href="用語/現地調査法">現地調査法</a>は監査技法のひとつにすぎません。関連する文書や資料等の閲覧，監査対象部門や関連部門へのインタビューなどの手法で予備調査をすることもあります。「イ」は誤りです。</p>
<p>実際に<a href="用語/監査証拠" class="internal-link" data-href="用語/監査証拠">監査証拠</a>を入手するプロセスは本調査です。予備調査は，本調査のための必要な情報を事前に入手するために行います。「ウ」は誤りです。</p>
<p>調査範囲は監査対象部門に限られません。<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の目的とその必要に応じて関連部門に対して行うこともあります。「エ」は誤りです。</p>
<p>正解は「ア」です。</p>
</div>
