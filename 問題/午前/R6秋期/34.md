<p class="ap-question"><strong>問34</strong>　IPv4ネットワークにおけるICMPの機能として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　MACアドレスだけが分かっているときにIPアドレスの解決を可能にする。</div>
<div class="ap-choice-note">これは<a href="用語/RARP" class="internal-link" data-href="用語/RARP">RARP</a>(Reverse Address Resolution Protocol)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　グローバルIPアドレスとプライベートIPアドレスとを相互に変換する。</div>
<div class="ap-choice-note">これは<a href="用語/NAT" class="internal-link" data-href="用語/NAT">NAT</a>(Network Address Translation)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　送信元ホストへ，IPパケットの送信エラー報告などの制御メッセージを通知する。</div>
<div class="ap-choice-note">正しい。<a href="用語/ICMP" class="internal-link" data-href="用語/ICMP">ICMP</a>は、IPパケット配送中に発生したエラーをホストに通知する機能を提供します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ネットワーク内のIPアドレスを一元管理し，クライアントに動的に割り当てる。</div>
<div class="ap-choice-note">これは<a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>(Dynamic Host Configuration Protocol)の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ICMP" class="internal-link" data-href="用語/ICMP">ICMP</a>(Internet Control Message Protocol)は、インターネットプロトコル(IP)の通信制御を補完するプロトコルで、データ配送中のエラー通知や送達エラーを通知する機能を持ちます。ネットワークのトラブルシューティングや管理のために非常に重要なプロトコルです。</p>
<p><a href="用語/ICMP" class="internal-link" data-href="用語/ICMP">ICMP</a>の代表的な機能は次のとおりです。</p>
<ul>
<li><a href="用語/ping" class="internal-link" data-href="用語/ping">ping</a>（ネットワークの疎通確認）</li>
<li>IPパケットの配送中にTTLが超過した時のエラーメッセージ</li>
<li>宛先に到達不能な時のエラーメッセージ</li>
<li>正しい経路へのリダイレクトメッセージ</li>
</ul>
