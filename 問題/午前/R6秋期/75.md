<p class="ap-question"><strong>問75</strong>　A社とB社がそれぞれ2種類の戦略を採る場合の市場シェアが表のように予想されるとき，ナッシュ均衡，すなわち互いの戦略が相手の戦略に対して最適になっている組合せはどれか。ここで，表の各欄において，左側の数値がA社のシェア，右側の数値がB社のシェアとする。<br><img class="ap-fig" src="../../../assets/問題/午前/R6秋期/75/75.png" alt="問題図" width="280" height="119" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　A社が戦略a1，B社が戦略b1を採る組合せ</div>
<div class="ap-choice-note">A社がa1を採る場合，B社はb2の方がシェアが高くなるため，b1を採る組合せは<a href="用語/ナッシュ均衡" class="internal-link" data-href="用語/ナッシュ均衡">ナッシュ均衡</a>ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　A社が戦略a1，B社が戦略b2を採る組合せ</div>
<div class="ap-choice-note">正しい。A社はB社の戦略にかかわらずa1，B社はA社の戦略にかかわらずb2を採るのがそれぞれ最適であり，(a1, b2)が<a href="用語/ナッシュ均衡" class="internal-link" data-href="用語/ナッシュ均衡">ナッシュ均衡</a>です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　A社が戦略a2，B社が戦略b1を採る組合せ</div>
<div class="ap-choice-note">A社はB社の戦略にかかわらずa1の方がシェアが高いため，a2を採る組合せは<a href="用語/ナッシュ均衡" class="internal-link" data-href="用語/ナッシュ均衡">ナッシュ均衡</a>ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　A社が戦略a2，B社が戦略b2を採る組合せ</div>
<div class="ap-choice-note">A社はB社の戦略にかかわらずa1の方がシェアが高いため，a2を採る組合せは<a href="用語/ナッシュ均衡" class="internal-link" data-href="用語/ナッシュ均衡">ナッシュ均衡</a>ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>A社が得るシェアに注目すると、B社がどちらの戦略を採った場合でも戦略a1を採るケースの方がシェアが高くなるのでA社は戦略a1を採用します。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R6秋期/75/75_1.png" alt="解説図 1" width="261" height="100" ></p>
<p>B社が得るシェアに注目すると、A社がどちらの戦略を採った場合でも戦略b2を採るケースの方がシェアが高くなるのでB社は戦略b2を採用します。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R6秋期/75/75_2.png" alt="解説図 2" width="261" height="100" ></p>
<p>したがってこの条件では、A社が戦略a1、B社が戦略b2を採る組合せになります。</p>
