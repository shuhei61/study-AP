<p class="ap-question"><strong>問36</strong>　IPv4のLANに接続されているプリンターのMACアドレスを，同一LAN上のPCから調べるときに使用するコマンドはどれか。ここで，PCはこのプリンターを直前に使用しており，プリンターのIPアドレスは分かっているものとする。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　arp</div>
<div class="ap-choice-note">正しい。arpは、同じネットワーク内の端末のMACアドレスを得るコマンドです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　ipconfig又はifconfig</div>
<div class="ap-choice-note">自端末のネットワーク設定情報を表示するコマンドです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　netstat</div>
<div class="ap-choice-note">自端末のTCP/IPネットワーク接続状況を調べるコマンドです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ping</div>
<div class="ap-choice-note">ネットワークの疎通診断を行うコマンドです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>arp(アープ)は、<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>テーブルの管理に使用されるコマンドで、<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>テーブルの表示およびエントリの追加／削除の機能を提供します。<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>から対応する機器のMACアドレスを得る、<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>(Address Resolution Protocol)と同名です。</p>
<p>設問の「PCはこのプリンターを直前に使用しており」という記述から、このPCの<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>テーブルにはプリンターの<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>に対応するMACアドレスが登録されていることが確認できます。arp -a で<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>テーブルを表示し、プリンターのMACアドレスを知ることが可能です。</p>
</div>
