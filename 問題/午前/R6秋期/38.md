<p class="ap-question"><strong>問38</strong>　インターネット上の使われていないIPアドレス空間に大量に届くパケットはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　インターネット上で公開されているWebサイトに対してPCから送信するパケット</div>
<div class="ap-choice-note">通常のWebアクセス用パケットであり、未使用IP宛てではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　インターネットにつながっており，実在するIoT機器から実在するサーバに送信されるパケット</div>
<div class="ap-choice-note">実在する通信先への通常のパケットです。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　攻撃者やマルウェアがIoT機器やサーバなどの攻撃対象を探すために送信するパケット</div>
<div class="ap-choice-note">正しい。未使用IPへのスキャンなど、攻撃対象探索に関連するパケットです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　有効な電子メールアドレスに対して攻撃者が標的型攻撃メールを送信するSMTPのパケット</div>
<div class="ap-choice-note">標的型攻撃メール用のパケットであり、未使用IP空間（ダークネット）宛てではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>インターネット上の使われていない<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>空間は、一般にダークネットと呼ばれています。通常のインターネット利用では、未使用の<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>にパケットが送信されることはほとんどありませんが、実際にダークネットを観測すると多数のパケットが未使用IPに向けて送出されています。</p>
<p>これらはマルウェアが次の感染対象を探すためのスキャン、脆弱性攻撃用のパケット、送信元IP詐称に対する応答パケットなど不正活動に関連します。したがって「ウ」が適切です。</p>
</div>
