<p class="ap-question"><strong>問24</strong>　顧客に，A～Zの英大文字26種類を用いた顧客コードを割り当てたい。現在の顧客総数は8,000人であって，毎年，前年対比で2割ずつ顧客が増えていくものとする。3年後まで全顧客にコードを割り当てられるようにするためには，顧客コードは少なくとも何桁必要か。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　3</div>
<div class="ap-choice-note">正しい。3桁で 26<sup>3</sup>＝17,576 となり、3年後の予想顧客総数（13,824人）を上回る最小の桁数です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　4</div>
<div class="ap-choice-note">4桁（26<sup>4</sup>＝456,976）は割り当て可能だが、設問の「少なくとも」より多い。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　5</div>
<div class="ap-choice-note">5桁（26<sup>5</sup>＝11,881,376）は必要以上の桁数です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　6</div>
<div class="ap-choice-note">6桁（26<sup>6</sup>＝308,915,776）は必要以上の桁数です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>3年後の予想顧客総数は、現在の顧客総数を1.2倍することを3回（3年分）繰り返すことで求めることができます。<br>
8,000人×1.2×1.2×1.2＝13,824人</p>
<p>26種類の文字を使った n 桁で表せるコードの総数は 26<sup>n</sup> 個なので、選択肢それぞれの桁数に設定したときの最大顧客数は次のように計算できます。</p>
<p>3桁 → 26<sup>3</sup>＝17,576<br>
※この段階で3年後の予想顧客総数（13,824人）を上回っているため、正解は3桁とわかります。<br>
4桁 → 26<sup>4</sup>＝456,976<br>
5桁 → 26<sup>5</sup>＝11,881,376<br>
6桁 → 26<sup>6</sup>＝308,915,776</p>
<p>設問では「少なくとも何桁必要か」と訊いているので、正解は「ア」になります。</p>
