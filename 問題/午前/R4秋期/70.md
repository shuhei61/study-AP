<p class="ap-question"><strong>問70</strong>　APIエコノミーの事例として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　既存の学内データベースのAPIを活用できるEAI(Enterprise Application Integration)ツールを使い，大学業務システムを短期間で再構築することによって経費を削減できた。</div>
<div class="ap-choice-note">社内・学内システムの統合に<a href="用語/EAI" class="internal-link" data-href="用語/EAI">EAI</a>を用いた事例であり，他社が公開する<a href="用語/Web API" class="internal-link" data-href="用語/Web API">Web API</a>を連携して価値を創出する<a href="用語/APIエコノミー" class="internal-link" data-href="用語/APIエコノミー">APIエコノミー</a>ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　自社で開発した音声合成システムの利用を促進するために，自部門で開発したAPIを自社内の他の部署に提供した。</div>
<div class="ap-choice-note">自社内の部門間でAPIを共有しているだけで，他社が公開する<a href="用語/Web API" class="internal-link" data-href="用語/Web API">Web API</a>を活用する経済圏の事例ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　不動産会社が自社で保持する顧客データをBI(Business Intelligence)ツールのAPIを使い可視化することによって，商圏における売上規模を分析できるようになった。</div>
<div class="ap-choice-note">自社データの分析・可視化であり，他社公開の<a href="用語/Web API" class="internal-link" data-href="用語/Web API">Web API</a>を組み込んでサービスを拡張する<a href="用語/APIエコノミー" class="internal-link" data-href="用語/APIエコノミー">APIエコノミー</a>の事例ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　ホテル事業者が，他社が公開しているタクシー配車アプリのAPIを自社のアプリに組み込み，サービスを提供した。</div>
<div class="ap-choice-note">正しい。他社が公開する<a href="用語/Web API" class="internal-link" data-href="用語/Web API">Web API</a>を自社サービスに組み込み価値を創出する<a href="用語/APIエコノミー" class="internal-link" data-href="用語/APIエコノミー">APIエコノミー</a>の事例です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>Webサービスを運営する企業によって提供され、httpやhttpsなどのWeb技術を使用してそのWebサービスの機能にアクセスできる仕組みを<a href="用語/Web API" class="internal-link" data-href="用語/Web API">Web API</a>といいます。Yahoo!API、TwitterAPI、GoogleマップAPI、AmazonAPIなどがこれに該当します。<br><a href="用語/APIエコノミー" class="internal-link" data-href="用語/APIエコノミー">APIエコノミー</a>(API経済圏)は、これらの他社から提供される<a href="用語/Web API" class="internal-link" data-href="用語/Web API">Web API</a>を連携・活用してサービスや価値を創出する経済の仕組みのことです。APIの利用側としては自社で同じ機能を開発をすることなく利用できるメリットがあり、APIを公開する側にはサービス使用に伴う課金で収益を得るほか、単純に自社サービスの広まりによりユーザー数が増加することがメリットとなります。<br>したがって他社の公開APIを利用している「エ」の事例が適切です。</p>
</div>
