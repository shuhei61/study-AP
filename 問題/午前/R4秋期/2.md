<p class="ap-question"><strong>問2</strong>　A，B，C，Dを論理変数とするとき，次のカルノー図と等価な論理式はどれか。ここで，・は論理積，＋は論理和，<span style="text-decoration:overline">X</span>はXの否定を表す。<br><img class="ap-fig" src="../../../assets/問題/午前/R4秋期/2/02.png" alt="問題図" width="235" height="154" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　A・B・<span style="text-decoration:overline">C</span>・D＋<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span></div>
<div class="ap-choice-note">赤枠・青枠のグループ化から得られる<a href="用語/論理式" class="internal-link" data-href="用語/論理式">論理式</a>と一致しない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　<span style="text-decoration:overline">A</span>・<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">C</span>・<span style="text-decoration:overline">D</span>＋B・D</div>
<div class="ap-choice-note">4変数すべてを<a href="用語/否定" class="internal-link" data-href="用語/否定">否定</a>した積項とB・Dの和であり，設問のカルノー図と等価ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　A・B・D＋<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span></div>
<div class="ap-choice-note"><span style="text-decoration:overline">A</span>・<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span>の項がなく，赤枠グループに対応する積項が欠けている。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　<span style="text-decoration:overline">A</span>・<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span>＋B・D</div>
<div class="ap-choice-note">正しい。2つのグループの<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>を<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>でつなぐとこの式になる。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>カルノー図は、行・列それぞれの論理変数の組合せの結果が"真"となる場合に"1"を、"偽"となる場合に"0"を、その該当セルに書きこむことで<a href="用語/論理式" class="internal-link" data-href="用語/論理式">論理式</a>を図式化したものです。論理回路を簡略化するために用いられます。</p>
<p>設問のカルノー図の"1"と"0"、4つの論理変数は下図のように対応します。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R4秋期/2/02_1.png" alt="解説図" width="329" height="169" ></p>
<p>カルノー図から<a href="用語/論理式" class="internal-link" data-href="用語/論理式">論理式</a>を導くには、図中の値が"1"のセルをグループ化して共通項を取り出すという手順を踏みます。このグループ化は次の4つのルールに従って行います。</p>
<ol><li>値が"1"のセルすべてを長方形で囲むこと</li><li>グループに含まれるセルの数は2<sup>n</sup>（1, 2, 4, 8, …）であること</li><li>カルノー図の上下左右の端は連続していると考える</li><li>1つのセルが複数のグループに属してもよい</li></ol>
<p>このルールに従って設問のカルノー図をグループ化すると、下図のように2つの長方形ですべての"1"を囲むことができます。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R4秋期/2/02_2.png" alt="解説図" width="250" height="145" ></p>
<p>次にグループごとに論理変数の組から共通項を取り出して、その<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>を作ります。</p>
<dl><dt>赤枠で囲ったグループ</dt><dd>論理変数の組は <span style="text-decoration:overline">A</span> <span style="text-decoration:overline">B</span> <span style="text-decoration:overline">C</span> <span style="text-decoration:overline">D</span> と <span style="text-decoration:overline">A</span> <span style="text-decoration:overline">B</span> C <span style="text-decoration:overline">D</span> なので、共通項は <span style="text-decoration:overline">A</span> <span style="text-decoration:overline">B</span> <span style="text-decoration:overline">D</span>、<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>は <span style="text-decoration:overline">A</span>・<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span></dd><dt>青枠で囲ったグループ</dt><dd>論理変数の組は <span style="text-decoration:overline">A</span> B <span style="text-decoration:overline">C</span> D、<span style="text-decoration:overline">A</span> B C D、A B <span style="text-decoration:overline">C</span> D、A B C D なので、共通項はB D、<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>はB・D</dd></dl>
<p>最後に、グループごとの<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>同士を<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>でつなぐことで、図と等価な<a href="用語/論理式" class="internal-link" data-href="用語/論理式">論理式</a>が得られます。</p>
<p>赤枠グループの<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>は「<span style="text-decoration:overline">A</span>・<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span>」、青枠グループの<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>は「B・D」であり、この2つを<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>（＋）でつなぐと「<span style="text-decoration:overline">A</span>・<span style="text-decoration:overline">B</span>・<span style="text-decoration:overline">D</span>＋B・D」になります。したがって「エ」が正解です。</p>
