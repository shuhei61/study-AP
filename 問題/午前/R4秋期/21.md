<p class="ap-question"><strong>問21</strong>　次の電子部品のうち，整流作用をもつ素子はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　コイル</div>
<div class="ap-choice-note"><a href="用語/コイル" class="internal-link" data-href="用語/コイル">コイル</a>には、回路に流れる電流を一定に保つ作用があります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　コンデンサ</div>
<div class="ap-choice-note"><a href="用語/コンデンサ" class="internal-link" data-href="用語/コンデンサ">コンデンサ</a>は、電荷(電気エネルギー)を蓄えたり放出したりする電子部品です。進相<a href="用語/コンデンサ" class="internal-link" data-href="用語/コンデンサ">コンデンサ</a>には力率の改善作用があります。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　ダイオード</div>
<div class="ap-choice-note">正しい。<a href="用語/ダイオード" class="internal-link" data-href="用語/ダイオード">ダイオード</a>は、電流を一方向にしか流さない整流作用をもつ電子部品です。整流作用を利用することで、一定時間ごとの電気の流れる向きが変わる交流電流から順方向の電流のみを取り出しています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　抵抗器</div>
<div class="ap-choice-note">抵抗器は、回路に流れる電流を変化させる作用があります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ダイオード" class="internal-link" data-href="用語/ダイオード">ダイオード</a>は、電流を一方向にしか流さない整流作用をもつ電子部品です。整流作用を利用することで、一定時間ごとの電気の流れる向きが変わる交流電流から順方向の電流のみを取り出しています。</p>
</div>
