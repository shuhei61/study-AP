<p class="ap-question"><strong>問22</strong>　フラッシュメモリの特徴として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　書込み回数は無制限である。</div>
<div class="ap-choice-note"><a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>は書換えを繰り返すとデータ保持能力が低下する。書込み回数に上限（数万～数百万回程度）がある。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　書込み時は回路基板から外して，専用のROMライターで書き込まなければならない。</div>
<div class="ap-choice-note">紫外線消去型<a href="用語/EPROM" class="internal-link" data-href="用語/EPROM">EPROM</a>（UV-<a href="用語/EPROM" class="internal-link" data-href="用語/EPROM">EPROM</a>）の説明です。<a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>は電気的に書換えでき、基板から外す必要はない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　定期的にリフレッシュしないと，データが失われる。</div>
<div class="ap-choice-note"><a href="用語/DRAM" class="internal-link" data-href="用語/DRAM">DRAM</a>の説明です。<a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>は不揮発性メモリで、<a href="用語/リフレッシュ" class="internal-link" data-href="用語/リフレッシュ">リフレッシュ</a>は不要である。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　データの書換え時には，あらかじめ前のデータを消去してから書込みを行う。</div>
<div class="ap-choice-note">正しい。<a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>はブロック単位で前のデータを消去してから書込む。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>のメモリセルは、書換えを繰り返すことでデータ保持能力が低下していく。したがって無限に書換えができるわけではなく、一般には数万回から数百万回が書換え回数の上限とされている。<a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>は電気的に書換えを行うので、書込み時にメモリを回路基板から外す必要はない（USBメモリなど）。メモリを回路基板から外して専用の装置で書込みや消去をするものとして、UV-<a href="用語/EPROM" class="internal-link" data-href="用語/EPROM">EPROM</a>（紫外線消去型<a href="用語/EPROM" class="internal-link" data-href="用語/EPROM">EPROM</a>）がある。<a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>は電源を切ってもデータが消えない不揮発性メモリなので、<a href="用語/DRAM" class="internal-link" data-href="用語/DRAM">DRAM</a>とは異なり、データを定期的に更新する<a href="用語/リフレッシュ" class="internal-link" data-href="用語/リフレッシュ">リフレッシュ</a>動作は不要である。<a href="用語/フラッシュメモリ" class="internal-link" data-href="用語/フラッシュメモリ">フラッシュメモリ</a>は、あらかじめブロック単位で前のデータを消去してから書込むことでデータの書換えを行う。したがって「エ」が適切である。</p>
</div>
