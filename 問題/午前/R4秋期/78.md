# [令和4年秋期 午前 問78](https://www.ap-siken.com/kakomon/04_aki/q78.html)

#問題 #ストラテジ #法務 #知的財産権

<div class="ap-quiz">
<details class="ap-reveal">

<p class="ap-question"><strong>問78</strong>　A社は顧客管理システムの開発を，情報システム子会社であるB社に委託し，B社は要件定義を行った上で，ソフトウェア設計・プログラミング・ソフトウェアテストまでを，協力会社であるC社に委託した。C社では自社の社員Dにその作業を担当させた。このとき，開発したプログラムの著作権はどこに帰属するか。ここで，関係者の間には，著作権の帰属に関する特段の取決めはないものとする。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　A社</div>
<div class="ap-choice-note"><a href="用語/請負契約" class="internal-link" data-href="用語/請負契約">請負契約</a>では原則として実際に開発を行う請負業者が<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>を得ます。発注者A社に帰属しません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　B社</div>
<div class="ap-choice-note">プログラム開発の実作業はC社が行っています。B社は要件定義までで、<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>は実作業を行った請負業者側に帰属します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　C社</div>
<div class="ap-choice-note">正しい。<a href="用語/職務著作" class="internal-link" data-href="用語/職務著作">職務著作</a>の原則により、C社の社員Dが作成したプログラムの<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>はC社に帰属します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　社員D</div>
<div class="ap-choice-note">職務上作成した著作物の<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>は、個人ではなく法人に帰属します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>設問ではA社がB社にシステム開発を委託し、さらにB社からC社にプログラム開発を委託しているので、A社とB社の間、B社とC社の間には<a href="用語/請負契約" class="internal-link" data-href="用語/請負契約">請負契約</a>が成立していると考えることができます。</p>
<p><a href="用語/著作権法" class="internal-link" data-href="用語/著作権法">著作権法</a>の原則では、<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>は著作物を創作した者に与えられますから、<a href="用語/請負契約" class="internal-link" data-href="用語/請負契約">請負契約</a>では、原則として実際に開発を行う請負業者がその成果物の<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>を得ることになります。また、法人の業務に従事する者が職務上作成する著作物に関する<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>は、個人ではなく法人に帰属します。</p>
<p>設問ではプログラム開発を担当したのはC社のD社員ですから、プログラムの<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>はD社員が在籍している「C社」に帰属することとなります。したがって「ウ」が正解です。</p>
</div>
</div>
