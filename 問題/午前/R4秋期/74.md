<p class="ap-question"><strong>問74</strong>　ハーシィ及びブランチャードが提唱したSL理論の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　開放の窓，秘密の窓，未知の窓，盲点の窓の四つの窓を用いて，自己理解と対人関係の良否を説明した理論</div>
<div class="ap-choice-note">自己分析に使用するジョハリの窓の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　教示的，説得的，参加的，委任的の四つに，部下の成熟度レベルによって，リーダーシップスタイルを分類した理論</div>
<div class="ap-choice-note">正しい。<a href="用語/SL理論" class="internal-link" data-href="用語/SL理論">SL理論</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　共同化，表出化，連結化，内面化の四つのプロセスによって，個人と組織に新たな知識が創造されるとした理論</div>
<div class="ap-choice-note">ナレッジマネジメントに使用する<a href="用語/SECIモデル" class="internal-link" data-href="用語/SECIモデル">SECIモデル</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　生理的，安全，所属と愛情，承認と自尊，自己実現といった五つの段階で欲求が発達するとされる理論</div>
<div class="ap-choice-note">マズローの自己実現理論（欲求の5段階説）の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/SL理論" class="internal-link" data-href="用語/SL理論">SL理論</a>（状況対応<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>理論）は、いかなる状況にも効果的な唯一万能のリーダー行動は存在しないという主張の下、<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>の有効性を状況との関係で捉え、状況要素のうち最も重要である部下や集団（フォロワー）の能力及び意欲の水準（レディネス）ごとに、有効性が高い<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>のスタイルを示したモデルです。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R4秋期/74/74.png" alt="解説図" width="419" height="247" ></p>
