<p class="ap-question"><strong>問62</strong>　経済産業省が取りまとめた"デジタル経営改革のための評価指標(DX推進指標)"によれば，DXを実現する上で基盤となるITシステムの構築に関する指標において，"ITシステムに求められる要素"について経営者が確認すべき事項はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ITシステムの全体設計や協働できるベンダーの選定などを行える人材を育成・確保できているか。</div>
<div class="ap-choice-note">"ITシステムに求められる要素"ではない。ベンダーに丸投げせず、ITシステムの全体設計や協創できるベンダーを選別できる人材の確保は、<a href="用語/ITガバナンス" class="internal-link" data-href="用語/ITガバナンス">ITガバナンス</a>等の"ガバナンス・体制について求められる要素"である。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　環境変化に迅速に対応し，求められるデリバリースピードに対応できるITシステムとなっているか。</div>
<div class="ap-choice-note">正しい。環境変化に迅速に対応し、求められるデリバリースピードに対応できるITシステムとなっているかどうかが<a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>となっている。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　データ処理において，リアルタイム性よりも，ビッグデータの蓄積と事後の分析が重視されているか。</div>
<div class="ap-choice-note">データをリアルタイムで利用できるITシステムになっているかどうかが<a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>であり、<a href="用語/ビッグデータ" class="internal-link" data-href="用語/ビッグデータ">ビッグデータ</a>の蓄積と事後分析のみを重視する内容は指標と逆である。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　データを迅速に活用するために，全体最適よりも，個別最適を志向したITシステムとなっているか。</div>
<div class="ap-choice-note">個別最適ではなく、全社最適を踏まえたITシステムになっているかどうかが<a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>である。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/デジタル経営改革のための評価指標" class="internal-link" data-href="用語/デジタル経営改革のための評価指標">デジタル経営改革のための評価指標</a>(DX推進指標)は、<a href="用語/デジタルトランスフォーメーション" class="internal-link" data-href="用語/デジタルトランスフォーメーション">デジタルトランスフォーメーション</a>(DX)の変革を後押しすることを目的とし、ビジネスのアクションに繋がるための気付きの機会を提供するものとして経済産業省によって策定されたものである。項目ごとにレベル0～5の成熟度が設定されており、各企業が簡易な自己診断を行うことが可能となっている。</p>
<p>本指標では、"ITシステムに求められる要素"として以下の3つの事項を規定している。</p>
<p>データ活用 … データを、リアルタイム等使いたい形で使えるITシステムとなっているか<br>スピード・アジリティ … 環境変化に迅速に対応し、求められるデリバリースピードに対応できるITシステムとなっているか<br>全社最適 … 部門を超えてデータを活用し、バリューチェーンワイドで顧客視点での価値創出ができるよう、システム間を連携させるなどにより、全社最適を踏まえたITシステムとなっているか</p>
</div>
</div>
