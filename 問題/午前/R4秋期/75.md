<p class="ap-question"><strong>問75</strong>　予測手法の一つであるデルファイ法の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　現状の指標の中に将来の動向を示す指標があることに着目して予測する。</div>
<div class="ap-choice-note">シナリオプランニングの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　将来予測のためのモデル化した連立方程式を解いて予測する。</div>
<div class="ap-choice-note"><a href="用語/回帰分析" class="internal-link" data-href="用語/回帰分析">回帰分析</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　同時点における複数の観測データの統計比較分析によって将来を予測する。</div>
<div class="ap-choice-note">統計的予測理論の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　複数の専門家へのアンケートの繰返しによる回答の収束によって将来を予測する。</div>
<div class="ap-choice-note">正しい。<a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>は、技術開発戦略の立案に必要となる将来の技術動向の予測などに用いられる技法であり、①複数の専門家からの意見収集、②得られた意見の統計的集約、③集約された意見のフィードバックという三手順を繰り返して回答を収束し、最終的な意見を得るものです。したがって「エ」が適切です。</p>
</div>
