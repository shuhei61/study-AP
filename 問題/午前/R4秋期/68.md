<p class="ap-question"><strong>問68</strong>　ターゲットリターン価格設定の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　競合の価格を十分に考慮した上で価格を決定する。</div>
<div class="ap-choice-note">これは実勢価格の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　顧客層，時間帯，場所など市場セグメントごとに異なった価格を決定する。</div>
<div class="ap-choice-note">これは差別価格の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　目標とする投資収益率を実現するように価格を決定する。</div>
<div class="ap-choice-note">正しい。ターゲットリターン価格設定の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　リサーチなどによる消費者の値頃感に基づいて価格を決定する。</div>
<div class="ap-choice-note">これは知覚価値価格の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>ターゲットリターン価格とは、コスト志向型の価格設定法で、目標とする投資収益率(<a href="用語/ROI" class="internal-link" data-href="用語/ROI">ROI</a>)を達成するために得るべき金額を計算し、それを原価に加えて販売価格とする方法です。目標収益法ともいいます。<br>ターゲットリターン価格＝原価＋{(投下資本×目標収益率)÷販売数量}</p>
</div>
</div>
