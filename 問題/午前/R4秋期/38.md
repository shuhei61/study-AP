<p class="ap-question"><strong>問38</strong>　デジタル証明書が失効しているかどうかをオンラインで確認するためのプロトコルはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　CHAP</div>
<div class="ap-choice-note">Challenge Handshake Authentication Protocolの略。チャレンジレスポンス方式を使用した安全性の高い認証方式で、PPP接続で利用されます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　LDAP</div>
<div class="ap-choice-note">Lightweight Directory Access Protocolの略。ディレクトリサービス(LANなどのコンピュータネットワーク上にあるユーザー情報、接続されているプリンターなどの資源を記憶し、検索しやすいようにまとめたもの)に対するアクセスを提供するプロトコルです。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　OCSP</div>
<div class="ap-choice-note">正しい。<a href="用語/OCSP" class="internal-link" data-href="用語/OCSP">OCSP</a>は<a href="用語/デジタル証明書" class="internal-link" data-href="用語/デジタル証明書">デジタル証明書</a>が失効しているかどうかをオンラインで確認するためのプロトコルです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　SNMP</div>
<div class="ap-choice-note">Simple Network Management Protocolの略。TCP/IPネットワーク上でネットワーク上の機器の情報を収集して、監視や制御を行うためのプロトコルです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/OCSP" class="internal-link" data-href="用語/OCSP">OCSP</a>(Online Certificate Status Protocol)は、<a href="用語/デジタル証明書" class="internal-link" data-href="用語/デジタル証明書">デジタル証明書</a>の失効情報をオンライン確認し、<a href="用語/デジタル証明書" class="internal-link" data-href="用語/デジタル証明書">デジタル証明書</a>の有効性をリアルタイムで検証するプロトコルです。</p>
<p>クライアントは、確認対象となる<a href="用語/デジタル証明書" class="internal-link" data-href="用語/デジタル証明書">デジタル証明書</a>のシリアル番号を<a href="用語/OCSP" class="internal-link" data-href="用語/OCSP">OCSP</a>レスポンダに送信し、有効性検証の結果を受け取ります。この仕組みを利用することで、クライアント自身が<a href="用語/CRL" class="internal-link" data-href="用語/CRL">CRL</a>を取得・検証する手間を省くことができます。</p>
</div>
