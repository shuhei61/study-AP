# [令和4年秋期 午前 問67](https://www.ap-siken.com/kakomon/04_aki/q67.html)

#問題 #ストラテジ #経営戦略マネジメント #経営戦略手法

<details class="ap-reveal">

<p class="ap-question"><strong>問67</strong>　M&amp;Aの際に，買収対象企業の経営実態，資産や負債，期待収益性といった企業価値などを買手が詳細に調査する行為はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　株主総会招集請求</div>
<div class="ap-choice-note">これは<a href="用語/株主総会" class="internal-link" data-href="用語/株主総会">株主総会</a>招集請求の説明です。総議決権の3%以上の議決権を有する株主が、<a href="用語/取締役" class="internal-link" data-href="用語/取締役">取締役</a>に対して議題と招集の理由を示して<a href="用語/株主総会" class="internal-link" data-href="用語/株主総会">株主総会</a>の招集を請求できる権利です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　公開買付開始公告</div>
<div class="ap-choice-note">これは公開買付開始公告の説明です。<a href="用語/TOB" class="internal-link" data-href="用語/TOB">TOB</a>(株式公開買付け)を行おうとする者が<a href="用語/金融商品取引法" class="internal-link" data-href="用語/金融商品取引法">金融商品取引法</a>の定めにより行う、買付期間・買付数量・買付価格などの情報開示です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　セグメンテーション</div>
<div class="ap-choice-note">これはセグメンテーションの説明です。市場を細分化して、細分化された市場ごとに最適な戦略・施策を立案・実施するマーケティング戦略です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　デューデリジェンス</div>
<div class="ap-choice-note">正しい。デューデリジェンスは、ビジネスの実態、財務状況、取引先、所有する不動産、<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>、<a href="用語/知的財産権" class="internal-link" data-href="用語/知的財産権">知的財産権</a>、従業員との関係など、買収対象企業のあらゆる事項について入念に詳細な調査を行うことです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/M&A" class="internal-link" data-href="用語/M&A">M&A</a>では買手が買収対象の企業価値を把握するため、経営実態・資産・負債・期待収益性などを詳細に調査します。この調査行為がデューデリジェンスです。<a href="用語/株主総会" class="internal-link" data-href="用語/株主総会">株主総会</a>招集請求や公開買付開始公告は手続・開示に関するもの、セグメンテーションは市場細分化のマーケティング手法であり、本問の「買手による詳細調査」とは異なります。</p>
</div>
