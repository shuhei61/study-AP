<p class="ap-question"><strong>問31</strong>　IPアドレスの自動設定をするためにDHCPサーバが設置されたLAN環境の説明のうち，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　DHCPによる自動設定を行うPCでは，IPアドレスは自動設定できるが，サブネットマスクやデフォルトゲートウェイアドレスは自動設定できない。</div>
<div class="ap-choice-note"><a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>は<a href="用語/サブネットマスク" class="internal-link" data-href="用語/サブネットマスク">サブネットマスク</a>やデフォルト<a href="用語/ゲートウェイ" class="internal-link" data-href="用語/ゲートウェイ">ゲートウェイ</a>も自動で設定します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　DHCPによる自動設定を行うPCと，IPアドレスが固定のPCを混在させることはできない。</div>
<div class="ap-choice-note">固定<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>を自動割り当ての範囲から除外することで混在が可能です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　DHCPによる自動設定を行うPCに，DHCPサーバのアドレスを設定しておく必要はない。</div>
<div class="ap-choice-note">正しい。<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>の割り当てを要求するPCは、まず自ネットワークに「<a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>発見<a href="用語/パケット" class="internal-link" data-href="用語/パケット">パケット</a>」を<a href="用語/ブロードキャスト" class="internal-link" data-href="用語/ブロードキャスト">ブロードキャスト</a>します。「<a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>発見<a href="用語/パケット" class="internal-link" data-href="用語/パケット">パケット</a>」を受け取った<a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a><a href="用語/サーバ" class="internal-link" data-href="用語/サーバ">サーバ</a>は、使用可能な<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>と<a href="用語/サブネットマスク" class="internal-link" data-href="用語/サブネットマスク">サブネットマスク</a>などのネットワーク設定を<a href="用語/クライアント" class="internal-link" data-href="用語/クライアント">クライアント</a>に通知します。<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>の自動割り当ては、このような手順で行われるため、<a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a><a href="用語/サーバ" class="internal-link" data-href="用語/サーバ">サーバ</a>のアドレスをあらかじめPCに設定しておく必要はありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　一度IPアドレスを割り当てられたPCは，その後電源が切られた期間があっても必ず同じIPアドレスを割り当てられる。</div>
<div class="ap-choice-note"><a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>によって割り当てられる<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>は空いているものから選ばれるため、前回と同じとは限りません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>(Dynamic Host Configuration Protocol)は、<a href="用語/TCP/IP" class="internal-link" data-href="用語/TCP/IP">TCP/IP</a>ネットワークで、ネットワークに接続するノードへの<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>の割り当てをアドレスプールから自動的に行うプロトコルです。</p>
</div>
