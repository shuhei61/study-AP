<p class="ap-question"><strong>問50</strong>　スクラムのスプリントにおいて，(1)～(3)のプラクティスを採用して開発を行い，スプリントレビューの後にKPT手法でスプリントレトロスペクティブを行った。"KPT"の"T"に該当する例はどれか。<br>〔プラクティス〕ペアプログラミングでコードを作成する。スタンドアップミーティングを行う。テスト駆動開発で開発を進める。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　開発したプログラムは欠陥が少なかったので，今後もペアプログラミングを継続する。</div>
<div class="ap-choice-note">K(KeeP)に該当する。継続すべき良かった点の例。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　スタンドアップミーティングにメンバー全員が集まらないことが多かった。</div>
<div class="ap-choice-note">P(Problem)に該当する。発生した問題の例。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　次のスプリントからは，スタンドアップミーティングにタイムキーパーを置き，終了5分前を知らせるようにする。</div>
<div class="ap-choice-note">正しい。T(Try)に該当する。今後試したいことの例。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　テストコードの作成に見積り以上の時間が掛かった。</div>
<div class="ap-choice-note">P(Problem)に該当する。発生した問題の例。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/スクラム" class="internal-link" data-href="用語/スクラム">スクラム</a>ガイドによれば、<a href="用語/スプリントレトロスペクティブ" class="internal-link" data-href="用語/スプリントレトロスペクティブ">スプリントレトロスペクティブ</a>(<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>後に行われる<a href="用語/ふりかえり" class="internal-link" data-href="用語/ふりかえり">ふりかえり</a>)では、①<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>中に何がうまくいったか、②どのような問題が発生したか、そしてそれらの③問題がどのように解決されたか（または解決されなかったか）について話し合うとされています。</p>
<p><a href="用語/KPT" class="internal-link" data-href="用語/KPT">KPT</a>(ケプト)手法は、<a href="用語/ふりかえり" class="internal-link" data-href="用語/ふりかえり">ふりかえり</a>の際に、継続すべき良かった点（KeeP)、発生した問題（Problem)、今後試したいこと（Try）を開発チーム全員で考える<a href="用語/フレームワーク" class="internal-link" data-href="用語/フレームワーク">フレームワーク</a>です。</p>
<p><a href="用語/KPT" class="internal-link" data-href="用語/KPT">KPT</a>の"T"は今後試したいことなので、次の<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>からの新しい施策である「ウ」が該当します。</p>
