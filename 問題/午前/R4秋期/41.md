<p class="ap-question"><strong>問41</strong>　JIS Q 31000:2019(リスクマネジメント－指針)におけるリスクアセスメントを構成するプロセスの組合せはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　リスク特定，リスク評価，リスク受容</div>
<div class="ap-choice-note">リスク受容は<a href="用語/リスク対応" class="internal-link" data-href="用語/リスク対応">リスク対応</a>の一形態であり、リスクアセスメントの構成要素ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　リスク特定，リスク分析，リスク評価</div>
<div class="ap-choice-note">正しい。<a href="用語/JIS Q 31000" class="internal-link" data-href="用語/JIS Q 31000">JIS Q 31000</a>ではリスクアセスメントをこの3プロセスの組合せと定義しています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　リスク分析，リスク対応，リスク受容</div>
<div class="ap-choice-note"><a href="用語/リスク対応" class="internal-link" data-href="用語/リスク対応">リスク対応</a>・リスク受容はリスク処理の段階であり、リスクアセスメントには含まれません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　リスク分析，リスク評価，リスク対応</div>
<div class="ap-choice-note"><a href="用語/リスク対応" class="internal-link" data-href="用語/リスク対応">リスク対応</a>はリスクアセスメントの後に行うリスク処理であり、構成要素ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/リスクマネジメント" class="internal-link" data-href="用語/リスクマネジメント">リスクマネジメント</a>の指針を示した規格である<a href="用語/JIS Q 31000" class="internal-link" data-href="用語/JIS Q 31000">JIS Q 31000</a>では、リスクアセスメントを「リスク特定，リスク分析及びリスク評価を網羅するプロセス全体を指す」と定義しています。したがって、正しい組合せは「リスク特定，リスク分析，リスク評価」の3つです。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R4秋期/41/41.png" alt="解説図" width="375" height="171" ></p>
<p>リスク特定は、組織の目的の達成を助ける又は妨害する可能性のあるリスクを発見し，認識し，記述するプロセスです。リスク分析は、必要に応じてリスクのレベルを含め，リスクの性質及び特徴を理解するプロセスです。リスク評価は、リスク分析の結果と確立された<a href="用語/リスク基準" class="internal-link" data-href="用語/リスク基準">リスク基準</a>との比較をし，追加する行為の決定を裏付けるプロセスです。</p>
