<p class="ap-question"><strong>問8</strong>　ディープラーニングの学習にGPUを用いる利点として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　各プロセッサコアが独立して異なるプログラムを実行し，異なるデータを処理できる。</div>
<div class="ap-choice-note">これはマルチコア<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>を用いる利点です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　行列演算ユニットを用いて，行列演算を高速に実行できる。</div>
<div class="ap-choice-note">正しい。<a href="用語/GPU" class="internal-link" data-href="用語/GPU">GPU</a>(Graphics Processing Unit)を用いる利点です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　浮動小数点演算ユニットをコプロセッサとして用い，浮動小数点演算ができる。</div>
<div class="ap-choice-note">これは<a href="用語/FPU" class="internal-link" data-href="用語/FPU">FPU</a>(Floating Point Unit、浮動小数点演算装置)を用いる利点です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　分岐予測を行い，パイプラインの利用効率を高めた処理を実行できる。</div>
<div class="ap-choice-note"><a href="用語/GPU" class="internal-link" data-href="用語/GPU">GPU</a>は<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>と違って命令制御関連の機能が弱く、分岐予測やアウトオブオーダー実行などの命令スケジューリングには向きません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/GPU" class="internal-link" data-href="用語/GPU">GPU</a>(Graphics Processing Unit)は、コンピュータにおいて画像処理を専門に担当するハードウェア部品です。動画再生や3DCGの<a href="用語/レンダリング" class="internal-link" data-href="用語/レンダリング">レンダリング</a>などの定型的かつ大量の演算が要求される処理において、<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の補助演算装置として機能します。</p>
<p>最近では、膨大な計算を必要とする科学<a href="用語/シミュレーション" class="internal-link" data-href="用語/シミュレーション">シミュレーション</a>や<a href="用語/機械学習" class="internal-link" data-href="用語/機械学習">機械学習</a>の分野でも<a href="用語/GPU" class="internal-link" data-href="用語/GPU">GPU</a>を利用すること(<a href="用語/GPGPU" class="internal-link" data-href="用語/GPGPU">GPGPU</a>)も増えています。<a href="用語/ディープラーニング" class="internal-link" data-href="用語/ディープラーニング">ディープラーニング</a>では、システムに学習をさせるときに行列演算を含む大量の並列演算が必要なので、同じ種類の計算を大量に行うことに特化した<a href="用語/GPU" class="internal-link" data-href="用語/GPU">GPU</a>が活用されています。</p>
<p>したがって「イ」の説明が適切です。</p>
</div>
