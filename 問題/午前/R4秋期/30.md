<p class="ap-question"><strong>問30</strong>　ACID特性の四つの性質に含まれないものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　一貫性</div>
<div class="ap-choice-note">ACIDのConsistency（<a href="用語/一貫性" class="internal-link" data-href="用語/一貫性">一貫性</a>）に該当する性質です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　可用性</div>
<div class="ap-choice-note">正しい。<a href="用語/可用性" class="internal-link" data-href="用語/可用性">可用性</a>は<a href="用語/ACID特性" class="internal-link" data-href="用語/ACID特性">ACID特性</a>には含まれません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　原子性</div>
<div class="ap-choice-note">ACIDのAtomicity（原子性）に該当する性質です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　耐久性</div>
<div class="ap-choice-note">ACIDのDurability（耐久性）に該当する性質です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ACID特性" class="internal-link" data-href="用語/ACID特性">ACID特性</a>は、<a href="用語/データベース" class="internal-link" data-href="用語/データベース">データベース</a>の<a href="用語/トランザクション" class="internal-link" data-href="用語/トランザクション">トランザクション</a>処理を行う上で必要不可欠とされる4つの性質(Atomicity・Consistency・Isolation・Durability)の頭文字を並べた言葉です。</p>
<p>Atomicity（原子性）<br><a href="用語/トランザクション" class="internal-link" data-href="用語/トランザクション">トランザクション</a>内の処理がすべて実行されるか、または全く実行されないことを保証する性質</p>
<p>Consistency（<a href="用語/一貫性" class="internal-link" data-href="用語/一貫性">一貫性</a>）<br><a href="用語/トランザクション" class="internal-link" data-href="用語/トランザクション">トランザクション</a>によりデータの矛盾が発生せず、常に<a href="用語/データベース" class="internal-link" data-href="用語/データベース">データベース</a>の整合性が保たれていることを保証する性質</p>
<p>Isolation（独立性）<br><a href="用語/隔離" class="internal-link" data-href="用語/隔離">隔離</a>性と呼ばれる。複数の<a href="用語/トランザクション" class="internal-link" data-href="用語/トランザクション">トランザクション</a>を同時に実行した場合と、順番に実行した場合の結果が等しくなることを保証する性質</p>
<p>Durability（永続性）<br>耐久性と呼ばれる。一旦正常終了した<a href="用語/トランザクション" class="internal-link" data-href="用語/トランザクション">トランザクション</a>の結果は、その後システムに障害が発生しても失われないことを保証する性質</p>
<p>したがって、<a href="用語/ACID特性" class="internal-link" data-href="用語/ACID特性">ACID特性</a>に含まれないのは「イ」の<a href="用語/可用性" class="internal-link" data-href="用語/可用性">可用性</a>です。</p>
</div>
