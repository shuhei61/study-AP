<p class="ap-question"><strong>問80</strong>　ソフトウェアやデータに欠陥がある場合に，製造物責任法の対象となるものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　ROM化したソフトウェアを内蔵した組込み機器</div>
<div class="ap-choice-note">正しい。ソフトウェアは無体物であり対象外ですが、この場合は<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>がある部品(ソフトウェア)を含むハードウェアに<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>があるものとされるため対象となります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　アプリケーションソフトウェアパッケージ</div>
<div class="ap-choice-note">ソフトウェアは無体物であるため対象外です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　利用者がPCにインストールしたOS</div>
<div class="ap-choice-note">OSもソフトウェア(=無体物)であるため対象外です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　利用者によってネットワークからダウンロードされたデータ</div>
<div class="ap-choice-note">無体物であるため対象外です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>製造物責任法(<a href="用語/PL法" class="internal-link" data-href="用語/PL法">PL法</a>)は、製造物の安全性上の<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>により人の生命、身体、財産に被害が生じた場合に、消費者が製造業者に対して損害賠償を請求できることを定めた法律です。</p>
<p>製造物責任法では製造物を「製造又は加工された動産」と定義しています。したがって、サービス、不動産、未加工のものは、この定義上の製造物には含まれず<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>による被害が生じた場合でも製造物責任法の対象外となります。同じくコンピュータプログラムのような無体物も動産ではないため対象外となりますが、<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>があるプログラムを組み込んだハードウェア(製造物)の使用によって損害が生じた場合は、その動産であるハードウェアに<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>があるものとして製造物責任法の対象となります。また、<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>による被害がその製造物自体の被害に留まった場合であれば、この法律の適用外となり民法による救済対象となります。</p>
