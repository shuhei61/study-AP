<p class="ap-question"><strong>問62</strong>　TCOの算定に当たって，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　エンドユーザーコンピューティングにおける利用部門の運用費用は考慮しない。</div>
<div class="ap-choice-note">運用費用は<a href="用語/ランニングコスト" class="internal-link" data-href="用語/ランニングコスト">ランニングコスト</a>なので<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>に含める。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　システム監査における監査対象データの収集費用や管理費用は考慮しない。</div>
<div class="ap-choice-note">システムに関する管理費用なので<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>に含める。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　システム障害の発生などによって，その障害とは直接関係のない仕入先企業が被るおそれがある，将来的な損失額も考慮する。</div>
<div class="ap-choice-note">システム障害と直接関係のない他社の損害額までは<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>に含めない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　利用部門におけるシステム利用に起因する，埋没原価などの見えない費用も考慮する。</div>
<div class="ap-choice-note">正しい。埋没原価などの見えない費用も<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>の算定に含める。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>(Total Cost of Ownership)は、ある設備・システムなどを取得してから廃棄に至るまでに発生するコストの総額を表します。総所有コストとも呼ばれます。特にITシステムでは購入金額や利用料金以外にもライフサイクルを通して様々なコストが生じるため、全体のコストを総合的かつ正確に把握することが効果的な投資判断につながります。</p>
<p><a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>は、初期投資額である<a href="用語/イニシャルコスト" class="internal-link" data-href="用語/イニシャルコスト">イニシャルコスト</a>と、維持管理費用である<a href="用語/ランニングコスト" class="internal-link" data-href="用語/ランニングコスト">ランニングコスト</a>に大別できます。<a href="用語/イニシャルコスト" class="internal-link" data-href="用語/イニシャルコスト">イニシャルコスト</a>の例はハードウェア購入・設置費用、パッケージソフトの購入費用・開発費、初期教育費などです。<a href="用語/ランニングコスト" class="internal-link" data-href="用語/ランニングコスト">ランニングコスト</a>の例は保守・サポート契約費、ライセンス料、運用人件費、消耗品費などです。</p>
<p>「エ」が正解です。埋没原価とは、過去に支出された費用や、意思決定の内容にかかわらず回収できない費用のことです。例えば、従来使用していた設備の<a href="用語/減価償却" class="internal-link" data-href="用語/減価償却">減価償却</a>費、旧設備の除却損、ユーザーサポートに伴う<a href="用語/機会損失" class="internal-link" data-href="用語/機会損失">機会損失</a>などが埋没原価に該当します。<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>の算定に当たっては、実際の支出として反映される費用の他にも埋没原価を含める必要があります。</p>
<p>「ア」の運用費用は<a href="用語/ランニングコスト" class="internal-link" data-href="用語/ランニングコスト">ランニングコスト</a>なので<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>に含めます。「イ」の<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>における収集・管理費用もシステムに関する管理費用なので<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>に含めます。「ウ」について、将来的に自社で負担することが予定される損失額は<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>に含めるべきですが、システム障害と直接関係のない別の企業が被る損害額まで<a href="用語/TCO" class="internal-link" data-href="用語/TCO">TCO</a>には含める必要はありません。</p>
</div>
