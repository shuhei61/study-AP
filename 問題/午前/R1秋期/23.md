<p class="ap-question"><strong>問23</strong>　3ビットのデータx1，x2，x3に偶数パリティビットcを付加する回路はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　<img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23a.png" alt="選択肢アの図" width="160" height="99" ></div>
<div class="ap-choice-note">x1=1、x2=0、x3=1 を入力するとcが1になり誤りです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　<img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23i.png" alt="選択肢イの図" width="186" height="100" ></div>
<div class="ap-choice-note">第1の検証ではc=0だが、x1=x2=x3=1 のときcが0になり誤りです。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　<img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23u.png" alt="選択肢ウの図" width="161" height="100" ></div>
<div class="ap-choice-note">正しい。x1 XOR x2 XOR x3 により偶数<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>を求めます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　<img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23e.png" alt="選択肢エの図" width="187" height="100" ></div>
<div class="ap-choice-note">x1=1、x2=0、x3=1 を入力するとcが1になり誤りです。出力前のNOTにより奇数<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a>になります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>回路図中の各記号の意味は次の通りです。</p>
<p><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23.png" alt="解説図" width="273" height="174" ></p>
<p>偶数<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>は、1の<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>の数が偶数になるように<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>を付加する方式です。</p>
<p>例）</p>
<p>101 … 1の<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>が偶数個なので<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>は0 → 1010</p>
<p>100 … 1の<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>が奇数個なので<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>は1 → 1001</p>
<p>回路図の問題では、仮の値を回路に入力してみることで正しい構成になっているか確認するのが確実です。ここでは、x1=1、x2＝0、x3＝1 を使用します。出力cが0になれば正しい<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>が出力されたことになります。</p>
<p>cの値が1になるので誤りです。<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23aa.png" alt="解説図" width="160" height="115" ></p>
<p>cの値が0になので正解の可能性があります。<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23ii.png" alt="解説図" width="186" height="115" ></p>
<p>cの値が0になので正解の可能性があります。<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23uu.png" alt="解説図" width="161" height="115" ></p>
<p>cの値が1になるので誤りです。<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23ee.png" alt="解説図" width="187" height="115" ></p>
<p>さらに残った2つで検証を続けます。次に x1=1、x2＝1、x3＝1 を与えてみます。出力cが1になれば正しい<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>が出力されたことになります。</p>
<p>「イ」出力が0になるので誤りです。<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23iii.png" alt="解説図" width="186" height="115" ></p>
<p>「ウ」出力が1になるので正解です。<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/23/23uuu.png" alt="解説図" width="161" height="115" ></p>
<p>したがって消去法で「ウ」が正解とわかります。</p>
<p>一般的に、<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>列中の1の個数が奇数が偶数かを確認する方法としてXOR演算を使う方法があります。全ての<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>をXOR演算でつないだとき結果が1となれば1の個数は奇数、0であれば偶数となります。「ウ」の回路では、この性質を利用し「x1 XOR x2 XOR x3」を行うことで偶数<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>の値を求めています。なお、「ウ」の出力cの前に<a href="用語/NOT回路" class="internal-link" data-href="用語/NOT回路">NOT回路</a>を加えた「エ」は奇数<a href="用語/パリティ" class="internal-link" data-href="用語/パリティ">パリティ</a><a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>を付加する回路になります。</p>
