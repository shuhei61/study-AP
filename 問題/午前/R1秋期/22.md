<p class="ap-question"><strong>問22</strong>　エネルギーハーベスティングの適用例として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　AC電源で充電したバッテリで駆動される携帯電話機</div>
<div class="ap-choice-note">AC電源からの充電であり、環境からの微弱エネルギーを収穫していない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　インバータ制御を用いるエアーコンディショナの室外機</div>
<div class="ap-choice-note">商用電源を変換して利用しており、環境発電ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　スイッチを押す力を電力に変換して作動するRFリモコン</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/エネルギーハーベスティング" class="internal-link" data-href="用語/エネルギーハーベスティング">エネルギーハーベスティング</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　無停電電源装置を備えたデータサーバ</div>
<div class="ap-choice-note"><a href="用語/UPS" class="internal-link" data-href="用語/UPS">UPS</a>による予備電源であり、環境からのエネルギー収穫ではない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/エネルギーハーベスティング" class="internal-link" data-href="用語/エネルギーハーベスティング">エネルギーハーベスティング</a>は、周りの環境からの微弱なエネルギーを収穫(harvest)して電力に変換する技術の総称です。日本語では「環境発電技術」と呼ばれています。<a href="用語/エネルギーハーベスティング" class="internal-link" data-href="用語/エネルギーハーベスティング">エネルギーハーベスティング</a>を利用した技術としては以下の事例が挙げられます。</p>
<ul>
<li>ヒトやモノが移動する際の圧力を利用した電力床</li>
<li>放送波、無線<a href="用語/LAN" class="internal-link" data-href="用語/LAN">LAN</a>などの電磁波の回収による電力の取り出し</li>
<li>体温で発電するウェアラブル端末</li>
</ul>
<p>「ウ」の事例は、"圧力"という非電力エネルギーを"電力"に換えているので<a href="用語/エネルギーハーベスティング" class="internal-link" data-href="用語/エネルギーハーベスティング">エネルギーハーベスティング</a>の適用例に該当します。</p>
</div>
