<p class="ap-question"><strong>問71</strong>　RPA(Robotic Process Automation)の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　ホワイトカラーの単純な間接作業を，ルールエンジンや認知技術などを活用して代行するソフトウェア</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/RPA" class="internal-link" data-href="用語/RPA">RPA</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　自動制御によって，対象物をつかみ，動かす機能や，自動的に移動できる機能を有し，また，各種の作業をプログラムによって実行できる産業用ロボット</div>
<div class="ap-choice-note">これは<a href="用語/産業用ロボット" class="internal-link" data-href="用語/産業用ロボット">産業用ロボット</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　車両の状態や周囲の環境を認識し，利用者が行き先を指定するだけで自律的な走行を可能とするレーダー，GPS，カメラなどの自動運転関連機器</div>
<div class="ap-choice-note">これは<a href="用語/自動運転" class="internal-link" data-href="用語/自動運転">自動運転</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　人の生活と同じ空間で安全性を確保しながら，食事，清掃，移動，コミュニケーションなどの生活支援に使用されるロボット</div>
<div class="ap-choice-note">ハードウェアである生活支援ロボットの説明であり、ソフトウェアによる定型作業の自動化ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/RPA" class="internal-link" data-href="用語/RPA">RPA</a>(Robotic Process Automation)は、これまで人間が行ってきた定型的なパソコン操作を、コンピュータ上で動作するソフトウェアの力で自動化するものです。具体的には、ユーザー・インタフェース上の操作を認識する技術とワークフロー実行を組み合わせ、表計算ソフト、Webブラウザ、メールソフト、<a href="用語/ERP" class="internal-link" data-href="用語/ERP">ERP</a>（基幹業務システム）など複数のアプリケーションを使用する一連の業務プロセスをオートメーション化します。主に間接部門のPC作業を効率化するために使用されます。</p>
<p>日本の生産労働人口が減少局面にあるなか、従来より少ない人数で生産力を高めるための手段として<a href="用語/RPA" class="internal-link" data-href="用語/RPA">RPA</a>への注目が高まっています。</p>
</div>
