<p class="ap-question"><strong>問79</strong>　中小受託取引適正化法において，委託事業者の違法となる行為はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　支払期日を，発注したソフトウェアの受領後45日と決めた。</div>
<div class="ap-choice-note">支払期日は受領日から起算して60日以内で定める必要があり、受領後45日は問題ない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　ソフトウェア開発の発注書面を，了解を得て電子メールで送った。</div>
<div class="ap-choice-note">発注内容は書面または電磁的方法で明示でき、電子メールでの提供も認められる。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　納品され受領したソフトウェアの仕様を変更したいので，返品した。</div>
<div class="ap-choice-note">正しい。<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>に帰責事由がないのに返品・受領拒否することは禁止される。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　納品されるソフトウェアに契約の内容に適合しない不具合があるので，受領拒否した。</div>
<div class="ap-choice-note">契約不適合がある場合は受領を拒めるため、違法行為とはならない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/中小受託取引適正化法" class="internal-link" data-href="用語/中小受託取引適正化法">中小受託取引適正化法</a>（旧称：下請法）は、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>（発注側）と<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>（受注側）の公平な取引関係を確保し、受託側の利益保護と取引の公正化を図るための法律です。一定規模以上の<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>を「優越的地位にある」ものと位置付け、受託取引における<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>の不当な行為を規制することを狙いとしています。</p>
<p>具体的な規制内容として、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>に対し、発注内容等の明示、書類等の作成・保存、代金の支払期日の設定、遅延利息の支払といった義務を課すとともに、受領拒否、返品、代金減額、支払遅延など11類型の禁止行為を定めています。</p>
<p><strong>ア</strong><br>支払期日は、受領日から起算して60日以内のできる限り短い期間内で定める必要があります。本肢は受領後45日のため問題ありません。</p>
<p><strong>イ</strong><br><a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>に<a href="用語/製造委託" class="internal-link" data-href="用語/製造委託">製造委託</a>等をした<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>は、直ちに、所定の事項の書面または電磁的方法により明示しなければなりません。電磁的方法（電子メール・記録媒体の交付）での提供も認められます。なお、<a href="用語/中小受託取引適正化法" class="internal-link" data-href="用語/中小受託取引適正化法">中小受託取引適正化法</a>への改正により、事前の承諾は不要になっています。</p>
<p><strong>ウ</strong><br>正しい。<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>が<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>から納品を受けた際、<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>に帰責事由がないのに、給付の受領を拒むことは禁止されています。<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>の一方的な設計変更は、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>側の事情であり、それを理由とした受領拒否は認められません。よって、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>の違法行為となります。</p>
<p><strong>エ</strong><br><a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>は、正当な理由なく納品物の受領を拒むことはできませんが、①給付内容が契約書に明記された内容と異なる場合、②納品物に契約不適合等がある場合、③納期遅れの場合には受領を拒むことができます。本肢はソフトウェアに契約不適合があるため、受領を拒んでも違法行為とはなりません。</p>
</div>
