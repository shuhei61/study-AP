<p class="ap-question"><strong>問68</strong>　売手側でのマーケティング要素4Pは，買手側での要素4Cに対応するという考え方がある。4Pの一つであるプロモーションに対応する4Cの構成要素はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　顧客価値(Customer Value)</div>
<div class="ap-choice-note">製品（Product）に対応する4Cの要素です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　顧客コスト(Customer Cost)</div>
<div class="ap-choice-note">価格（Price）に対応する4Cの要素です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　コミュニケーション(Communication)</div>
<div class="ap-choice-note">正しい。プロモーション（Promotion）に対応する4Cの要素です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　利便性(Convenience)</div>
<div class="ap-choice-note">流通（Place）に対応する4Cの要素です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>売手側での<a href="用語/マーケティングの4P" class="internal-link" data-href="用語/マーケティングの4P">マーケティングの4P</a>とは次の4つです。</p>
<p>Product（製品、サービス、品質）<br>Price（価格、割引、コスパ）<br>Place（流通、立地、流通範囲、品揃え）<br>Promotion（販売促進、宣伝、広告）</p>
<p>4Pとは逆に、買手側の視点である<a href="用語/マーケティングの4C" class="internal-link" data-href="用語/マーケティングの4C">マーケティングの4C</a>は次の4つです。</p>
<p>Customer value（<a href="用語/顧客価値" class="internal-link" data-href="用語/顧客価値">顧客価値</a>）<br>Customer cost（顧客コスト）<br>Convenience（利便性）<br>Communication（コミュニケーション・対話）</p>
<p><a href="用語/マーケティングの4P" class="internal-link" data-href="用語/マーケティングの4P">マーケティングの4P</a>と<a href="用語/マーケティングの4C" class="internal-link" data-href="用語/マーケティングの4C">マーケティングの4C</a>はそれぞれ次のように対応しています。</p>
<p><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/68/68.png" alt="解説図" width="439" height="215"></p>
<p><a href="用語/マーケティングの4P" class="internal-link" data-href="用語/マーケティングの4P">マーケティングの4P</a>のプロモーションに対応する<a href="用語/マーケティングの4C" class="internal-link" data-href="用語/マーケティングの4C">マーケティングの4C</a>の要素は「コミュニケーション」です。したがって「ウ」が正解となります。</p>
</div>
