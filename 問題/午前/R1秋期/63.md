<p class="ap-question"><strong>問63</strong>　オープンデータの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　営利・非営利の目的を問わず二次利用が可能という利用ルールが定められており，編集や加工をする上で機械判読に適し，原則無償で利用できる形で公開された官民データ</div>
<div class="ap-choice-note">正しい。<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　行政事務の効率化・迅速化を目的に，国，地方自治体を相互に接続する行政専用のネットワークを通じて利用するアプリケーションシステム内に，安全に保管されたデータ</div>
<div class="ap-choice-note">総合行政ネットワーク(<a href="用語/LGWAN" class="internal-link" data-href="用語/LGWAN">LGWAN</a>)に蓄積されたデータです。<a href="用語/LGWAN" class="internal-link" data-href="用語/LGWAN">LGWAN</a>は閉域ネットワークですのでデータは公開されていません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　コンビニエンスストアチェーンの売上データや運輸業者の運送量データなど，事業運営に役立つデータであり，提供元が提供先を限定して販売しているデータ</div>
<div class="ap-choice-note">限定提供データの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　商用のDBMSに代わりオープンソースのDBMSを用いて蓄積されている企業内の基幹業務データ</div>
<div class="ap-choice-note">オープンソースの<a href="用語/DBMS" class="internal-link" data-href="用語/DBMS">DBMS</a>で蓄積したデータを<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>と呼ぶわけではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>は、国民の誰もがインターネットを通じて、目的を問わず容易に加工、編集、再配布等の無償利用ができるように公開されたデータです。国・地方公共団体等が<a href="用語/官民データ" class="internal-link" data-href="用語/官民データ">官民データ</a>活用推進基本法に基づいて公開しているデータや、民間企業が独自に公開しているデータがあります。</p>
<p>平成29年に策定された「<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>基本指針」では、<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>を以下のように定義しています。</p>
<p>「国、地方公共団体及び事業者が保有する<a href="用語/官民データ" class="internal-link" data-href="用語/官民データ">官民データ</a>のうち、国民誰もがインターネット等を通じて容易に利用（加工、編集、再配布等）できるよう、次のいずれの項目にも該当する形で公開されたデータを<a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>と定義する」</p>
<ul>
<li>営利目的、非営利目的を問わず二次利用可能なルールが適用されたもの</li>
<li>機械判読に適したもの</li>
<li>無償で利用できるもの</li>
</ul>
<p><a href="用語/オープンデータ" class="internal-link" data-href="用語/オープンデータ">オープンデータ</a>の活用により、人工知能関連技術、<a href="用語/IoT" class="internal-link" data-href="用語/IoT">IoT</a>活用関連技術、<a href="用語/クラウドコンピューティング" class="internal-link" data-href="用語/クラウドコンピューティング">クラウドコンピューティング</a>サービス関連技術等の先端的な技術の促進が期待されています。</p>
<p>したがって「ア」が正解です。</p>
