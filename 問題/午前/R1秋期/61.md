<p class="ap-question"><strong>問61</strong>　BCPの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　企業の戦略を実現するために，財務，顧客，内部ビジネスプロセス，学習と成長という四つの視点から戦略を検討したもの</div>
<div class="ap-choice-note">BSC(バランススコアカード)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　企業の目標を達成するために業務内容や業務の流れを可視化し，一定のサイクルをもって継続的に業務プロセスを改善するもの</div>
<div class="ap-choice-note">BPM(Business Process Management)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　業務効率の向上，業務コストの削減を目的に，業務プロセスを対象としてアウトソースを実施するもの</div>
<div class="ap-choice-note"><a href="用語/BPO" class="internal-link" data-href="用語/BPO">BPO</a>(Business Process Outsourcing)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　事業の中断・阻害に対応し，事業を復旧し，再開し，あらかじめ定められたレベルに回復するように組織を導く手順を文書化したもの</div>
<div class="ap-choice-note">正しい。<a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>(Business Continuity Plan，<a href="用語/事業継続計画" class="internal-link" data-href="用語/事業継続計画">事業継続計画</a>)は、予期せぬ重大<a href="用語/災害" class="internal-link" data-href="用語/災害">災害</a>が発生した場合に、必要最低限の事業を継続しつつ、業務を早期に<a href="用語/復旧" class="internal-link" data-href="用語/復旧">復旧</a>・再開できるようにするための行動計画です。近年、地震、火災・爆発、大規模なシステム<a href="用語/障害" class="internal-link" data-href="用語/障害">障害</a>などが相次いでおり、その結果、基幹となる事業・業務の停止に追い込まれるケースが見られることから<a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>の重要性が高まっています。</p>
<p>経済産業省や厚生労働省の資料では、<a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>の発動から全面<a href="用語/復旧" class="internal-link" data-href="用語/復旧">復旧</a>に至るまでを以下の4段階のフェーズに分けています。</p>
<p>1.<a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>発動フェーズ<br><a href="用語/災害" class="internal-link" data-href="用語/災害">災害</a>や事故の発生（或いは発生の可能性）を検知してから、初期対応を実施し、<a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>発動に至るまでのフェーズ</p>
<p>2.業務再開フェーズ<br><a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>を発動してから、<a href="用語/バックアップサイト" class="internal-link" data-href="用語/バックアップサイト">バックアップサイト</a>・手作業などの代替手段により業務を再開し、軌道に乗せるまでのフェーズ</p>
<p>3.業務回復フェーズ<br>最も緊急度の高い業務や機能が再開された後、さらに<a href="用語/復旧" class="internal-link" data-href="用語/復旧">復旧</a>する業務の範囲を拡大するフェーズ</p>
<p>4.全面<a href="用語/復旧" class="internal-link" data-href="用語/復旧">復旧</a>フェーズ<br>代替設備・手段から平常運用へ切り替えるフェーズ</p>
</div>
