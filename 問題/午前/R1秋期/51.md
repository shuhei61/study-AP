#問題 #マネジメント #プロジェクトマネジメント #プロジェクトマネジメント

<p class="ap-question"><strong>問51</strong>　PMBOKガイド第6版によれば，プロジェクト・マネジャー，プログラム・マネジャー，ポートフォリオ・マネジャー，プロジェクトマネジメント・オフィス(PMO)は，それぞれ他と異なる役割を担っている。それぞれに対応した役割の説明のうち，PMOのものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　戦略目標に整合させるよう，関連する複数のプロジェクトに影響する制約条件及びコンフリクトを解消する。</div>
<div class="ap-choice-note">これは<a href="用語/プログラムマネジメント" class="internal-link" data-href="用語/プログラムマネジメント">プログラムマネジメント</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　戦略目標を達成するために，プログラム及びプロジェクトの最適な組合せを選択して，構成要素の優先順位を決定し，必要な資源を提供する。</div>
<div class="ap-choice-note">これは<a href="用語/プロジェクトポートフォリオマネジメント" class="internal-link" data-href="用語/プロジェクトポートフォリオマネジメント">プロジェクトポートフォリオマネジメント</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　プロジェクトに関連するガバナンス・プロセスを標準化し，資源，方法論，ツール及び技法の共有を促進する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/PMO" class="internal-link" data-href="用語/PMO">PMO</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プロジェクトの要求事項を満たすために，知識，スキル，ツールと技法をプロジェクトのアクティビティへ適用する。</div>
<div class="ap-choice-note">これは<a href="用語/プロジェクトマネージャ" class="internal-link" data-href="用語/プロジェクトマネージャ">プロジェクトマネージャ</a>の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>プロジェクトマネジメントオフィス(<a href="用語/PMO" class="internal-link" data-href="用語/PMO">PMO</a>:Project Management Office)は、企業内で並行して実施されている個々のプロジェクトのマネジメント業務の支援、<a href="用語/プロジェクトマネージャ" class="internal-link" data-href="用語/プロジェクトマネージャ">プロジェクトマネージャ</a>のサポート、部門間の調整などプロジェクトが円滑に実施されるように支援を行う専門の部署です。<a href="用語/PMBOK" class="internal-link" data-href="用語/PMBOK">PMBOK</a>第6版では<a href="用語/PMO" class="internal-link" data-href="用語/PMO">PMO</a>を「プロジェクトに関連するガバナンス・プロセスを標準化し、資源、方法論、ルール及び技法の共有を促進する組織」と定義しています。</p>
<p><a href="用語/プロジェクトマネージャ" class="internal-link" data-href="用語/プロジェクトマネージャ">プロジェクトマネージャ</a>が特定のプロジェクト目標の達成に焦点を当て、最も良い方法で与えられた組織の資源をコントロールするのに対し、<a href="用語/PMO" class="internal-link" data-href="用語/PMO">PMO</a>はプロジェクト間の相互関係をマネジメントし、全てのプロジェクトに対して組織内の共有資源の使用を最適化します。</p>
<p>日本PMO協会によれば、一般的な<a href="用語/PMO" class="internal-link" data-href="用語/PMO">PMO</a>の業務内容は次のようなものです。</p>
<ul>
<li><a href="用語/プロジェクトマネジメント" class="internal-link" data-href="用語/プロジェクトマネジメント">プロジェクトマネジメント</a>方式の標準化</li>
<li><a href="用語/プロジェクトマネジメント" class="internal-link" data-href="用語/プロジェクトマネジメント">プロジェクトマネジメント</a>に関する研修など人材開発</li>
<li><a href="用語/プロジェクトマネジメント" class="internal-link" data-href="用語/プロジェクトマネジメント">プロジェクトマネジメント</a>業務の支援</li>
<li>プロジェクト間のリソースやコストの各種調整</li>
<li>個別企業に適応したプロジェクト環境の整備</li>
<li>その他付随するプロジェクト関連管理業務</li>
</ul>
<p>したがって「ウ」が正解です。</p>
