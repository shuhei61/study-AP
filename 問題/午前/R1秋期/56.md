<p class="ap-question"><strong>問56</strong>　ITIL 2011 editionによれば， 7ステップの改善プロセスにおけるa，b及びcの適切な組合せはどれか。<br><br>〔7ステップの改善プロセス〕<br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/56/56.png" alt="問題図" width="423" height="213" ><br><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/56/56a.png" alt="選択肢の組合せ表" width="510" height="135" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア</div>
<div class="ap-choice-note">a＝情報とデータを分析する、となっており、④処理に対応しません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ</div>
<div class="ap-choice-note">a＝情報とデータを分析する、b＝データを処理する、となっており、④処理と⑤分析の順と一致しません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ</div>
<div class="ap-choice-note">正しい。④処理→データを処理する、⑤分析→情報とデータを分析する、⑥提示→情報を提示して利用する、に対応します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ</div>
<div class="ap-choice-note">b＝情報を提示して利用する、c＝情報とデータを分析する、となっており、⑤分析と⑥提示の順と一致しません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ITIL" class="internal-link" data-href="用語/ITIL">ITIL</a> 2011 editionに記載されている「7ステップの改善プロセス」とは、改善を①特定、②定義、③収集、④処理、⑤分析、⑥提示、⑦実装するために必要な手順を定義および管理するプロセスのことです。</p>
<p>設問では、ステップ4の処理、ステップ5の分析、ステップ6の提示が空欄になっているので、それぞれに該当する作業を当てはめます。</p>
<ul>
<li>④処理 → データを処理する</li>
<li>⑤分析 → 情報とデータを分析する</li>
<li>⑥提示 → 情報を提示して利用する</li>
</ul>
<p>と対応するので、正しい組合せは「ウ」となります。</p>
</div>
