<p class="ap-question"><strong>問19</strong>　三つの媒体A～Cに次の条件でファイル領域を割り当てた場合，割り当てた領域の総量が大きい順に媒体を並べたものはどれか。<br><br>〔条件〕<br>ファイル領域を割り当てる際の媒体選択アルゴリズムとして，空き領域が最大の媒体を選択する方式を採用する。<br>割当て要求されるファイル領域の大きさは，順に90，30，40，40，70，30(Mバイト)であり，割当てられたファイル領域は，途中で解放されない。<br>各媒体は容量が同一であり，割当て要求に対して十分な大きさをもち，初めは全て空きの状態である。<br>空き領域の大きさが等しい場合には，A，B，Cの順に選択する。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　A，B，C</div>
<div class="ap-choice-note">総量は A＝90、B＝100、C＝110 なので、この並びにはならない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　A，C，B</div>
<div class="ap-choice-note">総量は A＝90、B＝100、C＝110 なので、この並びにはならない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　B，A，C</div>
<div class="ap-choice-note">総量は A＝90、B＝100、C＝110 なので、この並びにはならない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　C，B，A</div>
<div class="ap-choice-note">正しい。総量は A＝90、B＝100、C＝110 なので、大きい順は C，B，A。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>条件(1)～(4)を総合して考えると、割当て時点において使用量が最も少ない媒体に<a href="用語/ファイル" class="internal-link" data-href="用語/ファイル">ファイル</a>を割り当てていけばよいことがわかります。</p>
<p><a href="用語/ファイル" class="internal-link" data-href="用語/ファイル">ファイル</a>がどの媒体に割り当てられていくかを1つずつ順番に検討します。</p>
<p>90：空き領域がすべて同じなので、Aに割り当てます。<br>A(90), B(0), C(0)</p>
<p>30：BとCで空き領域が同じなので、Bに割り当てます。<br>A(90), B(30), C(0)</p>
<p>40：まだ使用していないCに割り当てます。<br>A(90), B(30), C(40)</p>
<p>40：使用量が最も少ないBに割り当てます。<br>A(90), B(30・40), C(40)</p>
<p>70：使用量が最も少ないCに割り当てます。<br>A(90), B(30・40), C(40・70)</p>
<p>30：使用量が最も少ないBに割り当てます。<br>A(90), B(30・40・30), C(40・70)</p>
<p>ここまでに割り当てられた<a href="用語/ファイル" class="internal-link" data-href="用語/ファイル">ファイル</a>の容量の総量は、A＝90、B＝100、C＝110なので、これを多い順番に並べると「C，B，A」になります。</p>
