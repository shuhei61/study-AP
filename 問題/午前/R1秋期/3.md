<p class="ap-question"><strong>問3</strong>　通信回線を使用したデータ伝送システムにM/M/1の待ち行列モデルを適用すると，平均回線待ち時間，平均伝送時間，回線利用率の関係は，次の式で表すことができる。<br>平均回線待ち時間＝平均伝送時間×<span class="frac"><span>回線利用率</span><span>1－回線利用率</span></span><br>回線利用率が0から徐々に増加していく場合，平均回線待ち時間が平均伝送時間よりも最初に長くなるのは，回線利用率が幾つを超えたときか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　0.4</div>
<div class="ap-choice-note">利用率0.4では回線利用率÷(1－回線利用率)＝0.4÷0.6＜1となり、平均回線待ち時間はまだ平均伝送時間より短い。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　0.5</div>
<div class="ap-choice-note">正しい。利用率0.5でちょうど1となり、これより大きいと平均回線待ち時間が平均伝送時間より長くなる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　0.6</div>
<div class="ap-choice-note">0.6ではすでに1を超えているが、「最初に」長くなる閾値ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　0.7</div>
<div class="ap-choice-note">0.7ではすでに1を超えているが、「最初に」長くなる閾値ではない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>M/M/1の<a href="用語/待ち行列モデル" class="internal-link" data-href="用語/待ち行列モデル">待ち行列モデル</a>では、平均回線待ち時間＝平均伝送時間×<span class="frac"><span>回線利用率</span><span>1－回線利用率</span></span>で表せます。平均回線待ち時間が平均伝送時間より長くなるには、<span class="frac"><span>回線利用率</span><span>1－回線利用率</span></span>が1より大きくなることが条件です。<span class="frac"><span>回線利用率</span><span>1－回線利用率</span></span>に回線利用率を当てはめていくと、回線利用率が50％(0.5)のときにちょうど1になり、50%よりも大きくなると1を超えて、平均回線待ち時間が平均伝送時間より長くなることがわかります。<br><br>したがって正解は0.5です。</p>
</div>
