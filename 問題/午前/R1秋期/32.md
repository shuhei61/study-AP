<p class="ap-question"><strong>問32</strong>　イーサネットで使用されるメディアアクセス制御方式であるCSMA/CDに関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　それぞれのステーションがキャリア検知を行うとともに，送信データの衝突が起きた場合は再送する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/CSMA/CD" class="internal-link" data-href="用語/CSMA/CD">CSMA/CD</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　タイムスロットと呼ばれる単位で分割して，同一周波数において複数の通信を可能にする。</div>
<div class="ap-choice-note">これは<a href="用語/TDMA" class="internal-link" data-href="用語/TDMA">TDMA</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　データ送受信の開始時にデータ送受信のネゴシエーションとしてRTS/CTS方式を用い，受信の確認はACKを使用する。</div>
<div class="ap-choice-note">これは<a href="用語/CSMA/CA" class="internal-link" data-href="用語/CSMA/CA">CSMA/CA</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　伝送路上にトークンを巡回させ，トークンを受け取った端末だけがデータを送信できる。</div>
<div class="ap-choice-note">これは<a href="用語/トークンパッシング" class="internal-link" data-href="用語/トークンパッシング">トークンパッシング</a>方式の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/CSMA/CD" class="internal-link" data-href="用語/CSMA/CD">CSMA/CD</a>方式は、Carrier Sense Multiple Access with Collision Detectionの略で、日本語に訳すと「搬送波感知多重アクセス／<a href="用語/衝突" class="internal-link" data-href="用語/衝突">衝突</a>検出方式」となります。<a href="用語/CSMA/CD" class="internal-link" data-href="用語/CSMA/CD">CSMA/CD</a>方式は一本の<a href="用語/同軸ケーブル" class="internal-link" data-href="用語/同軸ケーブル">同軸ケーブル</a>に複数のコンピュータが接続されている<a href="用語/バス型" class="internal-link" data-href="用語/バス型">バス型</a>のトポロジを前提としており、次のような手順で送信が行われます。</p>
<p>伝送路上に他のノードから<a href="用語/フレーム" class="internal-link" data-href="用語/フレーム">フレーム</a>が送出されていないかを確認する</p>
<p>複数のクライアントは同じ回線を共用し、他者が通信をしていなければ自分の通信を開始する</p>
<p>複数の通信が同時に行われた場合は<a href="用語/衝突" class="internal-link" data-href="用語/衝突">衝突</a>を検出し、送信を中止してランダム時間待ってから再び送信をする</p>
<p>上記のように伝送路上での<a href="用語/衝突" class="internal-link" data-href="用語/衝突">衝突</a>を許容する仕組みとなっていて、制御が簡単な割に伝送効率が良いという特徴があります。</p>
</div>
