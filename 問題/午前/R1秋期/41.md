<p class="ap-question"><strong>問41</strong>　内部ネットワークのPCからインターネット上のWebサイトを参照するときに，DMZに設置したVDI(Virtual Desktop Infrastructure)サーバ上のWebブラウザを利用すると，未知のマルウェアがPCにダウンロードされるのを防ぐというセキュリティ上の効果が期待できる。この効果を生み出すVDIサーバの動作の特徴はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　Webサイトからの受信データのうち，実行ファイルを削除し，その他のデータをPCに送信する。</div>
<div class="ap-choice-note">実行ファイルの削除は<a href="用語/コンテンツフィルタリング" class="internal-link" data-href="用語/コンテンツフィルタリング">コンテンツフィルタリング</a>の動作であり、<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>の特徴ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　Webサイトからの受信データは，IPsecでカプセル化し，PCに送信する。</div>
<div class="ap-choice-note"><a href="用語/IPsec" class="internal-link" data-href="用語/IPsec">IPsec</a>による<a href="用語/カプセル化" class="internal-link" data-href="用語/カプセル化">カプセル化</a>は<a href="用語/VPN" class="internal-link" data-href="用語/VPN">VPN</a>の動作であり、<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>の特徴ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　Webサイトからの受信データは，受信処理ののち生成したデスクトップ画面の画像データだけをPCに送信する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　Webサイトからの受信データは，不正なコード列が検知されない場合だけPCに送信する。</div>
<div class="ap-choice-note">不正コードの検知は<a href="用語/マルウェア対策ソフト" class="internal-link" data-href="用語/マルウェア対策ソフト">マルウェア対策ソフト</a>等の動作であり、<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>の特徴ではない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>(Virtual Desktop Infrastructure)は，サーバ内にクライアントごとの仮想マシンを用意して仮想デスクトップ環境を構築する技術です。利用者はネットワークを通じて<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>サーバ上の仮想デスクトップ環境に接続し、クライアントPCには<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>サーバからの操作結果画面のみが転送される仕組みになっています。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/41/41.png" alt="解説図" width="433" height="290" ></p>
<p>この仕組みにより、クライアントがインターネット上のサイトと直接的な通信を行わなくなるので、クライアントPCをインターネットから分離できます。もし利用者の操作により不正なマルウェアをダウンロードしてしまったとしても、それが保存されるのは<a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>サーバ上の<a href="用語/仮想環境" class="internal-link" data-href="用語/仮想環境">仮想環境</a>ですので、クライアントPCへの感染を防げます。汚染された<a href="用語/仮想環境" class="internal-link" data-href="用語/仮想環境">仮想環境</a>を削除してしまえば内部ネットワークへの影響もありません。</p>
<p><a href="用語/VDI" class="internal-link" data-href="用語/VDI">VDI</a>サーバからPCに送信されるのは「デスクトップ画面の画像データ」のみです。したがって正解は「ウ」です。</p>
</div>
