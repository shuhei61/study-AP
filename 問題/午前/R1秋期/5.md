<p class="ap-question"><strong>問5</strong>　組込みシステムにおけるリアルタイムシステムにおいて，システムへの入力に対する応答のうち，最も適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　OSを使用しないで応答する。</div>
<div class="ap-choice-note">リアルタイムシステムでは、時間資源の保護や実行時間の予測可能性の提供に特化したRTOS(Real-time OS)により管理されます。RTOSなしにリアルタイム性を確保するのは困難です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　定められた制限時間内に応答する。</div>
<div class="ap-choice-note">正しい。リアルタイムシステムでは、設定されている制限時間内に処理を終了することが要求されます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　入力された順序を守って応答する。</div>
<div class="ap-choice-note">リアルタイムシステムでは、処理の順序よりも時間的制約の遵守が重要です。緊急性の高い処理を優先するため、到着順方式ではなく優先度方式の<a href="用語/タスク" class="internal-link" data-href="用語/タスク">タスク</a><a href="用語/スケジューリング" class="internal-link" data-href="用語/スケジューリング">スケジューリング</a>が用いられます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　入力時刻を記録して応答する。</div>
<div class="ap-choice-note">「リアルタイム」というのは時刻を正しく記録することではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/リアルタイム処理" class="internal-link" data-href="用語/リアルタイム処理">リアルタイム処理</a>は、使用可能な資源が制限されている状態において、入力や<a href="用語/イベント" class="internal-link" data-href="用語/イベント">イベント</a>に対し、指定された時間内（デッドライン）に結果を出力することに着目した制御工学の概念です。「即時処理」とも呼ばれます。<a href="用語/リアルタイム処理" class="internal-link" data-href="用語/リアルタイム処理">リアルタイム処理</a>を実現するためのシステムが、リアルタイムシステム(Real-time System)です。</p>
<p>リアルタイムシステムでは、設定されている制限時間内に処理を終了することが要求されます。例として、自動車のエアバック制御、交通管制、医療機器、産業用ロボットの制御などがあります。いずれも正しく動作するだけでは不十分で、指定された時間内に処理を終えることが極めて重要なシステムです。</p>
<p>したがって「イ」が正解です。</p>
</div>
