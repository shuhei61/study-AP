<p class="ap-question"><strong>問1</strong>　あるホテルは客室を1,000部屋もち，部屋番号は，数字4と9を使用しないで0001から順に数字4桁の番号としている。部屋番号が0330の部屋は，何番目の部屋か。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　204</div>
<div class="ap-choice-note">8進数0330を10進数に直すと216であり、204ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　210</div>
<div class="ap-choice-note">8進数0330を10進数に直すと216であり、210ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　216</div>
<div class="ap-choice-note">正しい。部屋番号0330を8進数とみなして10進数に変換すると216になります。詳細：<a href="用語/基数" class="internal-link" data-href="用語/基数">基数</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　218</div>
<div class="ap-choice-note">8進数0330を10進数に直すと216であり、218ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>数字4と9を使用しないということは、0～3、5～8の8種類の数字のみを使うということです。1つの桁に8種類の数字を使えるということは、1つの桁に0～7を使う8進数と同じ考え方を適用できることになります。</p>
<p>つまり、部屋番号の数字を以下のように対応させて10進数に変換すれば、部屋番号の0001から数えて何番目の部屋なのかを素早く計算できます。</p>
<p><img class="ap-fig" src="../../../assets/問題/午前/R1秋期/1/01.png" alt="解説図" width="271" height="69" ></p>
<p>対象の部屋番号は0330なので数字の置き換えは発生せず、そのまま8進数を10進数に変換する計算を行えば大丈夫です。<br>8<sup>2</sup>×3＋8×3＝192＋24＝216</p>
<p>したがって「ウ」が正解です。</p>
