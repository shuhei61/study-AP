<p class="ap-question"><strong>問47</strong>　フェールセーフの考えに基づいて設計したものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　乾電池のプラスとマイナスを逆にすると，乾電池が装填できないようにする。</div>
<div class="ap-choice-note">これは<a href="用語/フールプルーフ" class="internal-link" data-href="用語/フールプルーフ">フールプルーフ</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　交通管制システムが故障したときには，信号機に赤色が点灯するようにする。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/フェールセーフ" class="internal-link" data-href="用語/フェールセーフ">フェールセーフ</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ネットワークカードのコントローラーを二重化しておき，片方のコントローラーが故障しても運用できるようにする。</div>
<div class="ap-choice-note">これはフォールトトレランスの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ハードディスクにRAID1を採用して，MTBFで示される信頼性が向上するようにする。</div>
<div class="ap-choice-note"><a href="用語/RAID1" class="internal-link" data-href="用語/RAID1">RAID1</a>はディスク<a href="用語/ミラーリング" class="internal-link" data-href="用語/ミラーリング">ミラーリング</a>。フォールトトレランスの考え方です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/フェールセーフ" class="internal-link" data-href="用語/フェールセーフ">フェールセーフ</a>(Fail safe)は、システムの不具合や<a href="用語/故障" class="internal-link" data-href="用語/故障">故障</a>が発生したときに、安全を最優先に考えて動作を制御し、問題の影響をできるだけ小さく抑える設計方針のことです。次のような安全優先の設計方針がこれに該当します。</p>
<ul>
<li>工業用機械で進入禁止区域をセンサーで監視し、人や物の侵入を感知したときには機械を緊急停止する</li>
<li>信号機が<a href="用語/故障" class="internal-link" data-href="用語/故障">故障</a>したとき、交差点内の信号機を全て赤信号にして侵入を止める</li>
<li>電気回路に過電流が流れたときにはヒューズやブレーカが落ちる</li>
</ul>
<p>したがって「イ」が正解です。</p>
</div>
