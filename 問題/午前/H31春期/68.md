<p class="ap-question"><strong>問68</strong>　バランススコアカードの四つの視点とは，財務，学習と成長，内部ビジネスプロセスと，もう一つはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ガバナンス</div>
<div class="ap-choice-note">これは<a href="用語/コーポレートガバナンス" class="internal-link" data-href="用語/コーポレートガバナンス">コーポレートガバナンス</a>の説明です。BSCの四つの視点には含まれません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　顧客</div>
<div class="ap-choice-note">正しい。BSCの四視点のうち「<a href="用語/顧客の視点" class="internal-link" data-href="用語/顧客の視点">顧客の視点</a>」に対応します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　自社の強み</div>
<div class="ap-choice-note">自社の強みはSWOT分析などで扱う要素であり、BSCの四つの視点ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　遵法</div>
<div class="ap-choice-note">これは<a href="用語/コンプライアンス" class="internal-link" data-href="用語/コンプライアンス">コンプライアンス</a>（法令遵守）の説明です。BSCの四つの視点には含まれません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>バランススコアカード（BSC: Balanced Score Card）は、企業のビジョンと戦略を実現するために、「財務」「顧客」「内部ビジネスプロセス」「学習と成長」という4つの視点から業績を評価・分析する手法です。したがって残る一つは「イ」の顧客になります。</p>
<p>各視点の概要は以下の通りです。</p>
<p><strong><a href="用語/財務の視点" class="internal-link" data-href="用語/財務の視点">財務の視点</a></strong> … 株主や従業員などの利害関係者の期待に応えて良い業績を出すために、財務的にどのように行動すべきかの指標を設定する。</p>
<p><strong><a href="用語/顧客の視点" class="internal-link" data-href="用語/顧客の視点">顧客の視点</a></strong> … 企業のビジョンを達成するために、顧客に対してどのように行動すべきかの指標を設定する。</p>
<p><strong><a href="用語/業務プロセスの視点" class="internal-link" data-href="用語/業務プロセスの視点">業務プロセスの視点</a></strong>（内部ビジネスプロセス） … 財務的目標の達成や顧客満足度を向上させるために、優れた業務プロセスを構築するための指標を設定する。</p>
<p><strong><a href="用語/学習と成長の視点" class="internal-link" data-href="用語/学習と成長の視点">学習と成長の視点</a></strong> … 企業のビジョンを達成するために、組織や個人としてどのように変化（改善）し能力向上を図るかの指標を設定する。</p>
</div>
