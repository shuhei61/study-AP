<p class="ap-question"><strong>問17</strong>　プログラムの実行時に利用される記憶領域にスタック領域とヒープ領域がある。それらの領域に関する記述のうち，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　サブルーチンからの戻り番地の退避にはスタック領域が使用され，割当てと解放の順序に関連がないデータにはヒープ領域が使用される。</div>
<div class="ap-choice-note">正しい。戻り番地の退避は<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>、順序に無関係な割当てはヒープ。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　スタック領域には未使用領域が存在するが，ヒープ領域には未使用領域は存在しない。</div>
<div class="ap-choice-note">ヒープの初期状態は未使用であり、動的確保により使用領域と未使用領域に分かれる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ヒープ領域はスタック領域の予備領域であり，スタック領域が一杯になった場合にヒープ領域が動的に使用される。</div>
<div class="ap-choice-note">ヒープと<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>の役割は異なり、予備領域ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ヒープ領域も構造的にはスタックと同じプッシュとポップの操作によって，データの格納と取出しを行う。</div>
<div class="ap-choice-note">ヒープ領域は<a href="用語/双方向リスト" class="internal-link" data-href="用語/双方向リスト">双方向リスト</a>によって構成される。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>領域とヒープ領域の違いについては以下の通りです。</p>
<p><strong><a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>領域</strong><br>一般にコールスタック・制御スタックと呼ばれている。LIFO方式で構成され、プログラムの実行中<a href="用語/サブルーチン" class="internal-link" data-href="用語/サブルーチン">サブルーチン</a>の情報を記憶しておくメモリ領域。<a href="用語/サブルーチン" class="internal-link" data-href="用語/サブルーチン">サブルーチン</a>終了後の戻りアドレスや局所変数などを保持する。</p>
<p><strong>ヒープ領域</strong><br>2つのラベルを持つ<a href="用語/双方向リスト" class="internal-link" data-href="用語/双方向リスト">双方向リスト</a>で構成され、プログラム上から動的（任意）に確保できるメモリ領域。動的にメモリ取得・解放を繰り返すことにより、メモリ上にどこからも参照されない領域（ガベージ）が発生する。</p>
<p>したがって正しい説明は「ア」です。</p>
