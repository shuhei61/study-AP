<p class="ap-question"><strong>問51</strong>　JIS Q 21500:2018(プロジェクトマネジメントの手引)によれば，プロジェクトマネジメントの"実行のプロセス群"の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　プロジェクトの計画に照らしてプロジェクトパフォーマンスを監視し，測定し，管理するために使用する。</div>
<div class="ap-choice-note">これは<a href="用語/管理のプロセス群" class="internal-link" data-href="用語/管理のプロセス群">管理のプロセス群</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　プロジェクトフェーズ又はプロジェクトが完了したことを正式に確定するために使用し，必要に応じて考慮し，実行するように得た教訓を提供するために使用する。</div>
<div class="ap-choice-note">これは<a href="用語/終結のプロセス群" class="internal-link" data-href="用語/終結のプロセス群">終結のプロセス群</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　プロジェクトフェーズ又はプロジェクトを開始するために使用し，プロジェクトフェーズ又はプロジェクトの目標を定義し，プロジェクトマネージャがプロジェクト作業を進める許可を得るために使用する。</div>
<div class="ap-choice-note">これは<a href="用語/立ち上げのプロセス群" class="internal-link" data-href="用語/立ち上げのプロセス群">立ち上げのプロセス群</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　プロジェクトマネジメントの活動を遂行し，プロジェクトの全体計画に従ってプロジェクトの成果物の提示を支援するために使用する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/実行のプロセス群" class="internal-link" data-href="用語/実行のプロセス群">実行のプロセス群</a></div>
</li>
</ul>
