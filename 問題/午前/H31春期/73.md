<p class="ap-question"><strong>問73</strong>　SEOの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ECサイトにおいて，個々の顧客の購入履歴を分析し，新たに購入が見込まれる商品を自動的に推奨する機能</div>
<div class="ap-choice-note">これは<a href="用語/レコメンデーション" class="internal-link" data-href="用語/レコメンデーション">レコメンデーション</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　Webページに掲載した広告が契機となって商品が購入された場合，売主から成功報酬が得られる仕組み</div>
<div class="ap-choice-note">これは<a href="用語/アフィリエイト" class="internal-link" data-href="用語/アフィリエイト">アフィリエイト</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　検索エンジンの検索結果一覧において自社サイトがより上位にランクされるようにWebページの記述内容を見直すなど様々な試みを行うこと</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/SEO" class="internal-link" data-href="用語/SEO">SEO</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　検索エンジンを運営する企業と契約し，自社の商品・サービスと関連したキーワードが検索に用いられた際に広告を表示する仕組み</div>
<div class="ap-choice-note">これは検索連動型広告(<a href="用語/リスティング広告" class="internal-link" data-href="用語/リスティング広告">リスティング広告</a>)の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/SEO" class="internal-link" data-href="用語/SEO">SEO</a>(Search Engine Optimization)とは、Webサイトを制作するときに、Googleなどの検索エンジンの検索で上位に表示されることを目指してページやサイト全体を最適化することをいいます。</p>
<p>具体的な施策には、上位表示を狙うキーワードを意識した文章にする、Webサイトの更新頻度を増やす、良質なコンテンツページを増やす、外部のWebサイトからのリンクを増やす、ページの表示速度を速くする、<a href="用語/ユーザビリティ" class="internal-link" data-href="用語/ユーザビリティ">ユーザビリティ</a>、<a href="用語/アクセシビリティ" class="internal-link" data-href="用語/アクセシビリティ">アクセシビリティ</a>に配慮した設計を行うなどがあり、検索エンジンがランキングを決定する際のシグナルとして用いられていると考えられている要素を強化し、検索アルゴリズムで高評価を得ることを目標とします。</p>
<p>検索結果における1ページ目(1～10位)と2ページ目(11～20位)では、5倍以上の集客数の差が生じるといわれており、ECサイトにとっては上位に表示されることが売上の増加に直結するため、<a href="用語/SEO" class="internal-link" data-href="用語/SEO">SEO</a>はWebマーケティングにとって重要な意味を持っています。</p>
