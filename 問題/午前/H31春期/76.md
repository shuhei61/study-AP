<p class="ap-question"><strong>問76</strong>　表の事業計画案に対して，新規設備投資に伴う減価償却費(固定費)の増加1,000万円を織り込み，かつ，売上総利益を3,000万円とするようにしたい。変動費率に変化がないとすると，売上高の増加を何万円にすればよいか。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/76/76.png" alt="問題図" width="238" height="189" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　2,000</div>
<div class="ap-choice-note">必要売上高と計画案売上高の差から求まる増加額ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　3,000</div>
<div class="ap-choice-note">必要売上高と計画案売上高の差から求まる増加額ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　4,000</div>
<div class="ap-choice-note">正しい。必要売上高24,000万円から計画案の20,000万円を差し引くと4,000万円です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　5,000</div>
<div class="ap-choice-note">必要売上高と計画案売上高の差から求まる増加額ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>まず、必要な売上高を求めるのに必要な数値を以下のように求めます。</p>
<ul>
<li>変動費率 … 10,000÷20,000＝0.5</li>
<li>固定費は … 8,000＋(増加分)1,000＝9,000</li>
</ul>
<p>上記の変動費率と固定費をもとに、「売上高＝変動費＋固定費＋売上総利益」の関係を使って計算します。<br><br>売上高を a とおくと、<br><br>　a＝0.5a＋9,000＋3,000<br>　0.5a＝12,000<br>　a＝24,000(万円)<br><br>なので、3,000万円の売上総利益を得るためには24,000万円の売上高が必要であることになります。<br><br>現在の事業計画案では20,000万円の売上高を予定しているため、<br><br>　24,000－20,000＝4,000(万円)<br><br>4,000万円の売上高の増加が必要となります。<br><br>また、<a href="用語/損益分岐点" class="internal-link" data-href="用語/損益分岐点">損益分岐点</a>売上高の公式とは別に「目標利益達成売上高」の公式というものがあります。<br><br>　目標利益達成売上高＝(固定費＋目標利益)÷(1－変動費率)<br><br><a href="用語/損益分岐点" class="internal-link" data-href="用語/損益分岐点">損益分岐点</a>を求める公式「固定費÷(1－変動費率)」の固定費の部分に目標利益を組み入れた式となっています。これを利用して、<br><br>　必要売上高＝(9,000＋3,000)÷0.5＝24,000(万円)<br>　売上高の増加＝24,000－20,000＝4,000(万円)<br><br>と求める方法もあります。</p>
