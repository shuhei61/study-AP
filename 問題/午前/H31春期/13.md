<p class="ap-question"><strong>問13</strong>　稼働率の等しい装置を直列や並列に組み合わせたとき，システム全体の稼働率を高い順に並べたものはどれか。ここで，各装置の稼働率は0より大きく1未満である。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/13/13.png" alt="問題図" width="464" height="63" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　A，B，C</div>
<div class="ap-choice-note">Bの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>はAより低く、CはBより高いため、この順序は正しくありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　A，C，B</div>
<div class="ap-choice-note">正しい。Aが最も高く、次いでC、Bの順になります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　C，A，B</div>
<div class="ap-choice-note">Aの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>はCより高いため、この順序は正しくありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　C，B，A</div>
<div class="ap-choice-note">Aの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>が最も高いため、この順序は正しくありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>各装置の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>は等しく、0より大きく1未満とあるので、仮に<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>0.9を当てはめてみて、システム全体の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>を比較します。直列接続の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>を求める式はR<sup>2</sup>、並列接続の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>を求める式は「1－(1－R)<sup>2</sup>」です。</p>
<p>【A】<br>2台の並列接続なので、<br>1－(1－0.9)<sup>2</sup><br>＝1－0.1<sup>2</sup><br>＝1－0.01＝0.99</p>
<p>【B】<br><a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>0.99であるAの構成部分に、装置1台が直列接続されているので、<br>0.9×0.99＝0.891</p>
<p>【C】<br>装置2台の直列接続は、0.9×0.9＝0.81<br><a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>0.81の構成部分が、並列に接続されているので、<br>1－(1－0.81)<sup>2</sup><br>＝1－0.19<sup>2</sup><br>＝1－0.0361＝0.9639</p>
<p>したがって、<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>の高い順に「A，C，B」となります。</p>
<p>【別解】<br>各システムの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>を式で表して、それを比較する論理的な解法です。各装置の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>をRとします。</p>
<p>Aの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>：1－(1－R)<sup>2</sup> … aとする<br>Bの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>：R×[Aのシステム構成]なので、Ra<br>Cの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>：1－(1－R<sup>2</sup>)<sup>2</sup></p>
<p>それぞれの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>の大小を比較します。</p>
<p>AとB：0＜R＜1なので「a＞Ra」、よってA＞B</p>
<p>AとC：1から減じる部分を比較すると、0＜R＜1なので「(1－R)＜(1－R<sup>2</sup>)」、よってA＞C</p>
<p>BとC：Bの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a> Ra を展開すると、<br>R(1－(1－R)<sup>2</sup>)＝R(1－(1－2R＋R<sup>2</sup>))<br>＝R(1－1＋2R－R<sup>2</sup>)＝R(2R－R<sup>2</sup>)<br>＝R<sup>2</sup>(2－R)<br><br>Cの<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>を展開すると、<br>1－(1－R<sup>2</sup>)<sup>2</sup>＝1－(1－2R<sup>2</sup>＋R<sup>4</sup>)<br>＝1－1＋2R<sup>2</sup>－R<sup>4</sup>＝2R<sup>2</sup>－R<sup>4</sup><br>＝R<sup>2</sup>(2－R<sup>2</sup>)<br><br>R<sup>2</sup>に乗ずる部分を比較すると、0＜R＜1なので「2－R＜2－R<sup>2</sup>」、よってB＜C</p>
<p>以上より、3つのシステム構成を<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>が高い順に並べると「A，C，B」になります。</p>
