<p class="ap-question"><strong>問80</strong>　技術者倫理の観点から，職務遂行において技術者が優先すべきこととして，最も適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　会社の利益</div>
<div class="ap-choice-note">公衆の安全と対立する場合でも，会社の利益を最優先にはしない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　技術者個人の名誉</div>
<div class="ap-choice-note">技術者個人の名誉は，職務遂行で最優先すべき事項ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　公衆の安全</div>
<div class="ap-choice-note">正しい。<a href="用語/技術者の倫理綱領" class="internal-link" data-href="用語/技術者の倫理綱領">技術者の倫理綱領</a>では公衆の安全を最優先に考慮する。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　コストの低減</div>
<div class="ap-choice-note">コストの低減は，公衆の安全より優先すべき事項ではない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>日本技術士協会の<a href="用語/技術者の倫理綱領" class="internal-link" data-href="用語/技術者の倫理綱領">技術者の倫理綱領</a>では、1つ目に「技術士は、公衆の安全、健康及び福利を最優先に考慮する」と明記しています。つまり、技術者が職務遂行する上で最も優先されることは公衆の安全です。その他の事項と"公衆の安全"が対抗する場合もありますが、その際には必ず公衆の安全を優先するのが技術者としての適切な態度です。</p>
<p>また、技術士法45条の2では「技術士又は技術士補は、その業務を行うに当たつては、公共の安全、環境の保全その他の公益を害することのないよう努めなければならない」と規定しており、公益確保の責務は法的にも義務付けられています。</p>
<p>したがって「ウ」が正解です。</p>
