<p class="ap-question"><strong>問62</strong>　A社は，ソリューションプロバイダから，顧客に対するワントゥワンマーケティングを実現する統合的なソリューションの提案を受けた。この提案が該当するソリューションとして，最も適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　CRMソリューション</div>
<div class="ap-choice-note">正しい。<a href="用語/CRM" class="internal-link" data-href="用語/CRM">CRM</a>は、連絡先情報、購買履歴、コンタクト履歴などを記録し、顧客との関係を効率的に管理するためのソリューションです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　HRMソリューション</div>
<div class="ap-choice-note">Human Resource Managementの略。人事資源管理のためのソリューションです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　SCMソリューション</div>
<div class="ap-choice-note">Supply Chain Managementの略。生産・在庫・購買・販売・物流などの一連の業務の流れを一元管理するためのソリューションです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　財務管理ソリューション</div>
<div class="ap-choice-note">会計業務を管理するためのソリューションです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ワントゥワンマーケティング" class="internal-link" data-href="用語/ワントゥワンマーケティング">ワントゥワンマーケティング</a>とは、顧客1人ひとりの嗜好やニーズに合わせて個別に対応を変化させて展開されるマーケティング活動です。市場シェアの拡大よりも、顧客との好ましい関係を築き、長期にわたって自社製品を購入する顧客の割合を高めることに重点を置くことで、1人の顧客から得られる生涯利益（<a href="用語/LTV" class="internal-link" data-href="用語/LTV">LTV</a>：Life Time Value）を最大化することを目的としています。</p>
<p><a href="用語/ワントゥワンマーケティング" class="internal-link" data-href="用語/ワントゥワンマーケティング">ワントゥワンマーケティング</a>を展開するには顧客1人ひとりの詳細かつ網羅的な情報が欠かせません。<a href="用語/CRM" class="internal-link" data-href="用語/CRM">CRM</a>(Customer Relationship Management)システムは、顧客に関するあらゆる情報を統合管理して企業活動に役立てる経営システムですので、<a href="用語/ワントゥワンマーケティング" class="internal-link" data-href="用語/ワントゥワンマーケティング">ワントゥワンマーケティング</a>の目的に合致しています。</p>
<p>したがって「ア」が正解です。</p>
