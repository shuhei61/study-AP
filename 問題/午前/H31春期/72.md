<p class="ap-question"><strong>問72</strong>　PDM(Product Data Management)の目的はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　NC工作機械，自動搬送装置，倉庫などを有機的に結合し，コンピュータで集中管理することによって，多品種少量生産に対応できる生産の自動化を実現する。</div>
<div class="ap-choice-note">これは<a href="用語/FMS" class="internal-link" data-href="用語/FMS">FMS</a>(Flexible Manufacturing System)の目的です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　図面・部品構成データ，仕様書データなどの設計及び開発の段階で発生する情報を一元管理することによって，設計業務及び開発業務の効率を向上させる。</div>
<div class="ap-choice-note">正しい。<a href="用語/PDM" class="internal-link" data-href="用語/PDM">PDM</a>の目的です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　製品の生産計画に基づいてその生産に必要な資材の所要量を展開し，これを基準にして資材の需要とその発注時期を算出する。</div>
<div class="ap-choice-note">これは<a href="用語/MRP" class="internal-link" data-href="用語/MRP">MRP</a>(Material Requirements Planning)の目的です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　部品の供給から製品の販売までの一連のプロセスの情報をリアルタイムで交換することによって，在庫の削減とリードタイムの短縮を実現する。</div>
<div class="ap-choice-note">これは<a href="用語/SCM" class="internal-link" data-href="用語/SCM">SCM</a>(Supply Chain Management)の目的です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/PDM" class="internal-link" data-href="用語/PDM">PDM</a>(Product Data Management)は、直訳すれば「製造情報管理」という意味で、設計図面、部品構成表、コスト情報、技術仕様などの設計・開発に関連する様々な情報をシステムで一元管理する考え方です。これまで各部門に散在していたデータが共有されることにより、開発期間の短縮や開発コスト削減などの効率化が期待できます。</p>
</div>
</div>
