<p class="ap-question"><strong>問23</strong>　次の二つの回路の入力に値を与えたとき，表の入力A，B，C，Dと出力E，Fの組合せのうち，全ての素子が論理積素子で構成された左側の回路でだけ成立するものはどれか。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23.png" alt="問題図" width="449" height="116" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　<img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23a.png" alt="選択肢アの図" width="224" height="90" ></div>
<div class="ap-choice-note">両方で成立します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　<img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23i.png" alt="選択肢イの図" width="223" height="90" ></div>
<div class="ap-choice-note">右側の回路でだけ成立します。</div>
</li>
<li class="ap-choice-item ap-correct">

<div class="ap-choice-note">正しい。左側の回路でだけ成立します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　<img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23e.png" alt="選択肢エの図" width="224" height="90" ></div>
<div class="ap-choice-note">両方で成立します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>は、2つの入力がともに1のときにだけ1を出力し、そうでなければ0を出力します。<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>は、2つの入力のうち少なくとも一方が1のときに1を出力し、そうでなければ0を出力する回路です。</p>
<p>両方の回路にA，B，C，Dの入力を与えたときの出力E，Fを見て正しい答えを導きます。「ア」と「エ」に関しては両方で成立することが直ぐにわかると思うので、実質的には「イ」と「ウ」の比較になると思います。</p>
<p>両方で成立します。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23aa.png" alt="解説図" width="449" height="121" ></p><p>右側の回路でだけ成立します。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23ii.png" alt="解説図" width="449" height="121" ></p><p>正しい。左側の回路でだけ成立します。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23uu.png" alt="解説図" width="449" height="121" ></p><p>両方で成立します。<br><img class="ap-fig" src="../../../assets/問題/午前/H31春期/23/23ee.png" alt="解説図" width="449" height="121" ></p></div>
