<p class="ap-question"><strong>問66</strong>　サブスクリプション方式のソフトウェア調達はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ERPソフトウェアの利用人数分の永続使用ライセンスをイニシャルコストとして購入し，必要に応じてライセンスを追加購入する。</div>
<div class="ap-choice-note">ソフトウェア自体を買い取る永続ライセンス方式であり、利用期間に応じた課金ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　新しいOS上で動作する最新バージョンのソフトウェアパッケージを販売代理店から購入する。</div>
<div class="ap-choice-note">パッケージの一括購入であり、利用期間ごとの利用権課金ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　新規開発した業務システムのソフトウェア開発費を無形固定資産として計上して，自社で利用する。</div>
<div class="ap-choice-note">自社開発費の資産計上であり、ベンダーからの期間利用権取得ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　ベンダーが提供するソフトウェアを，利用料金を支払うことによって一定期間の利用権を得て利用する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/サブスクリプションモデル" class="internal-link" data-href="用語/サブスクリプションモデル">サブスクリプションモデル</a></div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>サブスクリプション(Subscription)とは、ソフトウェアやサービスの料金形態の一種で、従来のようにソフトウェア自体を一括して買い取るのではなく、ソフトウェアの使用権を借り、その利用期間に応じて使用料が発生する方式です。一般的には1年単位や1カ月単位の課金となっていることが多くなっています。利用者にとっては初期費用が掛からず、不要になったらいつでも契約解除できる利点があります。サブスクリプションは英語で「定期購読」「予約購読」等の意味があります。</p>
<p>月額固定料金で音楽や動画が利用し放題のサービスや、月額課金で購読できる新聞社のWebサイト、月額課金のオンラインストレージ、PhotoShopで有名なAdobe社のCreative Cloudなどが代表的な事例です。</p>
<p>サブスクリプションは、ソフトウェアを買い取るのではなく使用期間ごとの利用権に料金を支払う形態ですので「エ」が該当します。</p>
</div>
