<p class="ap-question"><strong>問2</strong>　桁落ちによる誤差の説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　値のほぼ等しい二つの数値の差を求めたとき，有効桁数が減ることによって発生する誤差</div>
<div class="ap-choice-note">正しい。<a href="用語/桁落ち" class="internal-link" data-href="用語/桁落ち">桁落ち</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　指定された有効桁数で演算結果を表すために，切捨て，切上げ，四捨五入などで下位の桁を削除することによって発生する誤差</div>
<div class="ap-choice-note">これは<a href="用語/丸め誤差" class="internal-link" data-href="用語/丸め誤差">丸め誤差</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　絶対値の非常に大きな数値と小さな数値の加算や減算を行ったとき，小さい数値が計算結果に反映されないことによって発生する誤差</div>
<div class="ap-choice-note">これは<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　無限級数で表される数値の計算処理を有限項で打ち切ったことによって発生する誤差</div>
<div class="ap-choice-note">これは<a href="用語/打切り誤差" class="internal-link" data-href="用語/打切り誤差">打切り誤差</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/桁落ち" class="internal-link" data-href="用語/桁落ち">桁落ち</a>(けたおち)は、計算誤差のひとつで、絶対値の差が非常に小さい2つの値の差を求めたときに、<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>の大半が打ち消しあい、計算結果の有効桁数が少なくなることによって生じる誤差です。浮動小数点数の計算において、値がほぼ等しい値同士の減算や、絶対値がほぼ等しく正負が異なる値同士の加算を行うと生じることがあります。</p>
<p>コンピュータの内部では無理数など無限桁の数値も有限桁で表現しているため、有効桁数で表現できない差が"00…0"に丸められてしまい桁の欠落が生じます。<a href="用語/桁落ち" class="internal-link" data-href="用語/桁落ち">桁落ち</a>を防ぐ方法として"分子の有理化"があります。</p>
<p>例）有効桁数が8桁から2桁に減少する計算<br>1.2345678－1.2345666＝0.0000012＝1.2×10<sup>-6</sup></p>
