<p class="ap-question"><strong>問36</strong>　情報セキュリティにおけるエクスプロイトコードの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　同じセキュリティ機能をもつ製品に乗り換える場合に，CSV形式など他の製品に取り込むことができる形式でファイルを出力するプログラム</div>
<div class="ap-choice-note">これはデータ移行プログラムの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　コンピュータに接続されたハードディスクなどの外部記憶装置や，その中に保存されている暗号化されたファイルなどを閲覧，管理するソフトウェア</div>
<div class="ap-choice-note">これはファイル管理ツールの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　セキュリティ製品を設計する際の早い段階から実際に動作する試作品を作成し，それに対する利用者の反応を見ながら徐々に完成に近づける開発手法</div>
<div class="ap-choice-note">これはプロトタイプモデルの説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　ソフトウェアやハードウェアの脆弱性を検査するために作成されたプログラム</div>
<div class="ap-choice-note">正しい。<a href="用語/エクスプロイトコード" class="internal-link" data-href="用語/エクスプロイトコード">エクスプロイトコード</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/エクスプロイトコード" class="internal-link" data-href="用語/エクスプロイトコード">エクスプロイトコード</a>(Exploit Code)とは、ソフトウェアの<a href="用語/脆弱性" class="internal-link" data-href="用語/脆弱性">脆弱性</a>を悪用した不正な動作を再現するために作成されたスクリプトやプログラムを指す言葉です。</p>
<p>このようなプログラムは、善意の第三者や製品開発元の間で安全に流通している状況であれば問題ありませんが、世の中に出回ると攻撃方法の詳細が広く知られることになり、攻撃者が個人や企業、組織などへの攻撃を実行する際に悪用することが可能になります。実際に様々な<a href="用語/脆弱性" class="internal-link" data-href="用語/脆弱性">脆弱性</a>に対して攻撃が成功するように、複数の<a href="用語/エクスプロイトコード" class="internal-link" data-href="用語/エクスプロイトコード">エクスプロイトコード</a>をまとめた<a href="用語/エクスプロイトキット" class="internal-link" data-href="用語/エクスプロイトキット">エクスプロイトキット</a>というツールも存在し、攻撃に利用されています。</p>
<p>したがって「エ」が正解です。</p>
<p><a href="用語/エクスプロイトコード" class="internal-link" data-href="用語/エクスプロイトコード">エクスプロイトコード</a>は、多くの場合、悪意を持ったマルウェアのことを指しますが、セキュリティ向上目的での研究材料や<a href="用語/脆弱性" class="internal-link" data-href="用語/脆弱性">脆弱性</a>検査のために作成された実証用コードという意味でも用いられます。</p>
