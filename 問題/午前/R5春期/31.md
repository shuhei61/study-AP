<p class="ap-question"><strong>問31</strong>　通信技術の一つであるPLCの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　音声データをIPネットワークで伝送する技術</div>
<div class="ap-choice-note">これは<a href="用語/VoIP" class="internal-link" data-href="用語/VoIP">VoIP</a>(Voice over Internet Protocol)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　電力線を通信回線として利用する技術</div>
<div class="ap-choice-note">正しい。<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　無線LANの標準規格であるIEEE 802.11シリーズの総称</div>
<div class="ap-choice-note">これはWi-Fiの説明です。<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>技術はIEEE 1901として標準化されており、IEEE 802.11シリーズとは無関係です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　無線通信における暗号化技術</div>
<div class="ap-choice-note">これはWPA(Wi-Fi Protected Access)の説明です。<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>とは関係ありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>(Power Line Communications：電力線搬送通信)とは、電力線をデータ通信回線として利用する技術であり、数Mbps～数百Mbpsのデータ通信が可能です。450kHz以下の周波数を用いるものを低速<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>、2-30MHzを用いるものを高速<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>と呼びます。</p>
<p>法令により、<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a>機器の使用が屋内に限定されていたり、<a href="用語/セキュリティ" class="internal-link" data-href="用語/セキュリティ">セキュリティ</a>上の問題を残していたりするなどの課題もありますが、既存の配線を利用でき、配線工事が不要であるため導入コストが低いという利点があります。電気コンセントに取り付けた<a href="用語/PLC" class="internal-link" data-href="用語/PLC">PLC</a><a href="用語/モデム" class="internal-link" data-href="用語/モデム">モデム</a>をパソコンと繋ぐだけで簡単にネットワークを構築できるなど配線の利便性に優れています。</p>
<p>したがって「イ」が正解です。</p>
