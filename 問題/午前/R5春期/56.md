<p class="ap-question"><strong>問56</strong>　JIS Q 20000-1:2020(サービスマネジメントシステム要求事項)によれば，組織は，サービスレベル目標に照らしたパフォーマンスを監視し，レビューし，顧客に報告しなければならない。レビューをいつ行うかについて，この規格はどのように規定しているか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　SLAに大きな変更があったときに実施する。</div>
<div class="ap-choice-note">規格はレビューの実施時期を「あらかじめ定めた間隔」と規定しており，<a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>の変更時のみ実施するとはしていません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　あらかじめ定めた間隔で実施する。</div>
<div class="ap-choice-note">正しい。<a href="用語/サービスレベル管理" class="internal-link" data-href="用語/サービスレベル管理">サービスレベル管理</a>では，定めた間隔でパフォーマンスを監視・レビュー・報告することが求められます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　間隔を定めず，必要に応じて実施する。</div>
<div class="ap-choice-note">規格は定期的な実施を求めており，間隔を定めず必要に応じてのみ実施する規定ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　サービス目標の未達成が続いたときに実施する。</div>
<div class="ap-choice-note">未達成時の対応は別途考えられますが，レビュー実施のタイミングは「あらかじめ定めた間隔」として規定されています。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>JIS Q 20000-1:2020では、<a href="用語/サービスレベル管理" class="internal-link" data-href="用語/サービスレベル管理">サービスレベル管理</a>において実施すべき作業のひとつとして、あらかじめ定めた間隔で、組織は、①<a href="用語/サービスレベル目標" class="internal-link" data-href="用語/サービスレベル目標">サービスレベル目標</a>に照らしたパフォーマンス、②<a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>の作業負荷限度と比較した実績および周期的な変化を監視し、レビューし、報告しなければならないとしています。</p>
<p>組織において期ごとに目標に対する実績報告があるように、サービス提供においても目標に対する実績、達成度、課題などを、定期的に顧客に対して報告します。<a href="用語/ITIL" class="internal-link" data-href="用語/ITIL">ITIL</a>ではサービス・レビューと呼ばれます。</p>
<p>したがって「イ」が正解です。</p>
