<p class="ap-question"><strong>問21</strong>　NAND素子を用いた次の組合せ回路の出力Zを表す式はどれか。ここで，"・"は論理積，"＋"は論理和，"<span style="text-decoration:overline">X</span>"はXの否定を表す。<br><img class="ap-fig" src="../../../assets/問題/午前/R5春期/21/21.png" alt="問題図" width="240" height="91" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　X・Y</div>
<div class="ap-choice-note">2入力の<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>であり、設問の回路の<a href="用語/真理値表" class="internal-link" data-href="用語/真理値表">真理値表</a>から得られる<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>とは等価ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　X＋Y</div>
<div class="ap-choice-note">正しい。<a href="用語/真理値表" class="internal-link" data-href="用語/真理値表">真理値表</a>より、設問の回路は2つの入力の<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>を得る<a href="用語/OR回路" class="internal-link" data-href="用語/OR回路">OR回路</a>と等価である。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　<span style="text-decoration:overline">X・Y</span></div>
<div class="ap-choice-note">2入力の<a href="用語/論理積" class="internal-link" data-href="用語/論理積">論理積</a>の否定であり、設問の回路の出力とは一致しない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　<span style="text-decoration:overline">X＋Y</span></div>
<div class="ap-choice-note">2入力の<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>の否定であり、設問の回路の出力とは一致しない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><strong><a href="用語/NAND回路" class="internal-link" data-href="用語/NAND回路">NAND回路</a></strong>は、NAND(Not AND)の名称どおり<a href="用語/AND回路" class="internal-link" data-href="用語/AND回路">AND回路</a>の逆を出力する回路で、2つの入力がともに"1"のときだけ"0"を出力し、それ以外の入力では"1"を出力します。<br><img class="ap-fig" src="../../../assets/問題/午前/R5春期/21/21_1.png" alt="解説図 1" width="178" height="161" ></p>
<p>設問の回路に入力値XとYのすべての組合せ（X=0,Y=0・X=1,Y=0・X=0,Y=1・X=1, Y=1）を試すと、それぞれ以下の出力が得られます。<br><img class="ap-fig" src="../../../assets/問題/午前/R5春期/21/21_2.png" alt="解説図 2" width="450" height="187" ></p>
<p>上記結果の入力値XとYおよび出力値Zの関係を整理すると、次の<a href="用語/真理値表" class="internal-link" data-href="用語/真理値表">真理値表</a>が得られます。<br><img class="ap-fig" src="../../../assets/問題/午前/R5春期/21/21_3.png" alt="解説図 3" width="178" height="161" ></p>
<p>この<a href="用語/真理値表" class="internal-link" data-href="用語/真理値表">真理値表</a>より、設問の回路は2つの入力の<a href="用語/論理和" class="internal-link" data-href="用語/論理和">論理和</a>を得る<a href="用語/OR回路" class="internal-link" data-href="用語/OR回路">OR回路</a>と等価であることがわかるので、正解は「X＋Y」です。</p>
<p>【別解】<br>回路図を<a href="用語/論理式" class="internal-link" data-href="用語/論理式">論理式</a>で表し、その<a href="用語/論理式" class="internal-link" data-href="用語/論理式">論理式</a>を変形することで答えを導く解く方法もあります。</p>
<p>Z＝<span style="text-decoration:overline"><span style="text-decoration:overline">X・X</span>・<span style="text-decoration:overline">Y・Y</span></span><br>　＝<span style="text-decoration:overline"><span style="text-decoration:overline">X・X</span></span>＋<span style="text-decoration:overline"><span style="text-decoration:overline">Y・Y</span></span>　//<a href="用語/ド・モルガンの法則" class="internal-link" data-href="用語/ド・モルガンの法則">ド・モルガンの法則</a>を適用<br>　＝X・X＋Y・Y　//<span style="text-decoration:overline">A</span>＝A<br>　＝X＋Y　//A・A＝A</p>
