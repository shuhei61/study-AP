<p class="ap-question"><strong>問42</strong>　デジタルフォレンジックスの手順は収集，検査，分析及び報告から成る。このとき，デジタルフォレンジックスの手順に含まれるものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　サーバとネットワーク機器のログをログ管理サーバに集約し，リアルタイムに相関分析することによって，不正アクセスを検出する。</div>
<div class="ap-choice-note">これは<a href="用語/SIEM" class="internal-link" data-href="用語/SIEM">SIEM</a>(Security Information and Event Management)に関する記述です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　サーバのハードディスクを解析し，削除されたログファイルを復元することによって，不正アクセスの痕跡を発見する。</div>
<div class="ap-choice-note">正しい。<a href="用語/デジタルフォレンジックス" class="internal-link" data-href="用語/デジタルフォレンジックス">デジタルフォレンジックス</a>に関する記述です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　電子メールを外部に送る際に，本文及び添付ファイルを暗号化することによって，情報漏えいを防ぐ。</div>
<div class="ap-choice-note">これは<a href="用語/DLP" class="internal-link" data-href="用語/DLP">DLP</a>(Data Loss Prevention)に関する記述です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プログラムを実行する際に，プログラムファイルのハッシュ値と脅威情報を突き合わせることによって，プログラムがマルウェアかどうかを検査する。</div>
<div class="ap-choice-note">これは<a href="用語/EDR" class="internal-link" data-href="用語/EDR">EDR</a>(Endpoint Detection and Response)に関する記述です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/デジタルフォレンジックス" class="internal-link" data-href="用語/デジタルフォレンジックス">デジタルフォレンジックス</a>は、<a href="用語/インシデント" class="internal-link" data-href="用語/インシデント">インシデント</a>の発生時に関連するデジタル機器上の電磁的記録を確実かつ正確に収集・取得し、保全するとともに、分析・解析を行い、法定紛争やトラブルにおける立証に役立てる活動です。</p>
<p>IPAが公開している「<a href="用語/インシデント" class="internal-link" data-href="用語/インシデント">インシデント</a>対応へのフォレンジック技法の統合に関するガイド」によれば、<a href="用語/デジタルフォレンジックス" class="internal-link" data-href="用語/デジタルフォレンジックス">デジタルフォレンジックス</a>の手順は、収集・検査・分析・報告の4つのフェーズから成ります。</p>
<ul>
<li>収集：特定の事象に関連するデータの識別、ラベル付け、記録、および収集を行い、その完全性を保護する</li>
<li>検査：収集されたデータの種類に適したフォレンジックツールやフォレンジック技法を実行することにより、データの完全性を保護しながら、収集されたデータから関連する情報を識別し、抽出する</li>
<li>分析：検査結果を分析することにより、収集と検査を行う契機となった疑問を解決するのに役立つ情報を導き出す</li>
<li>報告：分析フェーズによって得られた結果を報告する</li>
</ul>
<p>選択肢のうち、セキュリティ<a href="用語/インシデント" class="internal-link" data-href="用語/インシデント">インシデント</a>の痕跡を集めている「イ」が、<a href="用語/デジタルフォレンジックス" class="internal-link" data-href="用語/デジタルフォレンジックス">デジタルフォレンジックス</a>の手順のうち「収集・検査」に該当します。</p>
</div>
