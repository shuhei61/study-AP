<p class="ap-question"><strong>問46</strong>　モジュールの独立性を高めるには，モジュール結合度を低くする必要がある。モジュール間の情報の受渡し方法のうち，モジュール結合度が最も低いものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　共通域に定義したデータを関係するモジュールが参照する。</div>
<div class="ap-choice-note">共通結合に当たります。<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>より結合度が強く、独立性は低くなります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　制御パラメータを引数として渡し，モジュールの実行順序を制御する。</div>
<div class="ap-choice-note"><a href="用語/制御結合" class="internal-link" data-href="用語/制御結合">制御結合</a>に当たります。<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>より結合度が強く、独立性は低くなります。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　入出力に必要なデータ項目だけをモジュール間の引数として渡す。</div>
<div class="ap-choice-note">正しい。最もモジュール結合度が弱い<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>に当たります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　必要なデータを外部宣言して共有する。</div>
<div class="ap-choice-note">外部結合に当たります。<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>より結合度が強く、独立性は低くなります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>モジュール結合度とは、モジュール同士の関連性の強さを表す概念です。関連する別のモジュールに変更があった場合、モジュール同士の結合度が弱いほどその影響を受けにくくなるので、モジュールの<a href="用語/保守性" class="internal-link" data-href="用語/保守性">保守性</a>は向上します。よって、モジュール結合度が弱いほど、その<a href="用語/モジュールの独立性" class="internal-link" data-href="用語/モジュールの独立性">モジュールの独立性</a>は高いと言えます。</p>
<p>モジュール結合度の強弱は、モジュール間のデータの受渡し方法などにより6段階のレベルに分類されています。</p>
<ul>
<li><a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>（弱い↑）… 処理に必要なデータだけを単一のパラメータとして受け渡している</li>
<li>スタンプ結合 … 処理に必要なデータだけをレコードや構造体などのデータ構造として受け渡している</li>
<li><a href="用語/制御結合" class="internal-link" data-href="用語/制御結合">制御結合</a> … モジュールの動作を制御するための要素を受け渡している</li>
<li>外部結合 … 大域宣言された単一のデータを、複数のモジュールが参照している</li>
<li>共通結合 … 大域宣言されたレコードや構造体などのデータ構造を、複数のモジュールが参照している</li>
<li>内容結合（強い↓）… モジュールの内部を直接参照・使用している</li>
</ul>
<p>モジュール結合度が最も低いのは<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>です。したがって「ウ」が適切です。</p>
</div>
