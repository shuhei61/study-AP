<p class="ap-question"><strong>問55</strong>　JIS Q 20000-1:2020(サービスマネジメントシステム要求事項)によれば，サービスマネジメントシステム(SMS)における継続的改善の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　意図した結果を得るためにインプットを使用する，相互に関連する又は相互に作用する一連の活動</div>
<div class="ap-choice-note"><a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　価値を提供するため，サービスの計画立案，設計，移行，提供及び改善のための組織の活動及び資源を，指揮し，管理する，一連の能力及びプロセス</div>
<div class="ap-choice-note"><a href="用語/サービスマネジメント" class="internal-link" data-href="用語/サービスマネジメント">サービスマネジメント</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　サービスを中断なしに，又は合意した可用性を一貫して提供する能力</div>
<div class="ap-choice-note">サービス継続の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　パフォーマンスを向上するために繰り返し行われる活動</div>
<div class="ap-choice-note">正しい。<a href="用語/継続的改善" class="internal-link" data-href="用語/継続的改善">継続的改善</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/サービスマネジメントシステム" class="internal-link" data-href="用語/サービスマネジメントシステム">サービスマネジメントシステム</a>における<a href="用語/継続的改善" class="internal-link" data-href="用語/継続的改善">継続的改善</a>とは、常に変化するビジネスニーズに整合するために、またはサービスの適切性、妥当性、有効性などの向上を目的として、ITサービス、<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>、ITインフラストラクチャを常に改善していく<a href="用語/プロセス" class="internal-link" data-href="用語/プロセス">プロセス</a>です。JIS Q 20000-1:2020では「パフォーマンスを向上するために繰り返し行われる活動」と定義されています。他の<a href="用語/マネジメントシステム" class="internal-link" data-href="用語/マネジメントシステム">マネジメントシステム</a>と同じく、<a href="用語/PDCA" class="internal-link" data-href="用語/PDCA">PDCA</a>サイクルを回してサービスの継続的な改善を行います。</p>
<p>したがって「エ」が正解です。</p>
