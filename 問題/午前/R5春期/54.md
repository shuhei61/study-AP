<p class="ap-question"><strong>問54</strong>　プロジェクトのリスクマネジメントにおける，リスクの特定に使用する技法の一つであるデルファイ法の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　確率分布を使用したシミュレーションを行う。</div>
<div class="ap-choice-note"><a href="用語/シミュレーション" class="internal-link" data-href="用語/シミュレーション">シミュレーション</a>は<a href="用語/リスクの特定" class="internal-link" data-href="用語/リスクの特定">リスクの特定</a>では使用されません。<a href="用語/リスクの定量的分析" class="internal-link" data-href="用語/リスクの定量的分析">リスクの定量的分析</a>で使われます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　過去の情報や知識を基にして，あらかじめ想定されるリスクをチェックリストにまとめておき，チェックリストと照らし合わせることによってリスクを識別する。</div>
<div class="ap-choice-note"><a href="用語/チェックリスト法" class="internal-link" data-href="用語/チェックリスト法">チェックリスト法</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　何人かが集まって，他人のアイディアを批判することなく，自由に多くのアイディアを出し合う。</div>
<div class="ap-choice-note"><a href="用語/ブレーンストーミング" class="internal-link" data-href="用語/ブレーンストーミング">ブレーンストーミング</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　複数の専門家から得られた見解を要約して再配布し，再度見解を求めることを何度か繰り返して収束させる。</div>
<div class="ap-choice-note">正しい。<a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>は、複数の専門家から個別に意見の収集を行い、得られた意見の集約をフィードバックするということを繰り返して、最終的に意見の収束をしていく手法です。技術革新や社会変動などに関する未来予測においてよく用いられます。</p>
<p><a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>は次の手順に従って行われます。複数の専門家を回答者として選定し、個別に質問を行う。質問に対する回答結果をフィードバックし、再度質問を行う。回答結果を統計的に処理し，確率分布とともに回答結果を示す。したがって「エ」が正解です。</p>
