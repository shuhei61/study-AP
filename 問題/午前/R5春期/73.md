<p class="ap-question"><strong>問73</strong>　各種センサーを取り付けた航空機のエンジンから飛行中に収集したデータを分析し，仮想空間に構築したエンジンのモデルに反映してシミュレーションを行うことによって，各パーツの消耗状況や交換時期を正確に予測できるようになる。このように産業機器などにIoT技術を活用し，現実世界や物理的現象をリアルタイムに仮想空間で忠実に再現することを表したものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　サーバ仮想化</div>
<div class="ap-choice-note">これはサーバ<a href="用語/仮想化" class="internal-link" data-href="用語/仮想化">仮想化</a>の説明です。1台の物理サーバ上で複数のサーバOSを稼働させる技術であり、本問の仮想空間への現実の再現とは異なります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　スマートグリッド</div>
<div class="ap-choice-note">これは<a href="用語/スマートグリッド" class="internal-link" data-href="用語/スマートグリッド">スマートグリッド</a>の説明です。デジタル機器の通信・演算能力を活用して電力需要を自律的に調整する新しい電力網システムであり、本問の記述とは異なります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　スマートメーター</div>
<div class="ap-choice-note">これは<a href="用語/スマートメーター" class="internal-link" data-href="用語/スマートメーター">スマートメーター</a>の説明です。通信機能を有し電力使用量を定期的に電力会社へ遠隔報告できる電力量計であり、本問の記述とは異なります。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　デジタルツイン</div>
<div class="ap-choice-note">正しい。<a href="用語/デジタルツイン" class="internal-link" data-href="用語/デジタルツイン">デジタルツイン</a>は、コンピュータ上の仮想空間に現実空間の複製を用意し、その中でシミュレーションや将来予測を行う技術です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/デジタルツイン" class="internal-link" data-href="用語/デジタルツイン">デジタルツイン</a>は、各種センサーから得た情報を用いて、デジタル上に現実世界を忠実かつリアルタイムに再現することです。サイバー空間の世界を活用することで、現実世界を監視したり、シミュレーションを行った結果によって現実世界に起こる変化を予測したりといったことが可能になります。<a href="用語/IoT" class="internal-link" data-href="用語/IoT">IoT</a>の登場で、膨大なデータの収集が飛躍的に簡単になったことによって実現した技術です。</p>
</div>
