<p class="ap-question"><strong>問14</strong>　CPUと磁気ディスク装置で構成されるシステムで，表に示すジョブA，Bを実行する。この二つのジョブが実行を終了するまでのCPUの使用率と磁気ディスク装置の使用率との組み合わせのうち，適切なものはどれか。ここで，ジョブA，Bはシステムの動作開始時点ではいずれも実行可能状態にあり，A，Bの順で実行される。CPU及び磁気ディスク装置は，ともに一つの要求だけを発生順に処理する。ジョブA，Bとも，CPUの処理を終了した後，磁気ディスク装置の処理を実行する。<br><img class="ap-fig" src="../../../assets/問題/午前/R5春期/14/14.png" alt="問題図" width="375" height="94" ><br><img class="ap-fig" src="../../../assets/問題/午前/R5春期/14/14a.png" alt="選択肢の組合せ表" width="338" height="153" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア</div>
<div class="ap-choice-note"><a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の使用率と磁気ディスク装置の使用率の組合せが誤っています。組合せは選択肢表を参照してください。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ</div>
<div class="ap-choice-note">正しい。<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の使用率15秒÷25秒＝0.60、磁気ディスク装置の使用率17秒÷25秒＝0.68です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ</div>
<div class="ap-choice-note"><a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の使用率と磁気ディスク装置の使用率の組合せが誤っています。組合せは選択肢表を参照してください。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ</div>
<div class="ap-choice-note"><a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の使用率と磁気ディスク装置の使用率の組合せが誤っています。組合せは選択肢表を参照してください。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>各1つの<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>と磁気ディスク装置を使い、ジョブA→Bの順で処理をすると、<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>と磁気ディスクの使用状況は下の図で示すようになります。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R5春期/14/14_1.png" alt="解説図 1" width="544" height="80" ></p>
<p>2つのジョブが終了するまでに要する時間は25秒で、そのうち色が付いている部分が各機器を使用している時間です。</p>
<p><a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の使用率　15秒÷25秒＝0.60</p>
<p>磁気ディスク装置の使用率　17秒÷25秒＝0.68</p>
<p>したがって「イ」の組合せが正解です。</p>
