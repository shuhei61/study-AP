<p class="ap-question"><strong>問48</strong>　スクラムでは，一定の期間で区切ったスプリントを繰り返して開発を進める。各スプリントで実施するスクラムイベントの順序のうち，適切なものはどれか。<br>〔スクラムイベント〕1.スプリントプランニング　2.スプリントレトロスペクティブ　3.スプリントレビュー　4.デイリースクラム</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　1 → 4 → 2 → 3</div>
<div class="ap-choice-note"><a href="用語/スプリントレビュー" class="internal-link" data-href="用語/スプリントレビュー">スプリントレビュー</a>(3)の前に<a href="用語/スプリントレトロスペクティブ" class="internal-link" data-href="用語/スプリントレトロスペクティブ">スプリントレトロスペクティブ</a>(2)を置いており，終了時の成果検査と振り返りの順序が逆です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　1 → 4 → 3 → 2</div>
<div class="ap-choice-note">正しい。開始前に<a href="用語/スプリントプランニング" class="internal-link" data-href="用語/スプリントプランニング">スプリントプランニング</a>，実施中に<a href="用語/デイリースクラム" class="internal-link" data-href="用語/デイリースクラム">デイリースクラム</a>，終了時に<a href="用語/スプリントレビュー" class="internal-link" data-href="用語/スプリントレビュー">スプリントレビュー</a>，最後に<a href="用語/スプリントレトロスペクティブ" class="internal-link" data-href="用語/スプリントレトロスペクティブ">スプリントレトロスペクティブ</a>の順序です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　4 → 1 → 2 → 3</div>
<div class="ap-choice-note"><a href="用語/デイリースクラム" class="internal-link" data-href="用語/デイリースクラム">デイリースクラム</a>(4)を最初に置いており，<a href="用語/スプリントプランニング" class="internal-link" data-href="用語/スプリントプランニング">スプリントプランニング</a>(1)より前に実施する順序になっています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　4 → 1 → 3 → 2</div>
<div class="ap-choice-note"><a href="用語/デイリースクラム" class="internal-link" data-href="用語/デイリースクラム">デイリースクラム</a>(4)を最初に置いており，<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>開始前の計画(1)より前に実施する順序になっています。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/スクラム" class="internal-link" data-href="用語/スクラム">スクラム</a>は、<a href="用語/アジャイル" class="internal-link" data-href="用語/アジャイル">アジャイル</a>開発の方法論のひとつで、開発プロジェクトを数週間程度の短期間ごとに区切り、その期間内に分析、設計、実装、テストの一連の活動を行い、一部分の機能を完成させるという作業を繰り返しながら、段階的に動作可能なシステムを作り上げるフレームワークです。</p>
<p><a href="用語/スクラム" class="internal-link" data-href="用語/スクラム">スクラム</a>開発では開発反復の単位を「<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>」といいます。<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>実施の際に行う各<a href="用語/スクラム" class="internal-link" data-href="用語/スクラム">スクラム</a>イベントの内容は次のとおりです。</p>
<ul>
<li><a href="用語/スプリントプランニング" class="internal-link" data-href="用語/スプリントプランニング">スプリントプランニング</a> … <a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>の起点となるミーティング。<a href="用語/プロダクトバックログ" class="internal-link" data-href="用語/プロダクトバックログ">プロダクトバックログ</a>の中から、その<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>で開発する機能を決定し、作業の計画を立てる</li>
<li><a href="用語/スプリントレトロスペクティブ" class="internal-link" data-href="用語/スプリントレトロスペクティブ">スプリントレトロスペクティブ</a> … <a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>の最後に実施した<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>全体を振り返る活動。次の<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>を見据えて、上手くいったこと、上手くいかなかったこと、それに対する改善法・解決法を話し合う</li>
<li><a href="用語/スプリントレビュー" class="internal-link" data-href="用語/スプリントレビュー">スプリントレビュー</a> … <a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>の終了時に<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>の成果を検査し、今後の適応を決定する活動。主要なステークホルダーに成果物をプレゼンテーションし、プロダクトゴールに対する進捗について話し合う</li>
<li><a href="用語/デイリースクラム" class="internal-link" data-href="用語/デイリースクラム">デイリースクラム</a> … 開発チーム全員の活動を共有するために、<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>実施中に毎日10～15分程度行われる確認・調整のための会議。昨日やったこと、今日やること、障害になっていることを各人が話す</li>
</ul>
<p>各<a href="用語/スプリント" class="internal-link" data-href="用語/スプリント">スプリント</a>内の時期と各イベントの実行タイミングは以下に対応します。</p>
<ul>
<li>開始前 … 1.<a href="用語/スプリントプランニング" class="internal-link" data-href="用語/スプリントプランニング">スプリントプランニング</a></li>
<li>実施中 … 4.<a href="用語/デイリースクラム" class="internal-link" data-href="用語/デイリースクラム">デイリースクラム</a></li>
<li>終了時 … 3.<a href="用語/スプリントレビュー" class="internal-link" data-href="用語/スプリントレビュー">スプリントレビュー</a></li>
<li>最後 … 2.<a href="用語/スプリントレトロスペクティブ" class="internal-link" data-href="用語/スプリントレトロスペクティブ">スプリントレトロスペクティブ</a></li>
</ul>
<p>したがって「イ」の実施順序が適切です。</p>
