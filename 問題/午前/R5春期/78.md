<p class="ap-question"><strong>問78</strong>　ソフトウェア開発を，中小受託取引適正化法(以下，取適法)の対象となる中小受託事業者に委託する場合，取適法に照らして，禁止されている行為はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　継続的な取引が行われているので，支払条件，支払期日などを記載した書面をあらかじめ交付し，個々の発注書面にはその事項の記載を省略する。</div>
<div class="ap-choice-note">原則として<a href="用語/製造委託" class="internal-link" data-href="用語/製造委託">製造委託</a>等の都度、発注内容を記載した所定の書面を交付する必要がありますが、支払方法・検査期間など一定期間共通の事項をあらかじめ書面で通知している場合は、都度交付する書面への記載は不要です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　顧客の求める仕様が確定していなかったので，発注の際に，中小受託事業者に仕様が未記載の書面を交付し，仕様が確定した時点では，内容を書面ではなく口頭で伝える。</div>
<div class="ap-choice-note">正しい。法定の発注書面には品目・品種・数量・規格・仕様など給付内容を明確に記載する必要があります。給付内容が定まっていないことにつき正当な理由がある場合は、発注書にその旨と理由・定める予定期日を記載し、内容が定められた後は当該事項を書面または電磁的方法で明示する方法が認められます。口頭では不可です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　顧客の都合で仕様変更の必要が生じたので，中小受託事業者と協議の上，発生する費用の増加分を製造委託等代金に加算することによって仕様変更に応じてもらう。</div>
<div class="ap-choice-note"><a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>が費用を負担せずに注文内容を変更する行為は不当な給付内容の変更として禁止されていますが、協議を行い増加費用を支払うことを約束した上での委託内容の変更は禁止されていません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　振込手数料を中小受託事業者が負担する旨を発注前に書面で合意していたが，振込手数料は委託事業者が負担し，製造委託等代金は減額せず全額を支払う。</div>
<div class="ap-choice-note">振込手数料を<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>側に負担させ代金の額から差し引くことは、合意の有無にかかわらず<a href="用語/製造委託" class="internal-link" data-href="用語/製造委託">製造委託</a>等代金の減額として禁止されています。書面合意があっても振込手数料は<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>が負担する必要があります（本肢は法改正に伴い改題。改正前は事前合意で認められていたが現行では禁止）。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/中小受託取引適正化法" class="internal-link" data-href="用語/中小受託取引適正化法">中小受託取引適正化法</a>（旧称：下請法）は、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>（発注側）と<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>（受注側）の公平な取引関係を確保し、受託側の利益保護と取引の公正化を図るための法律です。一定規模以上の<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>を「優越的地位にある」ものと位置付け、受託取引における<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>の不当な行為を規制することを狙いとしています。具体的な規制内容として、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>に対し、発注内容等の明示、書類等の作成・保存、代金の支払期日の設定、遅延利息の支払といった義務を課すとともに、受領拒否、返品、代金減額、支払遅延など11類型の禁止行為を定めています。</p>
</div>
</div>
