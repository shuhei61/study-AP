<p class="ap-question"><strong>問63</strong>　ビッグデータの利活用を促す取組の一つである情報銀行の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　金融機関が，自らが有する顧客の決済データを分析して，金融商品の提案や販売など，自らの営業活動に活用できるようにする取組</div>
<div class="ap-choice-note">FinTechの事例です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　国や自治体が，公共データに匿名加工を施した上で，二次利用を促進するために共通プラットフォームを介してデータを民間に提供できるようにする取組</div>
<div class="ap-choice-note">行政<a href="用語/ビッグデータ" class="internal-link" data-href="用語/ビッグデータ">ビッグデータ</a>(行政機関等<a href="用語/匿名加工情報" class="internal-link" data-href="用語/匿名加工情報">匿名加工情報</a>)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　事業者が，個人との契約などに基づき個人情報を預託され，当該個人の指示又は指定した条件に基づき，データを他の事業者に提供できるようにする取組</div>
<div class="ap-choice-note">正しい。<a href="用語/情報銀行" class="internal-link" data-href="用語/情報銀行">情報銀行</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　事業者が，自社工場におけるIoT機器から収集された産業用データを，インターネット上の取引市場を介して，他の事業者に提供できるようにする取組</div>
<div class="ap-choice-note">データ取引市場の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/情報銀行" class="internal-link" data-href="用語/情報銀行">情報銀行</a>は、個人とのデータ活用に関する契約等に基づいて預かった個人データを一括して管理するとともに、個人の指示や指定条件に従って個人データを管理したり、他の事業者に提供したりするサービスを提供する事業者のことを指します。データの提供・活用に関する便益が、事業者から個人に直接的または間接的に還元されるような仕組みになっています。</p>
<p><a href="用語/情報銀行" class="internal-link" data-href="用語/情報銀行">情報銀行</a>には、①個人データの安全な管理、②個人データが消費者のものであることを明確にすること、③<a href="用語/第三者提供" class="internal-link" data-href="用語/第三者提供">第三者提供</a>時のデータの用途と明確化、④消費者自身がデータを扱えるようにすること、⑤データ活用で得られた利益を消費者に還元すること、などの役割が期待されています。</p>
</div>
