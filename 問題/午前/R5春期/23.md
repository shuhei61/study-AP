<p class="ap-question"><strong>問23</strong>　車の自動運転に使われるセンサーの一つであるLiDARの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　超音波を送出し，その反射波を測定することによって，対象物の有無の検知及び対象物までの距離の計測を行う。</div>
<div class="ap-choice-note"><a href="用語/超音波センサー" class="internal-link" data-href="用語/超音波センサー">超音波センサー</a>(ソナー)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　道路の幅及び車線は無限遠の地平線で一点(消失点)に収束する，という遠近法の原理を利用して，対象物までの距離を計測する。</div>
<div class="ap-choice-note">単眼カメラの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ミリ波帯の電磁波を送出し，その反射波を測定することによって，対象物の有無の検知及び対象物までの距離の計測を行う。</div>
<div class="ap-choice-note"><a href="用語/ミリ波レーダー" class="internal-link" data-href="用語/ミリ波レーダー">ミリ波レーダー</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　レーザー光をパルス状に照射し，その反射光を測定することによって，対象物の方向，距離及び形状を計測する。</div>
<div class="ap-choice-note">正しい。<a href="用語/LiDAR" class="internal-link" data-href="用語/LiDAR">LiDAR</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/LiDAR" class="internal-link" data-href="用語/LiDAR">LiDAR</a>(Light Detection And Ranging：ライダー)は、対象物にレーザー光を照射し、その反射光を計測することで、物体の位置や距離、形状を知ることができるセンサー技術です。波長が短く、反射や屈折による誤差が少ないレーザー光を使うことで、従来のミリ波やカメラによる検知よりも高精度の測定が可能なことが特徴です。反射光が返ってくるまでの時間を基に算出するTOF方式や、反射光の周波数を基に算出するFMCW方式があります。</p>
<p><a href="用語/自動運転" class="internal-link" data-href="用語/自動運転">自動運転</a>車が一般道を走行する場合、信号や標識、他の車や人など多くの障害物が存在するため、自動車が周囲の状況を正確に把握する必要があります。<a href="用語/LiDAR" class="internal-link" data-href="用語/LiDAR">LiDAR</a>は、ドライバーが運転席を離れることができる<a href="用語/自動運転レベル" class="internal-link" data-href="用語/自動運転レベル">自動運転レベル</a>4・5を実現するために不可欠な技術の一つとして注目されています。</p>
