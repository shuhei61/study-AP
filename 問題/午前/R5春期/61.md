<p class="ap-question"><strong>問61</strong>　情報化投資計画において，投資価値の評価指標であるROIを説明したものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　売上増やコスト削減などによって創出された利益額を投資額で割ったもの</div>
<div class="ap-choice-note">正しい。<a href="用語/ROI" class="internal-link" data-href="用語/ROI">ROI</a>（Return On Investment：投資利益率）の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　売上高投資金額比，従業員当たりの投資金額などを他社と比較したもの</div>
<div class="ap-choice-note">これはIT投資評価の手法の一つである他社比較（<a href="用語/ベンチマーク" class="internal-link" data-href="用語/ベンチマーク">ベンチマーク</a>）の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　現金流入の現在価値から，現金流出の現在価値を差し引いたもの</div>
<div class="ap-choice-note">これは<a href="用語/NPV法" class="internal-link" data-href="用語/NPV法">NPV法</a>（Net Present Value：正味現在価値法）の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プロジェクトを実施しない場合の，市場での競争力を表したもの</div>
<div class="ap-choice-note">これはIT投資評価の手法の一つである<a href="用語/機会損失" class="internal-link" data-href="用語/機会損失">機会損失</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ROI" class="internal-link" data-href="用語/ROI">ROI</a>（Return On Investment：投資利益率）は、<a href="用語/投資回収期間" class="internal-link" data-href="用語/投資回収期間">投資回収期間</a>に得られる効果金額を投資額で割ったもので、投資したお金に対してどのくらい効率的に利益を生み出しているかを示す指標です。<a href="用語/ROI" class="internal-link" data-href="用語/ROI">ROI</a>は投資の効率性を測る上で重要な指標であり、値が大きいほど投資効率が高いということになります。</p>
<p><a href="用語/ROI" class="internal-link" data-href="用語/ROI">ROI</a>(%)＝利益÷投資額×100</p>
<p>「イ」は他社比較（<a href="用語/ベンチマーク" class="internal-link" data-href="用語/ベンチマーク">ベンチマーク</a>）、「ウ」は<a href="用語/NPV法" class="internal-link" data-href="用語/NPV法">NPV法</a>、「エ」は<a href="用語/機会損失" class="internal-link" data-href="用語/機会損失">機会損失</a>の説明です。正解は「ア」です。</p>
</div>
