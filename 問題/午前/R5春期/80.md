<p class="ap-question"><strong>問80</strong>　技術者倫理の遵守を妨げる要因の一つとして，集団思考というものがある。集団思考の説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　自分とは違った視点から事態を見ることができず，客観性に欠けること</div>
<div class="ap-choice-note">これは認知バイアスの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　組織内の権威に無批判的に服従すること</div>
<div class="ap-choice-note">これは権威主義的服従の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　正しいことが何かは知っているが，それを実行する勇気や決断力に欠けること</div>
<div class="ap-choice-note">これは平和主義者の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　強い連帯性をもつチームが自らへの批判的思考を欠いて，不合理な合意へと達すること</div>
<div class="ap-choice-note">正しい。<a href="用語/集団思考" class="internal-link" data-href="用語/集団思考">集団思考</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/集団思考" class="internal-link" data-href="用語/集団思考">集団思考</a>(グループシンク)とは、集団で意思決定を行う際に、集団の結束がマイナスに働いた結果、不合理な決定が下されてしまうことをいいます。不合理な判断がなされた例としてチャレンジャー号事件、ケネディ政権のピッグス湾侵攻作戦、ウォーターゲート事件を巡るニクソン大統領の判断ミスなどがよく挙げられます。</p>
<p><a href="用語/集団思考" class="internal-link" data-href="用語/集団思考">集団思考</a>はチーム内の結束が強く、閉鎖的な体制などの欠陥を有し、チームに掛かるストレスが高いという状況が揃うと発生しやすい傾向にあると言われています。アメリカの心理学者であるアーヴィング・ジャニスが1982年に発表した「<a href="用語/集団思考" class="internal-link" data-href="用語/集団思考">集団思考</a>の8つの兆候」では、<a href="用語/集団思考" class="internal-link" data-href="用語/集団思考">集団思考</a>に陥りやすい集団が示す兆候を以下の3類型8項目に整理しています。</p>
<p>第1類型（グループの能力や道徳性に対する過大評価）<br>自分たちを不死身と見なす幻想<br>集団に固有の道徳性についての再考の否定</p>
<p>第2類型（閉鎖的な関心）<br>集団の行動を合理的なものに見せようとする自己弁護<br>集団外部への批判・偏見及び責任の転嫁</p>
<p>第3類型（均一性への圧力）<br>自身の意見が集団の総意から外れていないかをチェックする自己検閲<br>沈黙を同意と見なすなどの全会一致の幻想<br>反論するメンバーへの直接的な圧力<br>集団の自己満足を妨げる情報が入ってくるのを阻止する意見監視員の出現</p>
<p>技術者が研究や設計などの業務を行う際は、集団の中で討議を行い意思決定をしていくことがほとんどです。技術者は集団における社会心理学的要素を十分に理解し、たとえ技術者倫理に則した自身の意見が組織の倫理と対立する場合であっても、常に技術者倫理側に立って行動することが求められます。</p>
