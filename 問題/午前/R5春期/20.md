<p class="ap-question"><strong>問20</strong>　コンテナ型仮想化の環境であって，アプリケーションソフトウェアの構築，実行，管理を行うためのプラットフォームを提供するOSSはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　Docker</div>
<div class="ap-choice-note">正しい。<a href="用語/Docker" class="internal-link" data-href="用語/Docker">Docker</a>は、コンテナ型<a href="用語/仮想化" class="internal-link" data-href="用語/仮想化">仮想化</a>の環境で、アプリケーションを実行するための独立した環境を提供する<a href="用語/OSS" class="internal-link" data-href="用語/OSS">OSS</a>。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　KVM</div>
<div class="ap-choice-note">KVM(Kernel-based Virtual Machine)は、Linuxカーネルが提供する<a href="用語/仮想化" class="internal-link" data-href="用語/仮想化">仮想化</a>ソフトウェア。<a href="用語/OSS" class="internal-link" data-href="用語/OSS">OSS</a>だがハイパーバイザー型なので不適切。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　QEMU</div>
<div class="ap-choice-note">QEMU(Quick Emulator)は、<a href="用語/OSS" class="internal-link" data-href="用語/OSS">OSS</a>の<a href="用語/仮想化" class="internal-link" data-href="用語/仮想化">仮想化</a>ソフトウェアだが、ホストOS型なので誤り。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　Xen</div>
<div class="ap-choice-note">Xenは、<a href="用語/OSS" class="internal-link" data-href="用語/OSS">OSS</a>の<a href="用語/仮想化" class="internal-link" data-href="用語/仮想化">仮想化</a>ソフトウェアだが、ハイパーバイザー型なので誤り。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/Docker" class="internal-link" data-href="用語/Docker">Docker</a>はコンテナ型<a href="用語/仮想化" class="internal-link" data-href="用語/仮想化">仮想化</a>のプラットフォームとして、アプリケーションの構築・実行・管理を行う<a href="用語/OSS" class="internal-link" data-href="用語/OSS">OSS</a>です。</p>
</div>
