#問題 #テクノロジ #ネットワーク #通信プロトコル

<p class="ap-question"><strong>問34</strong>　IPネットワークのプロトコルのうち，OSI基本参照モデルのトランスポート層に位置するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　HTTP</div>
<div class="ap-choice-note">これはHyperText Transfer Protocolの説明です。WebブラウザとWebサーバ間のHTMLの送受信に使われるプロトコルで、<a href="用語/アプリケーション層" class="internal-link" data-href="用語/アプリケーション層">アプリケーション層</a>に位置します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　ICMP</div>
<div class="ap-choice-note">これはInternet Control Message Protocolの説明です。インターネット制御通知のプロトコルで、<a href="用語/ネットワーク層" class="internal-link" data-href="用語/ネットワーク層">ネットワーク層</a>に位置します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　SMTP</div>
<div class="ap-choice-note">これはSimple Mail Transfer Protocolの説明です。メール転送のためのプロトコルで、<a href="用語/アプリケーション層" class="internal-link" data-href="用語/アプリケーション層">アプリケーション層</a>に位置します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　UDP</div>
<div class="ap-choice-note">正しい。<a href="用語/UDP" class="internal-link" data-href="用語/UDP">UDP</a>は<a href="用語/トランスポート層" class="internal-link" data-href="用語/トランスポート層">トランスポート層</a>のプロトコルです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/UDP" class="internal-link" data-href="用語/UDP">UDP</a>(User Datagram Protocol)は、<a href="用語/TCP/IP" class="internal-link" data-href="用語/TCP/IP">TCP/IP</a>の通信処理で使われる伝送制御プロトコルのひとつで、<a href="用語/TCP" class="internal-link" data-href="用語/TCP">TCP</a>と同じく<a href="用語/OSI基本参照モデル" class="internal-link" data-href="用語/OSI基本参照モデル">OSI基本参照モデル</a>の<a href="用語/トランスポート層" class="internal-link" data-href="用語/トランスポート層">トランスポート層</a>に位置します。</p>
<p><a href="用語/UDP" class="internal-link" data-href="用語/UDP">UDP</a>は、<a href="用語/TCP" class="internal-link" data-href="用語/TCP">TCP</a>と異なり"コネクションレス型"の通信であり、通信制御に係る処理が少ない分だけ効率的な通信が可能になっています。この利点を活かして、信頼性よりも即時性や効率性が求められるストリーミング配信や、<a href="用語/DNS" class="internal-link" data-href="用語/DNS">DNS</a>、<a href="用語/NTP" class="internal-link" data-href="用語/NTP">NTP</a>、SNMPなどのプロトコルにおいて使用されています。</p>
</div>
