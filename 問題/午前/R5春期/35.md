<p class="ap-question"><strong>問35</strong>　モバイル通信サービスにおいて，移動中のモバイル端末が通信相手との接続を維持したまま，ある基地局経由から別の基地局経由の通信へ切り替えることを何と呼ぶか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　テザリング</div>
<div class="ap-choice-note">これは<a href="用語/テザリング" class="internal-link" data-href="用語/テザリング">テザリング</a>の説明です。スマートフォンなどのモバイル端末の携帯回線を使い、他の端末をインターネットに接続する機能です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　ハンドオーバー</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/ハンドオーバー" class="internal-link" data-href="用語/ハンドオーバー">ハンドオーバー</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　フォールバック</div>
<div class="ap-choice-note">これはフォールバックの説明です。5Gから4G、4Gから3Gへなど、電波環境に応じて利用する回線を落とす自動切替です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ローミング</div>
<div class="ap-choice-note">これは<a href="用語/ローミング" class="internal-link" data-href="用語/ローミング">ローミング</a>の説明です。契約事業者のエリア外でも他事業者のサービスで端末を使える仕組みです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ハンドオーバー" class="internal-link" data-href="用語/ハンドオーバー">ハンドオーバー</a>は、無線LAN端末やモバイル端末を使用しながら移動するときに、端末の場所に応じて通信する基地局やアクセスポイントを切り替えることです。<a href="用語/テザリング" class="internal-link" data-href="用語/テザリング">テザリング</a>は、スマートフォンなどのモバイル端末がもつ携帯回線などのインターネット接続機能を使って、他のコンピュータや情報端末をインターネットに接続することです。フォールバックは、通信エリアや電波環境に応じて、データ通信網を5G回線から4G回線に、あるいは4G回線から3G回線に落とし、適切な回線に自動的に切り替える技術です。<a href="用語/ローミング" class="internal-link" data-href="用語/ローミング">ローミング</a>は、契約している通信事業者のサービスエリア外でも、他の事業者のサービスによってモバイル端末を使用できるサービスです。</p>
</div>
