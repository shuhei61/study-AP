<p class="ap-question"><strong>問49</strong>　日本国特許庁において特許Aを取得した特許権者から，実施許諾を受けることが必要な場合はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　特許Aと同じ技術を家庭内で個人的に利用するだけの場合</div>
<div class="ap-choice-note"><a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>の侵害となるのは特許<a href="用語/発明" class="internal-link" data-href="用語/発明">発明</a>を業として実施する場合。事業でない個人的な使用や家庭内での使用であれば、<a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>者から実施許諾を受ける必要はない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　特許Aと同じ技術を利用して日本国内で製品を製造し，その全てを日本国外に輸出する場合</div>
<div class="ap-choice-note">正しい。<a href="用語/特許法" class="internal-link" data-href="用語/特許法">特許法</a>では<a href="用語/発明" class="internal-link" data-href="用語/発明">発明</a>の実施を、生産、使用、譲渡等、輸出、輸入又は譲渡等の申出をする行為と定義している。特許技術を使用した製品を輸出する場合、<a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>者から実施許諾を受けなければならない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　特許Aの出願日から25年を越えた後に，特許Aと同じ技術を新たに事業化する場合</div>
<div class="ap-choice-note"><a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>の存続期間は<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>日から20年、存続期間延長登録制度を利用しても最長5年しか延長できない。<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>日から25年を超えた特許については、<a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>者から実施許諾を受けなくても事業に使用できる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　特許Aの出願日より前から特許Aと同じ技術を独自に開発し，特許Aの出願日に日本国内でその技術を用いた製品を製造販売していたことが証明できる場合</div>
<div class="ap-choice-note">特許<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>に係る<a href="用語/発明" class="internal-link" data-href="用語/発明">発明</a>の内容を知らずに、特許<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>の際に現にその技術を用いた事業をしている者は、その特許<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>に関する<a href="用語/通常実施権" class="internal-link" data-href="用語/通常実施権">通常実施権</a>(先使用権)が認められる。この条件を満たせば先使用権が認められるため、<a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>者から実施許諾を受ける必要はない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>の効力は、特許<a href="用語/発明" class="internal-link" data-href="用語/発明">発明</a>を業として実施する行為を独占するものである。家庭内での個人的な利用のように事業性のない使用では、実施許諾は不要である。</p>
<p><a href="用語/特許法" class="internal-link" data-href="用語/特許法">特許法</a>上、<a href="用語/発明" class="internal-link" data-href="用語/発明">発明</a>の実施には生産、使用、譲渡等、輸出、輸入又は譲渡等の申出をする行為が含まれる。特許技術を用いて国内で製造した製品を輸出する場合は実施に当たるため、<a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>者から実施許諾を受ける必要がある。したがって「イ」が適切である。</p>
<p><a href="用語/特許権" class="internal-link" data-href="用語/特許権">特許権</a>の存続期間は原則として<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>日から20年であり、存続期間延長登録を用いても最長5年までの延長にとどまる。<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>日から25年を超えた時点では権利が消滅しているため、事業化に実施許諾は不要である。</p>
<p>また、特許<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>の内容を知らずに<a href="用語/出願" class="internal-link" data-href="用語/出願">出願</a>時点で既に当該技術による事業を行っていた者には、<a href="用語/通常実施権" class="internal-link" data-href="用語/通常実施権">通常実施権</a>(先使用権)が認められ、実施許諾なく当該事業を継続できる。</p>
</div>
