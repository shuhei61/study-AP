<p class="ap-question"><strong>問50</strong>　自社開発したソフトウェアの他社への使用許諾に関する説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　既に自社の製品に搭載して販売していると，ソフトウェア単体では使用許諾できない。</div>
<div class="ap-choice-note">販売されている製品に組み込まれていても、ソフトウェア自体が著作物と見なされるため単体で使用許諾できる</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　既にハードウェアと組み合わせて特許を取得していると，ソフトウェア単体では使用許諾できない。</div>
<div class="ap-choice-note">ハードウェアとソフトウェアは別々の著作物なので、それぞれについて使用許諾できる</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ソースコードを無償で使用許諾すると，無条件でオープンソースソフトウェアになる。</div>
<div class="ap-choice-note">無償での使用許諾が直ちにオープンソースソフトウェアになるわけではない</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　特許で保護された技術を使っていないソフトウェアであっても，使用許諾することは可能である。</div>
<div class="ap-choice-note">正しい。ソフトウェアは著作物であるため、<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>に基づき利用希望者との<a href="用語/ソフトウェア使用許諾契約" class="internal-link" data-href="用語/ソフトウェア使用許諾契約">ソフトウェア使用許諾契約</a>を交わすことが可能である</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>ソフトウェアは著作物として<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>で保護される。特許で保護された技術を使っていなくても、著作者の権利に基づき利用希望者との<a href="用語/ソフトウェア使用許諾契約" class="internal-link" data-href="用語/ソフトウェア使用許諾契約">ソフトウェア使用許諾契約</a>を交わすことができる。したがって「エ」が適切である。</p>
<p>アは、販売されている製品に組み込まれていても、ソフトウェア自体が著作物と見なされるため単体で使用許諾できる。イは、ハードウェアとソフトウェアは別々の著作物なので、それぞれについて使用許諾できる。ウは、無償での使用許諾が直ちにオープンソースソフトウェアになるわけではない。</p>
</div>
