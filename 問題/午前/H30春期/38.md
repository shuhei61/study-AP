<p class="ap-question"><strong>問38</strong>　ディレクトリトラバーサル攻撃はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　OSコマンドを受け付けるアプリケーションに対して，攻撃者が，ディレクトリを作成するOSコマンドの文字列を入力して実行させる。</div>
<div class="ap-choice-note">これは<a href="用語/OSコマンドインジェクション" class="internal-link" data-href="用語/OSコマンドインジェクション">OSコマンドインジェクション</a>攻撃の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　SQL文のリテラル部分の生成処理に問題があるアプリケーションに対して，攻撃者が，任意のSQL文を渡して実行させる。</div>
<div class="ap-choice-note">これは<a href="用語/SQLインジェクション" class="internal-link" data-href="用語/SQLインジェクション">SQLインジェクション</a>攻撃の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　シングルサインオンを提供するディレクトリサービスに対して，攻撃者が，不正に入手した認証情報を用いてログインし，複数のアプリケーションを不正使用する。</div>
<div class="ap-choice-note"><a href="用語/不正アクセス" class="internal-link" data-href="用語/不正アクセス">不正アクセス</a>の事例です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　入力文字列からアクセスするファイル名を組み立てるアプリケーションに対して，攻撃者が，上位のディレクトリを意味する文字列を入力して，非公開のファイルにアクセスする。</div>
<div class="ap-choice-note">正しい。<a href="用語/ディレクトリトラバーサル" class="internal-link" data-href="用語/ディレクトリトラバーサル">ディレクトリトラバーサル</a>攻撃の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ディレクトリトラバーサル" class="internal-link" data-href="用語/ディレクトリトラバーサル">ディレクトリトラバーサル</a>攻撃は、ファイル名を要求するプログラムに対してサーバ内の想定外のファイル名(親ディレクトリの移動「../」など)を直接指定することによって、本来許されないファイルの不正な閲覧・取得を狙う攻撃方法です。したがって正しい記述は「エ」です。</p>
</div>
