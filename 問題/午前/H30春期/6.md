<p class="ap-question"><strong>問6</strong>　異なるn個のデータが昇順に整列された表がある。この表をm個のデータごとのブロックに分割し，各ブロックの最後尾のデータだけを線形探索することによって，目的のデータの存在するブロックを探し出す。次に，当該ブロック内を線形探索して目的のデータを探し出す。このときの平均比較回数を表す式はどれか。ここで，mは十分大きく，nはmの倍数とし，目的のデータは必ず表の中に存在するものとする。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　m＋<span class="frac"><span>n</span><span>m</span></span></div>
<div class="ap-choice-note">1回目と2回目の比較対象数を単純に足した形で、平均（÷2）になっていない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　<span class="frac"><span>m</span><span>2</span></span>＋<span class="frac"><span>n</span><span>2m</span></span></div>
<div class="ap-choice-note">正しい。ブロック特定の平均 n／(2m) とブロック内探索の平均 m／2 の合計。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　<span class="frac"><span>n</span><span>m</span></span></div>
<div class="ap-choice-note">ブロック数だけで、平均化もブロック内の線形探索も含まない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　<span class="frac"><span>n</span><span>2m</span></span></div>
<div class="ap-choice-note">1回目の平均比較回数だけで、ブロック内の線形探索分を含まない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/線形探索法" class="internal-link" data-href="用語/線形探索法">線形探索法</a>を2回組み合わせて目的データを探し出すときの平均比較回数を求める問題です。探索するデータ数に注目して考えていきましょう。</p>
<p><a href="用語/線形探索法" class="internal-link" data-href="用語/線形探索法">線形探索法</a>は、探索対象データを先頭から順に1つずつ比較していくことで目的のデータを見つける方法です。<a href="用語/線形探索法" class="internal-link" data-href="用語/線形探索法">線形探索法</a>において、N個のデータの中から目的のデータを探すときの平均比較回数は <span class="frac"><span>N＋1</span><span>2</span></span> 回です。</p>
<p>最初は、n個のデータをm個ごとのブロックに分割した最後尾のデータのみを探索します。表のデータは昇順に整列されているので、各ブロック最後尾の並びも昇順になっているはずです。この最後尾データの並びに対して「目的のデータ≦各ブロックの最後尾データ」を順次チェックし、目的のデータが存在するブロックを探します。</p>
<p>この1回目の探索では、データを1つずつチェックしていくので線形探索の考え方を準用できます。探索するデータ数は n／m 個なので、目的のデータが存在するブロックが決定するまでの平均比較回数は、<br><br>　<span class="frac"><span>n／m＋1</span><span>2</span></span> [回]</p>
<p>2回目は、1回目の探索によって見つけたブロック内を線形探索します。探索対象のデータ数は m 個なので、目的のデータを見つけるまでの平均比較回数は、<br><br>　<span class="frac"><span>m＋1</span><span>2</span></span> [回]</p>
<p>2つの比較回数の合計は、<br><br>　<span class="frac"><span>n／m＋1</span><span>2</span></span> ＋ <span class="frac"><span>m＋1</span><span>2</span></span> [回]<br><br>ここまでたどり着ければ、なんとなく「イ」が適切そうなことがわかります。</p>
<p>設問では m は十分に大きいという条件が与えられていますが、n／m が十分に大きいとは限らないため、定数項の"＋1"は無視できません。よって、以下のように式を変形します。<br><br>　<span class="frac"><span>n／m＋1</span><span>2</span></span> ＋ <span class="frac"><span>m＋1</span><span>2</span></span><br>＝<span class="frac"><span>n／m＋1 ＋ m＋1</span><span>2</span></span><br>＝<span class="frac"><span>n／m＋m＋2</span><span>2</span></span><br>＝<span class="frac"><span>n</span><span>2m</span></span>＋<span class="frac"><span>m</span><span>2</span></span>＋1</p>
<p><span class="frac"><span>n</span><span>2m</span></span>は n が m より十分大きい場合に十分大きくなるので無視できない、"＋1"は <span class="frac"><span>m</span><span>2</span></span> が十分大きいので無視できると考えると、定数項の"＋1"を除いて <span class="frac"><span>n</span><span>2m</span></span>＋<span class="frac"><span>m</span><span>2</span></span>。つまり、1回目と2回目の平均比較回数の合計は <span class="frac"><span>m</span><span>2</span></span>＋<span class="frac"><span>n</span><span>2m</span></span> になります。</p>
<p>【補足】<br>上記の解説ではmが十分に大きいという理由で＋1を除外していますが、線形探索の特性上、目的のデータが必ず存在するならば、最後の1つ前（n－1番目）の要素までの比較で目的のデータが見つからなかった場合、最後の要素が目的のデータであることが確定します。このため最小比較回数を 1回、最大比較回数を n－1回、平均比較回数を n／2回 として考えることもできるでしょう。この考え方だと1回目の平均比較回数が n／2m回、2回目の平均比較回数が m／2回となり、正解の式と一致します。</p>
