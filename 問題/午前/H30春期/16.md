<p class="ap-question"><strong>問16</strong>　4種類の装置で構成される次のシステムの稼働率は，およそ幾らか。ここで，アプリケーションサーバとデータベースサーバの稼働率は0.8であり，それぞれのサーバのどちらかが稼働していればシステムとして稼働する。また，負荷分散装置と磁気ディスク装置は，故障しないものとする。<br><img class="ap-fig" src="../../../assets/問題/午前/H30春期/16/16.png" alt="問題図" width="337" height="189" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　0.64</div>
<div class="ap-choice-note">サーバ単体を直列接続とみなして 0.8×0.8 とした値です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　0.77</div>
<div class="ap-choice-note">本問の並列・直列の組合せでは得られない値です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　0.92</div>
<div class="ap-choice-note">正しい。各サーバ群の並列<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a> 0.96 を直列で掛け合わせた値です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　0.96</div>
<div class="ap-choice-note">サーバ群1組の並列<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>のみで止めた値です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>4種類の装置のうち、負荷分散装置と磁気ディスク装置は故障しないので、システム全体の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>はアプリケーションサーバ（以下、APサーバ）とデータベースサーバ（以下、DBサーバ）の部分だけによって決まります。そしてAPサーバとDBサーバは、2台のうちどちらかが稼働していればよいため、下図のように2台が並列に接続されているものと見なせます。</p>
<p><img class="ap-fig" src="../../../assets/問題/午前/H30春期/16/16_1.png" alt="解説図 1" width="292" height="98" ></p>
<p>装置単体の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>がRのとき、2台が並列に接続されている場合の全体としての<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>は「1－(1－R)<sup>2</sup>」、直列の場合は「R<sup>2</sup>」で求められます。この公式を使って、まずは並列で接続されている部分の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>を計算します。</p>
<p>1－(1－0.8)<sup>2</sup>＝1－0.04＝0.96</p>
<p>APサーバ群、DBサーバ群の<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>はともに0.96とわかります。そして<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>0.96のサーバ群同士が直列で接続されているので、システム全体としての<a href="用語/稼働率" class="internal-link" data-href="用語/稼働率">稼働率</a>は、</p>
<p>0.96×0.96＝0.9216≒0.92</p>
<p>したがって「ウ」が正解です。</p>
