<p class="ap-question"><strong>問64</strong>　システム管理基準(平成16年)によれば，情報戦略策定段階の成果物はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　関連する他の情報システムと役割を分担し，組織体として最大の効果を上げる機能を実現するために，全体最適化計画との整合性を考慮して策定する開発計画</div>
<div class="ap-choice-note">開発計画は企画業務における成果物とされています。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　経営戦略に基づいて組織体全体で整合性及び一貫性を確保した情報化を推進するために，方針及び目標に基づいて策定する全体最適化計画</div>
<div class="ap-choice-note">正しい。情報戦略策定段階の成果物は全体最適化計画です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　情報システムの運用を円滑に行うために，運用設計及び運用管理ルールに基づき，さらに規模，期間，システム特性を考慮して策定する運用手順</div>
<div class="ap-choice-note"><a href="用語/運用手順" class="internal-link" data-href="用語/運用手順">運用手順</a>は運用業務における成果物とされています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　組織体として一貫し，効率的な開発作業を確実に遂行するために，組織体として標準化された開発方法に基づいて策定する開発手順</div>
<div class="ap-choice-note">開発手順は開発業務における成果物とされています。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/システム管理基準" class="internal-link" data-href="用語/システム管理基準">システム管理基準</a>(平成16年)によれば，情報戦略では，情報化投資の原則や全体最適化の方針・目標を明確にし，経営方針や<a href="用語/中長期計画" class="internal-link" data-href="用語/中長期計画">中長期計画</a>との整合性をもった全体最適化計画を策定するとされています。全体最適化計画とは，組織内の個々のシステムが有機的に関連し，相互の整合性を保ちながら効率的かつ効果的に目的を達成するように，組織体全体の情報システムをあるべき姿に改善していくための計画です。</p>
<p>したがって，情報戦略策定段階の成果物はイの全体最適化計画です。アの開発計画は企画業務，ウの<a href="用語/運用手順" class="internal-link" data-href="用語/運用手順">運用手順</a>は運用業務，エの開発手順は開発業務における成果物とされています。</p>
</div>
