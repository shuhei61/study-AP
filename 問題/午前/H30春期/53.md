# [平成30年春期 午前 問53](https://www.ap-siken.com/kakomon/30_haru/q53.html)

#問題 #マネジメント #プロジェクトマネジメント #プロジェクトの資源

<div class="ap-quiz">
<details class="ap-reveal">
<summary class="ap-quiz-btn"><span class="ap-label-show">解説を表示</span><span class="ap-label-hide">解説を隠す</span></summary>
</details>
<p class="ap-question"><strong>問53</strong>　ISO 21500:2012(プロジェクトマネジメントの手引き(英和対訳版))によれば，プロジェクト資源マネジメントにおけるプロセス"プロジェクトチームの育成"の目的はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　継続的にプロジェクトチームのメンバーのパフォーマンス及び相互関係を改善する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/プロジェクトチーム" class="internal-link" data-href="用語/プロジェクトチーム">プロジェクトチーム</a>の育成</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　プロジェクトチームの全てのメンバーの役割，責任及び権限を決め，決定事項に対して全てのメンバーからコミットメントを得る。</div>
<div class="ap-choice-note">これは<a href="用語/プロジェクト組織" class="internal-link" data-href="用語/プロジェクト組織">プロジェクト組織</a>の決定の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　プロジェクトチームのパフォーマンスを評価分析し，分析結果をフィードバックして問題を解決し，変更を調整する。</div>
<div class="ap-choice-note">これは<a href="用語/プロジェクトチーム" class="internal-link" data-href="用語/プロジェクトチーム">プロジェクトチーム</a>の管理の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プロジェクトの完遂に必要な能力をもったプロジェクトチームのメンバーを招集し，人的資源を得る。</div>
<div class="ap-choice-note">これは<a href="用語/プロジェクトチーム" class="internal-link" data-href="用語/プロジェクトチーム">プロジェクトチーム</a>の結成の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>ISO 21500:2012には、<a href="用語/プロジェクト" class="internal-link" data-href="用語/プロジェクト">プロジェクト</a>資源マネジメントのプロセスとして、<a href="用語/プロジェクト組織" class="internal-link" data-href="用語/プロジェクト組織">プロジェクト組織</a>の決定、<a href="用語/資源の見積り" class="internal-link" data-href="用語/資源の見積り">資源の見積り</a>、<a href="用語/プロジェクトチーム" class="internal-link" data-href="用語/プロジェクトチーム">プロジェクトチーム</a>の結成、<a href="用語/プロジェクトチーム" class="internal-link" data-href="用語/プロジェクトチーム">プロジェクトチーム</a>の育成、資源のコントロール、<a href="用語/プロジェクトチーム" class="internal-link" data-href="用語/プロジェクトチーム">プロジェクトチーム</a>の管理、という5つが定義されています。よって「ア」が正解です。</p>
</div>
