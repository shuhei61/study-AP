<p class="ap-question"><strong>問4</strong>　サンプリング周波数40kHz，量子化ビット数16ビットでA/D変換したモノラル音声の1秒間のデータ量は，何kバイトとなるか。ここで， 1kバイトは1,000バイトとする。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　20</div>
<div class="ap-choice-note"><a href="用語/量子化ビット数" class="internal-link" data-href="用語/量子化ビット数">量子化ビット数</a>を掛けず，<a href="用語/サンプリング" class="internal-link" data-href="用語/サンプリング">サンプリング</a>回数だけをk単位で数えた誤り（40,000÷2,000＝20）など。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　40</div>
<div class="ap-choice-note">ビットをバイトに直す際に8ではなく16で割った誤り（640kビット÷16＝40kバイト）。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　80</div>
<div class="ap-choice-note">正しい。40,000回×16ビット÷8＝80kバイト。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　640</div>
<div class="ap-choice-note">640kビットのままバイト換算（÷8）をしていない誤り。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>アナログデータからデジタルデータへの変換では、標本化、量子化、符号化の3段階の処理を行います。</p>
<p><strong>標本化</strong> … 時間的に連続したアナログ信号(振幅、周波数、電圧など)を一定の時間間隔で測定する</p>
<p><strong>量子化</strong> … 標本化で得られた数値を整数などの離散値で近似する</p>
<p><strong>符号化</strong> … 量子化した整数値を2進数のビットに対応付ける</p>
<p><a href="用語/サンプリング" class="internal-link" data-href="用語/サンプリング">サンプリング</a>周波数とは、標本化において、1秒間にアナログ音声からデータを取得する回数を示します。また<a href="用語/量子化ビット数" class="internal-link" data-href="用語/量子化ビット数">量子化ビット数</a>は、量子化において、1回ごとの取得データを何ビットで表現するかを示します。</p>
<p>つまり、この問題の<a href="用語/A/D変換" class="internal-link" data-href="用語/A/D変換">A/D変換</a>は以下の条件で行うことになります。</p>
<ul>
<li><a href="用語/サンプリング" class="internal-link" data-href="用語/サンプリング">サンプリング</a>周波数40kHz → 1秒間に40,000回データを取得する</li>
<li><a href="用語/量子化ビット数" class="internal-link" data-href="用語/量子化ビット数">量子化ビット数</a>16ビット → 各々の<a href="用語/サンプリング" class="internal-link" data-href="用語/サンプリング">サンプリング</a>データを16ビットで表現する</li>
</ul>
<p>1秒間のデータ量は「<a href="用語/サンプリング" class="internal-link" data-href="用語/サンプリング">サンプリング</a>回数×<a href="用語/量子化ビット数" class="internal-link" data-href="用語/量子化ビット数">量子化ビット数</a>」で求められるので、<br>40,000回×16ビット＝640,000ビット＝640kビット</p>
<p>単位をバイトに変換して、<br>640kビット÷8＝<strong>80kバイト</strong></p>
<p>したがって正解は「ウ」です。</p>
</div>
