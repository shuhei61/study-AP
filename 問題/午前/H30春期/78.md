<p class="ap-question"><strong>問78</strong>　製造物責任法(PL法)において，製造物責任を問われる事例はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　機器に組み込まれているROMに記録されたプログラムに欠陥があったので，その機器の使用者に大けがをさせた。</div>
<div class="ap-choice-note">正しい。ソフトウェア自体は対象外ですが、それを組み込んだハードウェアの使用者がソフトウェアの<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>により損害を被った場合には製造物責任を問われます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　工場に配備されている制御系コンピュータのオペレーションを誤ったので，製品製造のラインを長時間停止させ大きな損害を与えた。</div>
<div class="ap-choice-note">役務（サービス）は無体物なので対象外です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ソフトウェアパッケージに重大な欠陥が発見され，修復に時間が掛かったので，販売先の業務に大混乱をもたらした。</div>
<div class="ap-choice-note">ソフトウェア単体は無体物なので対象外です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　提供しているITサービスのうち，ヘルプデスクサービスがSLAを満たす品質になく，顧客から多大なクレームを受けた。</div>
<div class="ap-choice-note">ITサービスは無体物なので対象外です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>製造物責任法(<a href="用語/PL法" class="internal-link" data-href="用語/PL法">PL法</a>)は、製造物の安全性上の<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>により人の生命、身体、財産に被害が生じた場合に、消費者が製造業者に対して損害賠償を請求できることを定めた法律です。</p>
<p>製造物責任法では製造物を「製造又は加工された動産」と定義しています。このため、サービス、不動産、未加工のものは、この定義上の製造物には含まれず<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>があったとしても製造物責任法の対象外となります。同じくコンピュータプログラムのような無体物も動産ではないため適用対象外となりますが、<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>があるプログラムを組み込んだハードウェア(製造物)の使用によって損害が生じた場合は、その動産であるハードウェアに<a href="用語/欠陥" class="internal-link" data-href="用語/欠陥">欠陥</a>があるものとして製造物責任法の対象となります。</p>
