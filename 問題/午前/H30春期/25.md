<p class="ap-question"><strong>問25</strong>　3次元の物体を表すコンピュータグラフィックスの手法に関する記述のうち，サーフェスモデルの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　物体を，頂点と頂点をつなぐ線で結び、針金で構成されているように表現する。</div>
<div class="ap-choice-note">これはワイヤフレームモデルの説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　物体を，中身の詰まった固形物として表現する。</div>
<div class="ap-choice-note">これはソリッドモデルの説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　物体を，ポリゴンや曲面パッチを用いて表現する。</div>
<div class="ap-choice-note">正しい。詳細：サーフェスモデル</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　物体を，メタボールと呼ぶ構造を使い、球体を変形させることで得られる滑らかな曲線で表現する。</div>
<div class="ap-choice-note">これはメタボールの説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>サーフェスモデルは、3次元の<a href="用語/CG" class="internal-link" data-href="用語/CG">CG</a>において表面のみが定義された3次元構造、またはそれらを作成する目的のモデリング体系のことです。中身が詰まっていないため張り子、張りぼてとも形容されます。</p><p><img class="ap-fig" src="../../../assets/問題/午前/H30春期/25/25.png" alt="解説図" width="478" height="131" ></p>
</div>
