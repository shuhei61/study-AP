<p class="ap-question"><strong>問52</strong>　PMBOKガイド第5版によれば，プロジェクトのコスト見積り技法の説明のうち，パラメトリック見積りの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　WBSの下位レベルの構成要素単位の見積り結果を集計して，プロジェクトのコストを見積もる。</div>
<div class="ap-choice-note">これは<a href="用語/ボトムアップ見積り" class="internal-link" data-href="用語/ボトムアップ見積り">ボトムアップ見積り</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　関連する過去のデータとその他の変数との統計的関係を用いて，プロジェクトにおける作業のコストを見積もる。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/パラメトリック見積り" class="internal-link" data-href="用語/パラメトリック見積り">パラメトリック見積り</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　楽観値，悲観値，最可能値を使って，個々のアクティビティのコストを見積もる。</div>
<div class="ap-choice-note">これは<a href="用語/三点見積り" class="internal-link" data-href="用語/三点見積り">三点見積り</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　類似のプロジェクトにおける過去のコスト実績を使って，プロジェクトのコストを見積もる。</div>
<div class="ap-choice-note">これは<a href="用語/類推見積り" class="internal-link" data-href="用語/類推見積り">類推見積り</a>の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/パラメトリック見積り" class="internal-link" data-href="用語/パラメトリック見積り">パラメトリック見積り</a>(係数見積り)は、過去に蓄積したデータと関連する変数との統計上の関係を使って、見積りを行う手法です。試験でよく登場する"<a href="用語/ファンクションポイント法" class="internal-link" data-href="用語/ファンクションポイント法">ファンクションポイント法</a>"や"<a href="用語/COCOMO" class="internal-link" data-href="用語/COCOMO">COCOMO</a>"及び"プログラムステップ法"も<a href="用語/パラメトリック見積り" class="internal-link" data-href="用語/パラメトリック見積り">パラメトリック見積り</a>の一種です。</p>
<p>IPAのWebページ内では、「工数などを<a href="用語/目的変数" class="internal-link" data-href="用語/目的変数">目的変数</a>として、<a href="用語/説明変数" class="internal-link" data-href="用語/説明変数">説明変数</a>に規模や要因などを設定し、数学的な関数として表す方法」と説明されています。</p>
