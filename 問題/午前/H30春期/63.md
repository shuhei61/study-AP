<p class="ap-question"><strong>問63</strong>　定性的な評価項目を定量化するために評価点を与える方法がある。表に示す4段階評価を用いた場合，重み及び判定内容から評価されるシステム全体の目標達成度は何%となるか。<br><img class="ap-fig" src="../../../assets/問題/午前/H30春期/63/63.png" alt="問題図" width="279" height="132" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　27</div>
<div class="ap-choice-note">評価点の合計（27点）そのものであり、満点75点に対する達成率ではない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　36</div>
<div class="ap-choice-note">正しい。27点÷75点＝0.36＝36%。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　43</div>
<div class="ap-choice-note">重み付け<a href="用語/総合評価" class="internal-link" data-href="用語/総合評価">総合評価</a>の計算では得られない値です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　52</div>
<div class="ap-choice-note">重み付け<a href="用語/総合評価" class="internal-link" data-href="用語/総合評価">総合評価</a>の計算では得られない値です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>重み付け<a href="用語/総合評価" class="internal-link" data-href="用語/総合評価">総合評価</a>法は、各評価項目に重要度を設定し、その重みを考慮して<a href="用語/総合評価" class="internal-link" data-href="用語/総合評価">総合評価</a>点を求め、点数の大小で比較する手法です。</p>
<p>まず、各評価項目ごとに判定内容に重みを付けて評価点を算出します。</p>
<ul>
<li>省電力効果　3×5＝15点</li>
<li>期間の短縮　0×8＝0点</li>
<li>情報の統合化　1×12＝12点</li>
<li>評価点の合計　15＋0＋12＝27点</li>
</ul>
<p>次に、全ての評価項目が「目標通り」だったときの評価点を求めます。</p>
<ul>
<li>省電力効果　3×5＝15点</li>
<li>期間の短縮　3×8＝24点</li>
<li>情報の統合化　3×12＝36点</li>
<li>評価点の合計　15＋24＋36＝75点</li>
</ul>
<p>満点75点に対し、評価値は27点なので、全て目標通りだった場合の評価点に対する達成率は、<br>27点÷75点＝0.36＝36%<br>以上より、正解は「イ」となります。</p>
</div>
