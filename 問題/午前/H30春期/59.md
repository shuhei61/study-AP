<p class="ap-question"><strong>問59</strong>　企業において整備したシステム監査規程の最終的な承認者として，最も適切な者は誰か。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　監査対象システムの利用部門の長</div>
<div class="ap-choice-note">監査対象側の長であり、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>規程の最終承認者としては適切ではない</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　経営者</div>
<div class="ap-choice-note">正しい。<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準では最終承認者は経営層とされている</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　情報システム部門の長</div>
<div class="ap-choice-note">被監査側になり得る部門の長であり、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>規程の最終承認者としては適切ではない</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　被監査部門の長</div>
<div class="ap-choice-note">被監査側の長であり、監査の独立性の観点からも最終承認者としては適切ではない</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>規程は、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>を実施する目的及び対象範囲、並びに<a href="用語/システム監査人の権限と責任" class="internal-link" data-href="用語/システム監査人の権限と責任">システム監査人の権限と責任</a>など<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の基本的事項を定めた規程であり、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人が<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>を実施するにあたり必要とする一連の監査マニュアルの一つを構成するものです。</p>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>規程には次のような事項が文書化されます(<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準(平成30年)より)。</p>
<ul>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の目的に関する事</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の対象範囲に関する事項</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の報告に関する事項</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人の権限・責任に関する事項</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>実施のための監査資源（監査時間、監査要員、監査費用等）の確保に関する事項</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人の各種設備や情報資産等へのアクセス権限に関する事項</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>業務の委託の可否に関する事項</li>
<li><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人の守秘義務に関する事項</li>
</ul>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準(平成30年)によれば、「<a href="用語/内部監査規程" class="internal-link" data-href="用語/内部監査規程">内部監査規程</a>等の承認を行う主体は、組織体の個々の事情に応じて様々な形態をとりうる。…<a href="用語/取締役" class="internal-link" data-href="用語/取締役">取締役</a>等又は<a href="用語/取締役" class="internal-link" data-href="用語/取締役">取締役</a>会(監査委員会又は監査等委員会を含む。)でなされるかもしれない」としており、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>規程の最終承認者が経営層である旨を記載しています。</p>
<p>したがって最終的な承認者としては「経営者」が適切です。</p>
