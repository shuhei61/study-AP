<p class="ap-question"><strong>問14</strong>　物理サーバのスケールアウトに関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　サーバのCPUを高性能なものに交換することによって，サーバ当たりの処理能力を向上させること</div>
<div class="ap-choice-note">これは<a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　サーバの台数を増やして負荷分散することによって，サーバ群としての処理能力を向上させること</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/スケールアウト" class="internal-link" data-href="用語/スケールアウト">スケールアウト</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　サーバのディスクを増設して冗長化することによって，サーバ当たりの信頼性を向上させること</div>
<div class="ap-choice-note">これは<a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　サーバのメモリを増設することによって，単位時間当たりの処理能力を向上させること</div>
<div class="ap-choice-note">これは<a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>サーバの<a href="用語/処理能力" class="internal-link" data-href="用語/処理能力">処理能力</a>を向上させる施策には「<a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>」と「<a href="用語/スケールアウト" class="internal-link" data-href="用語/スケールアウト">スケールアウト</a>」という2つのアプローチがあります。</p>
<p><a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>は、サーバを構成する各処理装置をより性能の高いものに交換したり，プロセッサの数などを増やすことでサーバ当たりの<a href="用語/処理能力" class="internal-link" data-href="用語/処理能力">処理能力</a>を向上させる。</p>
<p><a href="用語/スケールアウト" class="internal-link" data-href="用語/スケールアウト">スケールアウト</a>は、接続されるサーバの台数を増やすことでサーバ群全体としての<a href="用語/処理能力" class="internal-link" data-href="用語/処理能力">処理能力</a>や<a href="用語/可用性" class="internal-link" data-href="用語/可用性">可用性</a>を向上させる。同等の性能アップであれば<a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>よりも低コストであることが多い。</p>
<p>これを踏まえると各事例を2つに分類できます。「ア」「ウ」「エ」は<a href="用語/スケールアップ" class="internal-link" data-href="用語/スケールアップ">スケールアップ</a>の例、「イ」は<a href="用語/スケールアウト" class="internal-link" data-href="用語/スケールアウト">スケールアウト</a>の例です。したがって「イ」が正解です。</p>
