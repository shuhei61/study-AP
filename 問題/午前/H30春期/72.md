<p class="ap-question"><strong>問72</strong>　図は，製品Aの構成部品を示し，括弧内の数字は上位の製品・部品1個当たりの所要数量である。この製品Aを10個生産する場合，部品Cは，少なくとも何個発注する必要があるか。ここで，現在の部品Bの在庫は0個，部品Cの在庫は5個である。<br><img class="ap-fig" src="../../../assets/問題/午前/H30春期/72/72.png" alt="問題図" width="236" height="85" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　15</div>
<div class="ap-choice-note">部品B由来の部品Cだけから在庫を差し引いた場合（20－5＝15）で、製品Aに直接必要な部品Cを含めていない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　20</div>
<div class="ap-choice-note">部品Bの生産に必要な部品Cの数量のみで、製品Aへの直接所要と在庫の控除を反映していない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　25</div>
<div class="ap-choice-note">正しい。構成部品に沿って所要量を積み上げ、部品Cの在庫5個を差し引いた発注数である。詳細：<a href="用語/MRP" class="internal-link" data-href="用語/MRP">MRP</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　30</div>
<div class="ap-choice-note">部品Cの総所要量（10＋20＝30）であり、在庫5個を差し引いていない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>問題文と図の条件を整理します。製品Aを1つ生産するためには、部品Bが2つ、部品Cが1つ必要となる。部品Bを1つ生産するためには、部品C、部品D、部品Eがそれぞれ1つ必要となる。部品Bの在庫は0個、部品Cの在庫は5個。まず製品Aを10個生産するためには、部品Bが20個、部品Cが10個必要であるとわかります。部品Bの在庫は0個なので全20個生産するために、部品C、D、Eがそれぞれ20個必要になります。この関係を視覚化したものが以下の図です。生産に必要となる部品Cの数量は、製品Aの生産に必要な10個と部品Bの生産に必要な20個を合わせた30個です。部品Cの在庫は5個なので、発注数は所要数30個から在庫5個を差し引いた「30－5＝25個」になります。</p><p><img class="ap-fig" src="../../../assets/問題/午前/H30春期/72/72a.png" alt="解説図" width="270" height="142" ></p>
</div>
