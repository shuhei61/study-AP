<p class="ap-question"><strong>問7</strong>　オブジェクト指向プログラミングにおいて，同一クラス内に，メソッド名が同一であって，引数の型，個数又は並び順が異なる複数のメソッドを定義することを何と呼ぶか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　オーバーライド</div>
<div class="ap-choice-note">詳細：<a href="用語/オーバーライド" class="internal-link" data-href="用語/オーバーライド">オーバーライド</a>。<a href="用語/スーパークラス" class="internal-link" data-href="用語/スーパークラス">スーパークラス</a>で定義された<a href="用語/メソッド" class="internal-link" data-href="用語/メソッド">メソッド</a>を<a href="用語/サブクラス" class="internal-link" data-href="用語/サブクラス">サブクラス</a>で再定義することです。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　オーバーロード</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/オーバーロード" class="internal-link" data-href="用語/オーバーロード">オーバーロード</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　カプセル化</div>
<div class="ap-choice-note">詳細：<a href="用語/カプセル化" class="internal-link" data-href="用語/カプセル化">カプセル化</a>。オブジェクト内の詳細な仕様や構造を外部から隠蔽することです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　汎化</div>
<div class="ap-choice-note">詳細：<a href="用語/汎化" class="internal-link" data-href="用語/汎化">汎化</a>。複数の<a href="用語/クラス" class="internal-link" data-href="用語/クラス">クラス</a>の共通する性質をまとめて，抽象化した<a href="用語/クラス" class="internal-link" data-href="用語/クラス">クラス</a>を作ることです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>【ア】<a href="用語/オーバーライド" class="internal-link" data-href="用語/オーバーライド">オーバーライド</a>は、<a href="用語/スーパークラス" class="internal-link" data-href="用語/スーパークラス">スーパークラス</a>で定義された<a href="用語/メソッド" class="internal-link" data-href="用語/メソッド">メソッド</a>を<a href="用語/サブクラス" class="internal-link" data-href="用語/サブクラス">サブクラス</a>で再定義することです。</p>
<p>【イ】正しい。<a href="用語/オーバーロード" class="internal-link" data-href="用語/オーバーロード">オーバーロード</a>は、ある<a href="用語/クラス" class="internal-link" data-href="用語/クラス">クラス</a>内で引数や型が異なる同じ名前の<a href="用語/メソッド" class="internal-link" data-href="用語/メソッド">メソッド</a>を定義することです。<br>例)<br>string date(string format); //現在時刻を書式化して返す<br>string date(string format, int time_stamp); //指定時刻で<br>array date(array format, array time_stamp); //配列で一括処理</p>
<p>【ウ】<a href="用語/カプセル化" class="internal-link" data-href="用語/カプセル化">カプセル化</a>は、オブジェクト内の詳細な仕様や構造を外部から隠蔽することです。</p>
<p>【エ】<a href="用語/汎化" class="internal-link" data-href="用語/汎化">汎化</a>は、複数の<a href="用語/クラス" class="internal-link" data-href="用語/クラス">クラス</a>の共通する性質をまとめて，抽象化した<a href="用語/クラス" class="internal-link" data-href="用語/クラス">クラス</a>を作ることです。</p>
</div>
