<p class="ap-question"><strong>問30</strong>　CSMA/CD方式のLANに接続されたノードの送信動作に関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　各ノードに論理的な順位付けを行い，送信権を順次受け渡し，これを受け取ったノードだけが送信を行う。</div>
<div class="ap-choice-note">これは<a href="用語/優先度" class="internal-link" data-href="用語/優先度">優先度</a>制御方式の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　各ノードは伝送媒体が使用中かどうかを調べ，使用中でなければ送信を行う。衝突を検出したらランダムな時間の経過後に再度送信を行う。</div>
<div class="ap-choice-note">正しい。<a href="用語/CSMA/CD" class="internal-link" data-href="用語/CSMA/CD">CSMA/CD</a>方式の送信動作です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　各ノードを環状に接続して，送信権を制御するための特殊なフレームを巡回させ，これを受け取ったノードだけが送信を行う。</div>
<div class="ap-choice-note">これは<a href="用語/トークンパッシング" class="internal-link" data-href="用語/トークンパッシング">トークンパッシング</a>方式の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　タイムスロットを割り当てられたノードだけが送信を行う。</div>
<div class="ap-choice-note">これは<a href="用語/TDMA" class="internal-link" data-href="用語/TDMA">TDMA</a>方式の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/CSMA/CD" class="internal-link" data-href="用語/CSMA/CD">CSMA/CD</a>方式は、Carrier Sense Multiple Access with Collision Detectionの略で、日本語に訳すと「搬送波感知多重アクセス／<a href="用語/衝突" class="internal-link" data-href="用語/衝突">衝突</a>検出方式」となります。<a href="用語/CSMA/CD" class="internal-link" data-href="用語/CSMA/CD">CSMA/CD</a>方式は一本の<a href="用語/同軸ケーブル" class="internal-link" data-href="用語/同軸ケーブル">同軸ケーブル</a>に複数のコンピュータが接続されている<a href="用語/バス型" class="internal-link" data-href="用語/バス型">バス型</a>のトポロジを前提としており、次のような手順で送信が行われます。伝送路上に他のノードから<a href="用語/フレーム" class="internal-link" data-href="用語/フレーム">フレーム</a>が送出されていないかを確認する複数のクライアントは同じ回線を共用し、他者が通信をしていなければ自分の通信を開始する。複数の通信が同時に行われた場合は<a href="用語/衝突" class="internal-link" data-href="用語/衝突">衝突</a>を検出し、送信を中止してランダム時間待ってから再び送信をする。</p>
