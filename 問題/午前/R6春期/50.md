<p class="ap-question"><strong>問50</strong>　ドキュメンテーションジェネレーターの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　HTML，CSSなどのリソースを読み込んで，画面などに描画又は表示するソフトウェア</div>
<div class="ap-choice-note">これは<a href="用語/Webブラウザ" class="internal-link" data-href="用語/Webブラウザ">Webブラウザ</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　ソースコード中にある，フォーマットに従って記述したコメント文などから，プログラムのドキュメントを生成するソフトウェア</div>
<div class="ap-choice-note">正しい。ドキュメンテーションジェネレーターの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　動的にWebページを生成するために，文書のテンプレートと埋込み入力データを合成して出力するソフトウェア</div>
<div class="ap-choice-note">これはテンプレートエンジンの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　文書構造がマーク付けされたテキストファイルを読み込んで，印刷可能なドキュメントを組版するソフトウェア</div>
<div class="ap-choice-note">これはTeXの説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>ドキュメンテーションジェネレーターは、開発者が書いたコードやコメントを解析した情報から、プログラムや<a href="用語/API" class="internal-link" data-href="用語/API">API</a>のドキュメントを生成するソフトウェアです。所定の構文でコメントを付けることにより<a href="用語/関数" class="internal-link" data-href="用語/関数">関数</a>や<a href="用語/クラス" class="internal-link" data-href="用語/クラス">クラス</a>、変数などの説明を含むドキュメントを自動的に作成できるので、<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>保守のために利用されているほか、<a href="用語/API" class="internal-link" data-href="用語/API">API</a>リファレンスを自動生成するためにも使用されています。Java言語向けのJavadoc、Python向けのSphinx、JavaScript向けのJSDocなどがあります。</p>
</div>
