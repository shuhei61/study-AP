<p class="ap-question"><strong>問66</strong>　組込み機器のハードウェアの製造を外部に委託する場合のコンティンジェンシープランの記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　実績のある外注先の利用によって，リスクの発生確率を低減する。</div>
<div class="ap-choice-note">調達リスク分析と<a href="用語/リスク対応" class="internal-link" data-href="用語/リスク対応">リスク対応</a>に関する記述です。コンティンジェンシープランではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　製造品質が担保されていることを確認できるように委託先と契約する。</div>
<div class="ap-choice-note"><a href="用語/調達" class="internal-link" data-href="用語/調達">調達</a>の<a href="用語/契約" class="internal-link" data-href="用語/契約">契約</a>締結に関する記述です。コンティンジェンシープランではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　複数の会社の見積りを比較検討して，委託先を選定する。</div>
<div class="ap-choice-note"><a href="用語/調達" class="internal-link" data-href="用語/調達">調達</a>先の選定に関する記述です。コンティンジェンシープランではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　部品調達のリスクが顕在化したときに備えて，対処するための計画を策定する。</div>
<div class="ap-choice-note">正しい。コンティンジェンシープランに関する記述です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>コンティンジェンシープランは、事故や災害など予期せぬ<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>が現実化した場合に、自社が被る損害を最小限にとどめて、速やかに事態の克服を図るための緊急事態対応計画です。既知の<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>に対する対応計画の一部として策定されるので、プランの発動条件となる事象とそのとき実行する対応策や行動手順を記述します。</p>
<p>したがって、既知の<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>が顕在化したときの計画を策定している「エ」が、コンティンジェンシープランに関する記述として適切です。</p>
</div>
