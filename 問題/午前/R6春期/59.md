<p class="ap-question"><strong>問59</strong>　システム監査基準(令和5年)によれば，システム監査において，監査人が一定の基準に基づいて総合的に点検評価を行う対象とするものは，情報システムのマネジメント，コントロールと，あと一つはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　ガバナンス</div>
<div class="ap-choice-note">正しい。<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準(令和5年)では、点検・評価・検証の対象はガバナンス、マネジメント、<a href="用語/コントロール" class="internal-link" data-href="用語/コントロール">コントロール</a>の3つである。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　コンプライアンス</div>
<div class="ap-choice-note">詳細：<a href="用語/コンプライアンス" class="internal-link" data-href="用語/コンプライアンス">コンプライアンス</a>。本問の点検評価対象の3要素には含まれない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　サイバーレジリエンス</div>
<div class="ap-choice-note">サイバー攻撃等への耐性・復旧力を指す概念であり、本問の点検評価対象の3要素には含まれない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　モニタリング</div>
<div class="ap-choice-note">詳細：<a href="用語/モニタリング" class="internal-link" data-href="用語/モニタリング">モニタリング</a>。本問の点検評価対象の3要素には含まれない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準(令和5年)は、有効かつ効率的な<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>を実現するための<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人の行為規範となるガイドラインです。本基準では、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>とは、専門性と客観性を備えた<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人が、一定の基準に基づいて情報システムのガバナンス、マネジメントまたは<a href="用語/コントロール" class="internal-link" data-href="用語/コントロール">コントロール</a>を点検・評価・検証して、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>報告の利用者にその適切性等に対する保証を与える、または改善のための助言を行うものとしています。これは、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>の判断尺度となる<a href="用語/システム管理基準" class="internal-link" data-href="用語/システム管理基準">システム管理基準</a>が令和5年版で<a href="用語/ITガバナンス" class="internal-link" data-href="用語/ITガバナンス">ITガバナンス</a>編とITマネジメント編に再構成されたことに対応しています。</p>
<p>したがって「ア」のガバナンスが適切です。</p>
