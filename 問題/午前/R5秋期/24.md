<p class="ap-question"><strong>問24</strong>　JIS X 9303-1:2006(ユーザーシステムインタフェース及びシンボル－アイコン及び機能－アイコン一般)で規定されているアイコンの習得性の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　アイコンによって表現されたシステム機能が，それが理解された後に，どれだけ容易に思い出すことができるかを示す。</div>
<div class="ap-choice-note">正しい。習得性(learnability)の説明です。<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>の意味やそれに関連付けられた機能を覚えやすいことを示します。ハートは"お気に入り"、ゴミ箱が"削除"など、機能と<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>が容易に結び付くほうが良いデザインと言えます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　アイコンの図柄の詳細を，どれだけ容易に区別できるかを示す。</div>
<div class="ap-choice-note">これは判読性(legibility)の説明です。<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>が明瞭で読み取りやすいことを示します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　同じ又は類似したアイコンによる以前の経験に基づいて，どれだけ容易にアイコンを識別できるかを示す。</div>
<div class="ap-choice-note">これは認識性(recognisability)の説明です。<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>の意味を迅速に認識できることを示します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　空間的，時間的又は文脈的に近くに表示された別のアイコンから，与えられたアイコンをどれだけ容易に区別できるかを示す。</div>
<div class="ap-choice-note">これは識別性(discriminability)の説明です。他の<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>と混同しないように設計されていることを示します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/JIS" class="internal-link" data-href="用語/JIS">JIS</a> X 9303-1:2006 では<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>の<a href="用語/評価指標" class="internal-link" data-href="用語/評価指標">評価指標</a>として習得性・判読性・認識性・識別性などが定義されています。本問の習得性(learnability)は、一度理解した<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>の意味や関連機能を、後からどれだけ容易に思い出せるかを表します。判読性は図柄の読み取りやすさ、認識性は過去の経験に基づく意味の把握、識別性は近接する他<a href="用語/アイコン" class="internal-link" data-href="用語/アイコン">アイコン</a>との区別のしやすさです。</p>
