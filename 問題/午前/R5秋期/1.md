<p class="ap-question"><strong>問1</strong>　2けたの2進数x<sub>1</sub>x<sub>2</sub>が表す整数をxとする。2進数x<sub>2</sub>x<sub>1</sub>が表す整数を，xの式で表したものはどれか。ここで，int(r)は非負の実数rの小数点以下を切り捨てた整数を表す。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　2x＋4int(x/2)</div>
<div class="ap-choice-note">x<sub>2</sub>x<sub>1</sub>＝2x－3int(x/2) となるので、＋4int(x/2) では一致しない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　2x＋5int(x/2)</div>
<div class="ap-choice-note">係数が＋5となっており、x<sub>2</sub>x<sub>1</sub> の値と一致しない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　2x－3int(x/2)</div>
<div class="ap-choice-note">正しい。2x－3int(x/2)＝x<sub>2</sub>×2＋x<sub>1</sub> となる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　2x－4int(x/2)</div>
<div class="ap-choice-note">係数が－4となっており、x<sub>2</sub>x<sub>1</sub> の値と一致しない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>x(エックス)と×(かける)が紛らわしいので、解説中では乗算の演算子を * としています。</p>
<p>整数xは10進数で x<sub>1</sub>*2 + x<sub>2</sub> なので、選択肢中の2xは10進数で以下のように示すことができます。<br>2x = x<sub>1</sub>*4 + x<sub>2</sub>*2　…①</p>
<p>次に、選択肢中の int(x/2) について考えます。x/2は、xを右に1<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>シフト（1/2）させたものなので、<br>x<sub>1</sub>x<sub>2</sub>→(右へ1<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>シフト)→x<sub>1</sub>.x<sub>2</sub> （"."は小数点）<br>さらに、int()は整数部を取り出す操作なので、<br>int(x<sub>1</sub>.x<sub>2</sub>) = x<sub>1</sub><br>つまり、int(x/2) は x<sub>1</sub> と同じ値ということになります。<br>int(x/2) = x<sub>1</sub>　…②</p>
<p>①と同様に、2進数x<sub>2</sub>x<sub>1</sub>を10進数で表すと x<sub>2</sub>*2 + x<sub>1</sub> です。これを先程の①と比べると、両者の差分は x<sub>1</sub> が3つ分となっています。<br>x<sub>1</sub>*4 + x<sub>2</sub>*2 - x<sub>1</sub>*3 = x<sub>2</sub>*2 + x<sub>1</sub>　…③</p>
<p>①②より 2x = x<sub>1</sub>*4＋x<sub>2</sub>*2、x<sub>1</sub> = int(x/2) なので、③の左辺の該当部分を置き換えると、以下のように表すことができます。<br>2x－3int(x/2)＝x<sub>2</sub>*2＋x<sub>1</sub><br>したがって「ウ」の式が適切です。</p>
<p>【別解】<br>ここまでがこの設問の正しい理解ですが、実際の試験本番では x<sub>1</sub>＝1，x<sub>2</sub>＝1 として、<br>x<sub>1</sub>x<sub>2</sub>＝11(2)＝3(10)<br>x<sub>2</sub>x<sub>1</sub>＝11(2)＝3(10)<br>int(x/2)＝1<br>3＝2*3－a<br>a＝3<br>で「ウ」が正解としたり、x<sub>1</sub>＝1，x<sub>2</sub>＝0 として、<br>x<sub>1</sub>x<sub>2</sub>＝10(2)＝2(10)<br>x<sub>2</sub>x<sub>1</sub>＝01(2)＝1(10)<br>int(x/2)＝1<br>1＝2*2－a<br>a＝3<br>とするなど、簡単に計算できる値を代入して消去法で解く方法が無難でしょう。</p>
