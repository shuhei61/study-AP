<p class="ap-question"><strong>問67</strong>　H.I.アンゾフが提唱した成長マトリクスを説明したものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　既存製品か新製品かという製品軸と既存市場か新市場かという市場軸の両軸で捉え、事業成長戦略を考える。</div>
<div class="ap-choice-note">正しい。成長マトリクス（<a href="用語/製品・市場マトリクス" class="internal-link" data-href="用語/製品・市場マトリクス">製品・市場マトリクス</a>）は、製品軸と市場軸の両軸で事業成長戦略を検討するフレームワークです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　コストで優位に立つかコスト以外で差別化するか，ターゲットを広くするか集中するかによって戦略を考える。</div>
<div class="ap-choice-note">ポーターの<a href="用語/競争戦略" class="internal-link" data-href="用語/競争戦略">競争戦略</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　市場成長率が高いか低いか，相対的市場シェアが大きいか小さいかによって事業を捉え，資源配分の戦略を考える。</div>
<div class="ap-choice-note">プロダクトポートフォリオマネジメント分析の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　自社の内部環境の強みと弱み，取り巻く外部環境の機会と脅威を抽出し，取組方針を整理して戦略を考える。</div>
<div class="ap-choice-note">SWOT分析の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>アンゾフの成長マトリクスは、経営学者のH・イゴール・アンゾフ（H. Igor Ansoff）が提唱したもので、縦軸に「市場」、横軸に「製品」をとり、それぞれに「既存」「新規」の2区分を設け、4象限（市場浸透、製品開発、市場開拓、<a href="用語/多角化" class="internal-link" data-href="用語/多角化">多角化</a>）のマトリクスとしたものです。事業が成長・発展できる経営戦略を検討するために適したフレームワークです。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R5秋期/67/67.png" alt="解説図" width="252" height="261" ></p>
</div>
