<p class="ap-question"><strong>問69</strong>　市場を消費者特性でセグメント化する際に，基準となる変数を，地理的変数，人口統計的変数，心理的変数，行動的変数に分類するとき，人口統計的変数に分類されるものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　社交性などの性格</div>
<div class="ap-choice-note">「性格」は心理的変数に該当します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　職業</div>
<div class="ap-choice-note">正しい。「職業」は人口統計的変数に該当します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　人口密度</div>
<div class="ap-choice-note">「人口密度」は地理的変数に該当します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　製品の使用割合</div>
<div class="ap-choice-note">「製品の使用割合」は行動的変数に該当します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>地理的変数、人口統計的変数、心理的変数、行動的変数は、市場を<a href="用語/セグメント化" class="internal-link" data-href="用語/セグメント化">セグメント化</a>するときに使う変数です。4つをまとめてセグメンテーション変数と呼びます。</p>
<p><strong>地理的変数（ジオグラフィック変数）</strong><br>地理的な位置や地域に関連する情報を表す変数<br>例）地域、人口密度、地形、気候など</p>
<p><strong>人口統計的変数（デモグラフィック変数）</strong><br>人々の属性や特性に関連する情報を表す変数<br>例）年齢、性別、収入、職業、世帯構成、学歴など</p>
<p><strong>心理的変数（サイコグラフィック変数）</strong><br>個人の心理的状態や特性に関連する情報を表す変数<br>例）態度、性格、趣味、価値観、幸福感、ストレスレベルなど</p>
<p><strong>行動的変数</strong><br>個人または集団の実際の行動に関連する情報を表す変数<br>例）購買行動、習慣、製品知識、製品の使用状況など</p>
<p>これを踏まえて、選択肢を分類すると以下のようになります。</p>
<ul>
<li>ア　社交性などの性格 → 心理的変数</li>
<li>イ　職業 → 人口統計的変数</li>
<li>ウ　人口密度 → 地理的変数</li>
<li>エ　製品の使用割合 → 行動的変数</li>
</ul>
</div>
