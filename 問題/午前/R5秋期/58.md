<p class="ap-question"><strong>問58</strong>　システム監査人が作成する監査調書に関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　監査調書の作成は任意であり，作成しなくても問題はない。</div>
<div class="ap-choice-note"><a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>人は、すべての<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>において、実施内容の客観性等を確保するために、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書を作成しなければなりません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　監査調書は，監査人自身の行動記録であり，監査チーム内でも他の監査人と共有すべきではない。</div>
<div class="ap-choice-note"><a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書は、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>の実施記録を公式な文書として残すものであり、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>チーム内で共有の記録となり、各<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>人が必要に応じて参照することになります。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　監査調書は，監査の結論を支える合理的な根拠とするために，発見した事実及び発見事実に関する所見を記載する。</div>
<div class="ap-choice-note">正しい。<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書には、実施した<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>手続、入手した<a href="用語/監査証拠" class="internal-link" data-href="用語/監査証拠">監査証拠</a>、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>人が発見した事実とそれに対する<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>人の所見などが記載されます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　監査調書は，保管の必要がない監査人の備忘録である。</div>
<div class="ap-choice-note"><a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書は、組織の<a href="用語/文書管理規程" class="internal-link" data-href="用語/文書管理規程">文書管理規程</a>に従って体系的に整理し、後日、容易に参照、活用できるように保管する必要があります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書は、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>手続の実施内容や発見した問題点、収集した情報およびそれらの分析結果、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>人の結論などをまとめた文書です。<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書は、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>報告書に記載される<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>結果、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>意見、改善勧告などの根拠となります。</p>
<p><a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>調書は主に次の目的のために作成されます。<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>計画、<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>手続、レビューを文書化する<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>報告書に記載する<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>結果などの裏付けとなること次回の<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>や別の<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>人による<a href="用語/監査" class="internal-link" data-href="用語/監査">監査</a>のための基礎資料となること</p>
</div>
