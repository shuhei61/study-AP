#問題 #マネジメント #プロジェクトマネジメント #プロジェクトのリスク

<p class="ap-question"><strong>問54</strong>　プロジェクトマネジメントにおいて，コンティンジェンシー計画を作成するプロセスはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　リスクの管理</div>
<div class="ap-choice-note"><a href="用語/リスクの管理" class="internal-link" data-href="用語/リスクの管理">リスクの管理</a>はリスクマネジメント全体を統括するプロセスであり、コンティンジェンシー計画の作成そのものを指すプロセスではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　リスクの特定</div>
<div class="ap-choice-note"><a href="用語/リスクの特定" class="internal-link" data-href="用語/リスクの特定">リスクの特定</a>は、<a href="用語/プロジェクト" class="internal-link" data-href="用語/プロジェクト">プロジェクト</a>が抱える<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>を洗い出すプロセス。対応計画の策定は行わない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　リスクの評価</div>
<div class="ap-choice-note"><a href="用語/リスクの評価" class="internal-link" data-href="用語/リスクの評価">リスクの評価</a>は、特定した<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>の影響や発生確率などを分析するプロセス。対応計画の策定は行わない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　リスクへの対応</div>
<div class="ap-choice-note">正しい。<a href="用語/リスクへの対応" class="internal-link" data-href="用語/リスクへの対応">リスクへの対応</a>では、コンティンジェンシー計画を含む対応計画を策定する。コンティンジェンシー計画は、既知の<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>に対する対応計画の一部として策定される。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/プロジェクトマネジメント" class="internal-link" data-href="用語/プロジェクトマネジメント">プロジェクトマネジメント</a>におけるコンティンジェンシー計画とは、<a href="用語/プロジェクト" class="internal-link" data-href="用語/プロジェクト">プロジェクト</a>が抱える潜在的な<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>が現実化した際に、<a href="用語/プロジェクト" class="internal-link" data-href="用語/プロジェクト">プロジェクト</a>が被る損害を最小限にとどめ、速やかにその緊急事態の克服を図るための手続きが記述された対応計画のことです。コンティンジェンシー計画は、既知の<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>に対する対応計画の一部として策定され、ある事象が起きた場合に限り使用するものなので、発動条件となる事象とそのとき実行する対応策を記述します。</p>
<p>したがって「エ」が正解です。</p>
