#問題 #テクノロジ #コンピュータ構成要素 #プロセッサ

<p class="ap-question"><strong>問8</strong>　スマートフォンなどで高い処理性能と低消費電力の両立のために、異なる目的に適した複数の種類のコアを搭載したプロセッサはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　スーパースカラプロセッサ</div>
<div class="ap-choice-note">これは<a href="用語/スーパースカラ" class="internal-link" data-href="用語/スーパースカラ">スーパースカラ</a>の説明です。パイプライン機構を複数用意し、同じステージを並列実行できるよう設計されたプロセッサであり、異種コアの搭載とは別の概念です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　ソフトコアプロセッサ</div>
<div class="ap-choice-note"><a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>の構成要素が必要に応じて機能するように、<a href="用語/FPGA" class="internal-link" data-href="用語/FPGA">FPGA</a>のロジック構造を用いてプログラミングできるプロセッサです。異種コアの搭載を指す用語ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　ヘテロジニアスマルチコアプロセッサ</div>
<div class="ap-choice-note">正しい。<a href="用語/ヘテロジニアスマルチコア" class="internal-link" data-href="用語/ヘテロジニアスマルチコア">ヘテロジニアスマルチコア</a>プロセッサは、1つのチップにアーキテクチャ、規模、特徴などが異なる種類の<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>コアを搭載するプロセッサです。ヘテロジニアス（heterogeneous）は「異種の」という意味です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ホモジニアスマルチコアプロセッサ</div>
<div class="ap-choice-note">1つのチップに同じ種類の<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>コアを複数個搭載するプロセッサです。ホモジニアス（homogeneous）は「同種の」という意味で、異なる種類のコアを搭載する本問の条件には合いません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>スマートフォンなどでは、重い処理用の高性能コアと常時稼働する省電力コアを組み合わせることで、処理性能と消費電力の両立を図ります。これは1つのチップにアーキテクチャや規模が異なる複数種類の<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>コアを搭載する<a href="用語/ヘテロジニアスマルチコア" class="internal-link" data-href="用語/ヘテロジニアスマルチコア">ヘテロジニアスマルチコア</a>プロセッサの特徴です。リアルタイム処理とそれ以外の処理、<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>と<a href="用語/GPU" class="internal-link" data-href="用語/GPU">GPU</a>、データ処理とコントロール処理、汎用<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>と特化<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>、通常のコアと省電力コアなどの組合せがあります。</p>
<p>アの<a href="用語/スーパースカラ" class="internal-link" data-href="用語/スーパースカラ">スーパースカラ</a>プロセッサは1コア内で命令を並列実行する方式、イのソフトコアプロセッサは<a href="用語/FPGA" class="internal-link" data-href="用語/FPGA">FPGA</a>上にプログラミング可能な<a href="用語/CPU" class="internal-link" data-href="用語/CPU">CPU</a>を構成する方式、エのホモジニアスマルチコアプロセッサは同種コアの複数搭載であり、いずれも「異なる目的に適した複数の種類のコアを搭載する」条件には合いません。したがって正解は「ウ」です。</p>
</div>
