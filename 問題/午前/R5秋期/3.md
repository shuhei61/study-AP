<p class="ap-question"><strong>問3</strong>　逆ポーランド表記法(後置記法)で表現されている式ABCD－×＋において，A＝16，B＝8，C＝4，D＝2のときの演算結果はどれか。逆ポーランド表記法による式AB＋は，中置記法による式A＋Bと同一である。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　32</div>
<div class="ap-choice-note">正しい。<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>で式を左から評価すると32になる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　46</div>
<div class="ap-choice-note">演算順序の誤りによる値。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　48</div>
<div class="ap-choice-note">(A＋B)×(C－D)のように括り方を誤った場合の値。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　94</div>
<div class="ap-choice-note">演算順序の誤りによる値。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/逆ポーランド表記法" class="internal-link" data-href="用語/逆ポーランド表記法">逆ポーランド表記法</a>は、通常の中置記法とは異なり、演算子（＋、－、×、÷）を2つの項の後ろに記述する数式の表現方法です。例えば、中置記法の「1＋2」は、<a href="用語/逆ポーランド表記法" class="internal-link" data-href="用語/逆ポーランド表記法">逆ポーランド表記法</a>で「12＋」と記述されます。</p>
<p><a href="用語/逆ポーランド表記法" class="internal-link" data-href="用語/逆ポーランド表記法">逆ポーランド表記法</a>による数式は、コンピュータがそのように処理するように、<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>（後入れ先出しの<a href="用語/データ構造" class="internal-link" data-href="用語/データ構造">データ構造</a>）を使って以下の手順で答えを求めることができます。数式の左側から値を1つずつ<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>に積んでいく。演算子が現れたら、<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>から上2つの値を取り出して演算し、その結果を<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>に積む。式を最後まで評価し、<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>に残っている値が答えとなる。</p>
<p>この問題の式も、次のように<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>を使って答えを求めることができます。</p>
<ul>
<li>Aを積む　[16]</li>
<li>Bを積む　[16, 8]</li>
<li>Cを積む　[16, 8, 4]</li>
<li>Dを積む　[16, 8, 4, 2]</li>
<li>－なので、4、2を取り出し、4－2の結果である2を積む　[16, 8, 2]</li>
<li>×なので、8、2を取り出し、8×2の結果である16を積む　[16, 16]</li>
<li>＋なので、16、16を取り出し、16＋16の結果である32を積む　[32]</li>
</ul>
<p>式の評価が終わった時点で<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>に残っている32が演算結果となります。したがって「ア」が正解です。</p>
<p>【別解】式中で最も左にある演算子を起点にして、中置記法の式に戻し、値を代入して演算します。</p>
<ul>
<li>"CD－"を"C－D"にする</li>
<li>"B(C－D)×"を"B×(C－D)"にする</li>
<li>"AB×(C－D)＋"を"A＋B×(C－D)"にする</li>
</ul>
<p>値を当てはめると、16＋8×(4－2)＝16＋8×2＝16＋16＝32</p>
