<p class="ap-question"><strong>問36</strong>　パスワードクラック手法の一種である，レインボーテーブル攻撃に該当するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　何らかの方法で事前に利用者IDと平文のパスワードのリストを入手しておき，複数のシステム間で使い回されている利用者IDとパスワードの組みを狙って，ログインを試行する。</div>
<div class="ap-choice-note">これは<a href="用語/パスワードリスト攻撃" class="internal-link" data-href="用語/パスワードリスト攻撃">パスワードリスト攻撃</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　パスワードに成り得る文字列の全てを用いて，総当たりでログインを試行する。</div>
<div class="ap-choice-note">これは<a href="用語/総当たり攻撃" class="internal-link" data-href="用語/総当たり攻撃">総当たり攻撃</a>(ブルートフォースアタック)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　平文のパスワードとハッシュ値をチェーンによって管理するテーブルを準備しておき，それを用いて，不正に入手したハッシュ値からパスワードを解読する。</div>
<div class="ap-choice-note">正しい。<a href="用語/レインボーテーブル攻撃" class="internal-link" data-href="用語/レインボーテーブル攻撃">レインボーテーブル攻撃</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　利用者の誕生日，電話番号などの個人情報を言葉巧みに聞き出して，パスワードを類推する。</div>
<div class="ap-choice-note">類推攻撃の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/レインボーテーブル攻撃" class="internal-link" data-href="用語/レインボーテーブル攻撃">レインボーテーブル攻撃</a>は、ハッシュ値からパスワードを特定するための逆引き表（レインボーテーブル）を用いて、ハッシュ値の元となったパスワードを効率的に解読する手法です。</p>
<p>レインボーテーブルは、使用される文字種と文字数の組合せごとに作成されます。レインボーテーブル内では、パスワードとハッシュ値を数多くのチェーンとして管理しており、実際のテーブルにはチェーンの先頭であるパスワードと最後のハッシュ値だけを格納しておきます。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R5秋期/36/36_1.png" alt="解説図 1" width="493" height="207" ></p>
<p>解読対象のハッシュ値を入手したら、チェーンの各位置からチェーン化で行ったのと同様の計算を施し、チェーンの最後のハッシュ値を計算します。これがレインボーテーブルに格納されているハッシュ値のいずれかと一致すれば、対応するパスワードが存在するチェーンがわかる仕組みになっています。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R5秋期/36/36_2.png" alt="解説図 2" width="233" height="176" ></p>
