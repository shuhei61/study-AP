<p class="ap-question"><strong>問19</strong>　Linuxカーネルの説明として、適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　CUIによるコマンド入力のためのシェルと呼ばれるソフトウェアが組み込まれていて，文字での操作が可能である。</div>
<div class="ap-choice-note"><a href="用語/シェル" class="internal-link" data-href="用語/シェル">シェル</a>の機能はBashなどのソフトウェアで実現されており、カーネルに組み込まれているわけではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　GUIを利用できるデスクトップ環境が組み込まれていて，マウスを使った直感的な操作が可能である。</div>
<div class="ap-choice-note"><a href="用語/GUI" class="internal-link" data-href="用語/GUI">GUI</a>はGNOMEやKDEなどのソフトウェアの機能で実現されており、カーネルの機能ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　Webブラウザ，ワープロソフト，表計算ソフトなどが含まれており，Linuxカーネルだけで多くの業務が行える。</div>
<div class="ap-choice-note">カーネルだけでは応用ソフトウェアを動作させることはできません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　プロセス管理やメモリ管理などの，アプリケーションソフトウェアが動作するための基本機能を提供する。</div>
<div class="ap-choice-note">正しい。カーネルは、アプリケーションとデバイスを管理するための基本的な機能だけを提供するソフトウェアです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/Linuxカーネル" class="internal-link" data-href="用語/Linuxカーネル">Linuxカーネル</a>は、OSとしてコンピュータやアプリケーションソフトを制御するために必要となる、<a href="用語/メモリ" class="internal-link" data-href="用語/メモリ">メモリ</a>管理・ファイルシステム・<a href="用語/プロセス管理" class="internal-link" data-href="用語/プロセス管理">プロセス管理</a>・デバイス制御・ネットワーキングなどの最も基本的機能だけを実装したソフトウェアです。カーネルはOSの基盤となるソフトウェアですが、カーネルだけでアプリケーションを動作させることはできず、OSとして機能させるためにはglibcなどの<a href="用語/ライブラリ" class="internal-link" data-href="用語/ライブラリ">ライブラリ</a>やGNUソフトウェアを組み合わせる必要があります。</p>
<p>glibcは標準入出力、文字列操作、数値演算などの機能がC言語で記述された<a href="用語/ライブラリ" class="internal-link" data-href="用語/ライブラリ">ライブラリ</a>です。GNUソフトウェアは<a href="用語/シェル" class="internal-link" data-href="用語/シェル">シェル</a>のBash、CやC++<a href="用語/コンパイラ" class="internal-link" data-href="用語/コンパイラ">コンパイラ</a>であるGCC、多数の外部コマンドなどが含まれる<a href="用語/GPL" class="internal-link" data-href="用語/GPL">GPL</a>でライセンスされる<a href="用語/フリーソフトウェア" class="internal-link" data-href="用語/フリーソフトウェア">フリーソフトウェア</a>群です。</p>
<p>したがって適切な記述は「エ」です。</p>
