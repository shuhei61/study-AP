<p class="ap-question"><strong>問68</strong>　顧客から得る同意の範囲を段階的に広げながら，プロモーションを行うことが特徴的なマーケティング手法はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　アフィリエイトマーケティング</div>
<div class="ap-choice-note"><a href="用語/アフィリエイト" class="internal-link" data-href="用語/アフィリエイト">アフィリエイト</a>マーケティングは、成約した場合に報酬を支払うことを条件に、ブロガーなどのパートナーのWebサイト上で商品やサービスを紹介してもらう手法です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　差別型マーケティング</div>
<div class="ap-choice-note">差別型マーケティングは、ターゲットとする市場セグメントごとに異なる<a href="用語/マーケティングミックス" class="internal-link" data-href="用語/マーケティングミックス">マーケティングミックス</a>を展開する手法です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　パーミッションマーケティング</div>
<div class="ap-choice-note">正しい。<a href="用語/パーミッションマーケティング" class="internal-link" data-href="用語/パーミッションマーケティング">パーミッションマーケティング</a>は、同意を得た顧客に対して直接的にパーソナルな売り込みを行う手法です。同意を得た上でプロモーションを行うので、押しつけがましくなく、また顧客からの反応も通常に比べて良いなどの強みがあります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　バイラルマーケティング</div>
<div class="ap-choice-note"><a href="用語/バイラルマーケティング" class="internal-link" data-href="用語/バイラルマーケティング">バイラルマーケティング</a>は、人から人への口コミやインターネット上での情報共有をしたくなる仕組みを構築して、商品やサービスの情報を自然発生的に広めてもらうことで認知を高める手法です。情報が人から人へ次々と伝わっていく様子から、「ウイルス性の」という意味のバイラル（viral）が付けられています。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/パーミッションマーケティング" class="internal-link" data-href="用語/パーミッションマーケティング">パーミッションマーケティング</a>は、マスメディアを使って不特定多数の消費者にプロモーションを行うのではなく、何らかの同意（パーミッション）を得た潜在顧客を売り込みの対象とし、時間をかけて潜在顧客との接触頻度を高め、同意の範囲を広げることで最終的には利益を生み出す顧客に変えるというマーケティング手法です。</p>
<p><a href="用語/パーミッションマーケティング" class="internal-link" data-href="用語/パーミッションマーケティング">パーミッションマーケティング</a>の基本的な流れは次のとおりです。</p>
<ol>
<li>潜在顧客が自ら行動を起こすようなインセンティブを用意する</li>
<li>インセンティブに同意した潜在顧客に対して、時間をかけて商品やサービスの良さを説明する</li>
<li>インセンティブを強化し、より大きな同意を得る</li>
<li>同意を活用して、消費者の行動を変化させて顧客にする</li>
</ol>
<p>今となっては特に目新しい手法というわけではなく、サンプル品や資料提供を送付する同意をもらってそれを糸口に製品のプロモーションを行う、無料コンテンツダウンロードやメールマガジンのためにメールを登録してもらってメールで売り込みを行う、LINEの友達登録をしてもらい宣伝をするなどが<a href="用語/パーミッションマーケティング" class="internal-link" data-href="用語/パーミッションマーケティング">パーミッションマーケティング</a>の一例です。</p>
</div>
