<p class="ap-question"><strong>問75</strong>　リーダーシップ論のうち，F.E.フィードラーが提唱するコンティンジェンシー理論の特徴はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　優れたリーダーシップを発揮する，リーダー個人がもつ性格，知性，外観などの個人的資質の分析に焦点を当てている。</div>
<div class="ap-choice-note">リーダーの個人的な資質に注目する考え方は特性論的アプローチです。<a href="用語/コンティンジェンシー理論" class="internal-link" data-href="用語/コンティンジェンシー理論">コンティンジェンシー理論</a>ではリーダー個人の生来的な特質や資質は考慮しません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　リーダーシップのスタイルについて，目標達成能力と集団維持能力の二つの次元に焦点を当てている。</div>
<div class="ap-choice-note"><a href="用語/PM理論" class="internal-link" data-href="用語/PM理論">PM理論</a>の説明です。<a href="用語/コンティンジェンシー理論" class="internal-link" data-href="用語/コンティンジェンシー理論">コンティンジェンシー理論</a>では、ポジションパワー、メンバーとの関係性、課題構造の3つの変数に焦点を当てています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　リーダーシップの有効性は，部下の成熟(自律性)の度合いという状況要因に依存するとしている。</div>
<div class="ap-choice-note">ハーシィ＆ブランチャードの<a href="用語/SL理論" class="internal-link" data-href="用語/SL理論">SL理論</a>（状況対応<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>理論）の特徴です。<a href="用語/コンティンジェンシー理論" class="internal-link" data-href="用語/コンティンジェンシー理論">コンティンジェンシー理論</a>では部下の成熟度は考慮しません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　リーダーシップの有効性は，リーダーがもつパーソナリティと，リーダーがどれだけ統制力や影響力を行使できるかという状況要因に依存するとしている。</div>
<div class="ap-choice-note">正しい。<a href="用語/コンティンジェンシー理論" class="internal-link" data-href="用語/コンティンジェンシー理論">コンティンジェンシー理論</a>の特徴です。リーダーのもつパワーおよびリーダーとメンバー間の対人関係や職位等によって決まる権力行使の幅により<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>の有効性が左右されるという考え方です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>フィードラーの<a href="用語/コンティンジェンシー理論" class="internal-link" data-href="用語/コンティンジェンシー理論">コンティンジェンシー理論</a>（状況呼応型<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>）は、①職位等から生まれるリーダーのパワー、②リーダーとメンバーとの対人関係、③仕事が明確で構造化されているかどうか、の3つの変数から生じる状況に応じて、効果的な<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>のスタイルは変化するとする<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>論です。</p>
<p><a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>のスタイルを大きく分けると、仕事を細かく規定し、統制し、メンバーを厳しく監督することで目標の達成を追及する「課題指向的」と、コミュニケーションや協調を通じてメンバーを支援し、メンバー自身の自己統制能力やエンゲージメントを高めることで目標の達成を追及する「関係指向的」に分類することができます。<a href="用語/コンティンジェンシー理論" class="internal-link" data-href="用語/コンティンジェンシー理論">コンティンジェンシー理論</a>では、リーダーが影響力を行使するのに最良の状況（パワー：強、関係性：良、課題構造：高い）のときと最悪の状況（パワー：弱、関係性：悪、課題構造：低）のときに「課題指向的」な<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>スタイルが有効に機能し、有利さが中程度のときには「関係指向的」な<a href="用語/リーダーシップ" class="internal-link" data-href="用語/リーダーシップ">リーダーシップ</a>スタイルが有効であると論じています。</p>
</div>
