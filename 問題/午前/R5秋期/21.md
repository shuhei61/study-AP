<p class="ap-question"><strong>問21</strong>　MOSトランジスタの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　pn接合における電子と正孔の再結合によって光を放出するという性質を利用した半導体素子</div>
<div class="ap-choice-note">発光<a href="用語/ダイオード" class="internal-link" data-href="用語/ダイオード">ダイオード</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　pn接合部に光が当たると電流が発生するという性質を利用した半導体素子</div>
<div class="ap-choice-note">フォト<a href="用語/ダイオード" class="internal-link" data-href="用語/ダイオード">ダイオード</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　金属と半導体との間に酸化物絶縁体を挟んだ構造をもつことが特徴の半導体素子</div>
<div class="ap-choice-note">正しい。MOS<a href="用語/トランジスタ" class="internal-link" data-href="用語/トランジスタ">トランジスタ</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　逆方向電圧をある電圧以上印加すると，電流だけが増加し電圧がほぼ一定に保たれるという特性をもつ半導体素子</div>
<div class="ap-choice-note">ツェナー<a href="用語/ダイオード" class="internal-link" data-href="用語/ダイオード">ダイオード</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>MOS<a href="用語/トランジスタ" class="internal-link" data-href="用語/トランジスタ">トランジスタ</a>は、電流や電圧のON/OFFを切り替えたり増幅したりする電子部品である<a href="用語/トランジスタ" class="internal-link" data-href="用語/トランジスタ">トランジスタ</a>の一種で、金属(Metal)、酸化膜(Oxide)、半導体(Semiconductor)の3層構造を持ちます。ゲート、ドレイン、ソースの3つの端子があり、ゲートに印加される電圧によってドレインからソース間の電流を制御する仕組みになっています。</p>
