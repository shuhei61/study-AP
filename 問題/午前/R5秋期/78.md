<p class="ap-question"><strong>問78</strong>　プログラムの著作物について，著作権法上，適法である行為はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　海賊版を複製したプログラムと事前に知りながら入手し，業務で使用した。</div>
<div class="ap-choice-note">違法。<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>侵害行為によって作成された複製物を、その事情を知りながら業務で使用した場合は<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>侵害行為とみなされる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　業務処理用に購入したプログラムを複製し，社内教育用として各部門に配布した。</div>
<div class="ap-choice-note">違法。複製権・頒布権は<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>者にのみ認められる権利であり、承諾なく複製・配布すると侵害となる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　職務著作のプログラムを，作成した担当者が独断で複製し，他社に貸与した。</div>
<div class="ap-choice-note">違法。業務上作成した<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>の<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>は原則として法人に帰属し、担当者が独断で貸与すると法人の<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>（貸与権）を侵害する。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　処理速度を向上させるために，購入したプログラムを改変した。</div>
<div class="ap-choice-note">正しい。<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>を効果的に実行できるようにするために必要な改変は、<a href="用語/著作権法" class="internal-link" data-href="用語/著作権法">著作権法</a>で明示的に認められている。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>アは違法な行為です。<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>を侵害する行為によって作成された<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>の著作物の複製物を、その事情を知りながら業務で使用した場合、<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>侵害行為とみなされます。</p>
<p>イは違法な行為です。著作物を複製することは複製権、著作物を配布することは頒布権として、それぞれ<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>者だけに認められる権利です。承諾を得ずにこれらの行為を行うと<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>侵害行為となります。</p>
<p>ウは違法な行為です。業務上作成した<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>の<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>は、原則として法人に帰属します。作成した担当者は<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>者ではないので、勝手に貸与すると法人の<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>（貸与権）を侵害することになります。</p>
<p>エは正しい。<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>を効果的に実行できるようにするために必要な改変は、<a href="用語/著作権法" class="internal-link" data-href="用語/著作権法">著作権法</a>で明示的に認められているので適法な行為です。<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>の著作物を改変して使用することは、原則として、著作者の同一性保持権を侵害する行為として<a href="用語/著作権法" class="internal-link" data-href="用語/著作権法">著作権法</a>で禁止されます。しかし、例外的に次の4つのケースでは、<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>者の承諾なしに改変することが認められています。</p>
<ul>
<li>学校教育の目的上やむを得ない改変</li>
<li>建築物の増築、改築、修繕、模様替えによる改変</li>
<li><a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>が対応していないコンピュータで実行できるようにするため、または<a href="用語/プログラム" class="internal-link" data-href="用語/プログラム">プログラム</a>を効果的に実行できるようにするために必要な改変</li>
<li>著作物の性質、その利用の目的や態様に照らしてやむを得ない改変</li>
</ul>
<p>したがって「エ」が正解です。</p>
</div>
