<p class="ap-question"><strong>問2</strong>　複数の変数をもつデータに対する分析手法の記述のうち，主成分分析はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　変数に共通して影響を与える新たな変数を計算して，データの背後にある構造を取得する方法</div>
<div class="ap-choice-note">これは<a href="用語/因子分析" class="internal-link" data-href="用語/因子分析">因子分析</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　変数の値からほかの変数の値を予測して，データがもつ変数間の関連性を確認する方法</div>
<div class="ap-choice-note">これは<a href="用語/回帰分析" class="internal-link" data-href="用語/回帰分析">回帰分析</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　変数の値が互いに類似するものを集めることによって，データを分類する方法</div>
<div class="ap-choice-note">これは<a href="用語/クラスタ" class="internal-link" data-href="用語/クラスタ">クラスタ</a>分析の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　変数を統合した新たな変数を使用して，データがもつ変数の数を減らす方法</div>
<div class="ap-choice-note">正しい。<a href="用語/主成分分析" class="internal-link" data-href="用語/主成分分析">主成分分析</a>は、多変量データの次元削減やデータの<a href="用語/特徴抽出" class="internal-link" data-href="用語/特徴抽出">特徴抽出</a>に用いられる手法です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/主成分分析" class="internal-link" data-href="用語/主成分分析">主成分分析</a>は、複数の要因が相互に関連している場合に、その群の特性を決定づけている主な要因を合成変数として求める統計的な手法です。合成された変数を主成分と呼びます。情報の損失を最小限に抑えながら変数を減らすことで、分析対象のデータを理解しやすくする効果があります。</p>
<p>国数理社英の5教科の得点から総合評価点を求める、身長と体重をBMIという1つの指標にする、アンケート調査の分析で寄与度が高い要素を抽出するなどが<a href="用語/主成分分析" class="internal-link" data-href="用語/主成分分析">主成分分析</a>の例です。</p>
<p>したがって「エ」が正解です。</p>
</div>
