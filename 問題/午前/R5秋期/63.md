<p class="ap-question"><strong>問63</strong>　SOAを説明したものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　企業改革において既存の組織やビジネスルールを抜本的に見直し，業務フロー，管理機構及び情報システムを再構築する手法のこと</div>
<div class="ap-choice-note"><a href="用語/BPR" class="internal-link" data-href="用語/BPR">BPR</a>(Business Process Reengineering)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　企業の経営資源を有効に活用して経営の効率を向上させるために，基幹業務を部門ごとでなく統合的に管理するための業務システムのこと</div>
<div class="ap-choice-note"><a href="用語/ERP" class="internal-link" data-href="用語/ERP">ERP</a>(Enterprise Resource Planning)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　発注者とITアウトソーシングサービス提供者との間で，サービスの品質について合意した文書のこと</div>
<div class="ap-choice-note"><a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>(Service Level Agreement)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　ビジネスプロセスの構成要素とそれを支援するIT基盤を，ソフトウェア部品であるサービスとして提供するシステムアーキテクチャのこと</div>
<div class="ap-choice-note">正しい。<a href="用語/SOA" class="internal-link" data-href="用語/SOA">SOA</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/SOA" class="internal-link" data-href="用語/SOA">SOA</a>(Service Oriented Architecture：サービス指向アーキテクチャ)は、従来ソフトウェアで実現されていた機能を独立したサービスとして設計し、それらを組み合わせて相互作用させることによりシステムを構築するという設計思想です。大規模なシステムやアプリケーションを効率的に構築、統合、管理するためのアーキテクチャとして採用されています。</p>
<p>各構成要素は、特定の機能やプロセスを提供する部品として再利用可能な形で設計され、それぞれが独立しているため、開発・変更・保守がしやすいという利点があります。各構成要素は、他のサービスと通信するインタフェースを備えていて、HTTP、<a href="用語/SOAP" class="internal-link" data-href="用語/SOAP">SOAP</a>、<a href="用語/REST" class="internal-link" data-href="用語/REST">REST</a>などの仕組みによって相互通信を行います。</p>
<p>したがって「エ」が正解です。</p>
</div>
