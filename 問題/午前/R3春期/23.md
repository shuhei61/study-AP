<p class="ap-question"><strong>問23</strong>　図はDCモーターの正転逆転制御の動作原理を示す回路である。A1からA4の四つの制御信号の組合せの中で，モーターが逆転するものはどれか。ここで，モーターの＋端子から－端子に電流が流れるときモーターは正転し，S1からS4のそれぞれのスイッチ素子は，対応するA1からA4の制御信号がそれぞれHighのとき導通するものとする。<br><img class="ap-fig" src="../../../assets/問題/午前/R3春期/23/23.png" alt="問題図" width="302" height="230" ><br><img class="ap-fig" src="../../../assets/問題/午前/R3春期/23/23a.png" alt="選択肢の組合せ表" width="316" height="148" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア</div>
<div class="ap-choice-note">モーターに電流が流れない組合せです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ</div>
<div class="ap-choice-note">モーターに電流が流れない組合せです。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ</div>
<div class="ap-choice-note">正しい。モーターの－端子から＋端子へ電流が流れ，逆転します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ</div>
<div class="ap-choice-note">モーターの＋端子から－端子へ電流が流れるため，正転します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>図中の電源側は＋極、回路の基準電位はグランド（GND）の－極です。電気は＋極から－極へ流れます。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/23/23_1.png" alt="解説図" width="37" height="14" ></p>
<p>問題文のとおり，モーターの＋端子から－端子へ電流が流れるとき正転します。逆転させるには，モーターの－端子から＋端子へ電流を流す必要があります。本問のDCモーターは，<a href="用語/ブラシレスDCモーター" class="internal-link" data-href="用語/ブラシレスDCモーター">ブラシレスDCモーター</a>をはじめとする直流モーターで同様に，通電方向で回転方向が決まります。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/23/23_2.png" alt="解説図" width="21" height="18" ></p>
<p>各肢（ア〜エ）の電流の流れを確認すると，「ア」「イ」はモーターに電流が流れず，「エ」は＋端子から－端子へ流れるため正転です。「ウ」だけが－端子から＋端子へ流れる組合せなので，正解は「ウ」です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/23/23_3.png" alt="解説図" width="547" height="426" ></p>
</div>
