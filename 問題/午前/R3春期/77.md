<p class="ap-question"><strong>問77</strong>　キャッシュフロー計算書において，営業活動によるキャッシュフローに該当するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　株式の発行による収入</div>
<div class="ap-choice-note">株式の発行は<a href="用語/財務活動によるキャッシュフロー" class="internal-link" data-href="用語/財務活動によるキャッシュフロー">財務活動によるキャッシュフロー</a>に該当します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　商品の仕入れによる支出</div>
<div class="ap-choice-note">正しい。商品の仕入は本業に係る支出であり、<a href="用語/営業活動によるキャッシュフロー" class="internal-link" data-href="用語/営業活動によるキャッシュフロー">営業活動によるキャッシュフロー</a>に該当します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　短期借入金の返済による支出</div>
<div class="ap-choice-note">借入金の返済は<a href="用語/財務活動によるキャッシュフロー" class="internal-link" data-href="用語/財務活動によるキャッシュフロー">財務活動によるキャッシュフロー</a>に該当します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　有形固定資産の売却による収入</div>
<div class="ap-choice-note">固定資産の売却は<a href="用語/投資活動によるキャッシュフロー" class="internal-link" data-href="用語/投資活動によるキャッシュフロー">投資活動によるキャッシュフロー</a>に該当します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/キャッシュフロー計算書" class="internal-link" data-href="用語/キャッシュフロー計算書">キャッシュフロー計算書</a>は、一会計期間における資金（現金及び現金同等物）の増減を、<a href="用語/営業活動によるキャッシュフロー" class="internal-link" data-href="用語/営業活動によるキャッシュフロー">営業活動</a>・<a href="用語/投資活動によるキャッシュフロー" class="internal-link" data-href="用語/投資活動によるキャッシュフロー">投資活動</a>・<a href="用語/財務活動によるキャッシュフロー" class="internal-link" data-href="用語/財務活動によるキャッシュフロー">財務活動</a>に区分して表示する会計書類です。営業活動によるキャッシュフローは、商品・サービスの販売による収入や仕入・管理による支出など、企業の本業に係る事項を記載する区分です。商品の仕入れによる支出は営業活動に該当するため、正解は「イ」です。</p>
</div>
