<p class="ap-question"><strong>問68</strong>　企業の競争戦略におけるフォロワー戦略はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　上位企業の市場シェアを奪うことを目標に，製品，サービス，販売促進，流通チャネルなどのあらゆる面での差別化戦略をとる。</div>
<div class="ap-choice-note">これはチャレンジャー企業の<a href="用語/差別化戦略" class="internal-link" data-href="用語/差別化戦略">差別化戦略</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　潜在的な需要がありながら，大手企業が参入してこないような専門特化した市場に，限られた経営資源を集中する。</div>
<div class="ap-choice-note">これはニッチャー企業のニッチ戦略（<a href="用語/集中戦略" class="internal-link" data-href="用語/集中戦略">集中戦略</a>）の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　目標とする企業の戦略を観察し，迅速に模倣することで，開発や広告のコストを抑制し，市場での存続を図る。</div>
<div class="ap-choice-note">正しい。<a href="用語/競争戦略" class="internal-link" data-href="用語/競争戦略">競争戦略</a>におけるフォロワー企業の模倣による戦略です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　利潤，名声の維持・向上と最適市場シェアの確保を目標として，市場内の全ての顧客をターゲットにした全方位戦略をとる。</div>
<div class="ap-choice-note">これはリーダー企業の全方位戦略（フルライン戦略）の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>アメリカの経営学者であるフィリップ・コトラーが提唱した競争地位戦略において、企業の業界内地位は大きく次の4つに分けられます。</p>
<p>リーダー企業 … 市場においてナンバー1のシェアを誇る企業<br>チャレンジャー企業 … リーダーに次ぐシェアを保持し、リーダーに競争をしかける企業<br>ニッチャー企業 … 小さいながらも、特定の市場で独自の地位を築いている企業<br>フォロワー企業 … リーダーやチャレンジャーの戦略を模倣して、市場での地位を維持している企業</p>
<p>フォロワー企業は、リーダー企業からの報復を招かないことに注意をして<a href="用語/収益" class="internal-link" data-href="用語/収益">収益</a>性を高めていく必要があります。フォロワー企業には、リーダー企業にとってあまり美味しくない市場(例えば、中低価格志向の市場)にターゲットを定め、選択と集中を進める戦略がよく見られます。</p>
<p>アはチャレンジャー企業のとる<a href="用語/差別化戦略" class="internal-link" data-href="用語/差別化戦略">差別化戦略</a>、イはニッチャー企業のとるニッチ戦略（<a href="用語/集中戦略" class="internal-link" data-href="用語/集中戦略">集中戦略</a>）、エはリーダー企業のとる全方位戦略(フルライン戦略)です。ウが<a href="用語/競争戦略" class="internal-link" data-href="用語/競争戦略">競争戦略</a>におけるフォロワー企業の戦略として正しいです。</p>
</div>
</div>
