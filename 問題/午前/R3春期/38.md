<p class="ap-question"><strong>問38</strong>　攻撃者が行うフットプリンティングに該当するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　Webサイトのページを改ざんすることによって，そのWebサイトから社会的・政治的な主張を発信する。</div>
<div class="ap-choice-note"><a href="用語/ハクティビズム" class="internal-link" data-href="用語/ハクティビズム">ハクティビズム</a>の説明</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　攻撃前に，攻撃対象となるPC，サーバ及びネットワークについての情報を得る。</div>
<div class="ap-choice-note">正しい。<a href="用語/フットプリンティング" class="internal-link" data-href="用語/フットプリンティング">フットプリンティング</a>の説明</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　攻撃前に，攻撃に使用するPCの<span class="ap-mistake">メモリを増設することによって</span>，効率的に攻撃できるようにする。</div>
<div class="ap-choice-note"><a href="用語/フットプリンティング" class="internal-link" data-href="用語/フットプリンティング">フットプリンティング</a>と同じく攻撃の事前準備ですが，特に該当する用語はありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　<span class="ap-mistake">システムログに偽の痕跡を加えることによって</span>，攻撃後に追跡を逃れる。</div>
<div class="ap-choice-note"><a href="用語/ルートキット" class="internal-link" data-href="用語/ルートキット">ルートキット</a>の説明</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><strong><a href="用語/フットプリンティング" class="internal-link" data-href="用語/フットプリンティング">フットプリンティング</a></strong>は、特定のコンピュータにサイバー攻撃を行う前の事前準備として、攻撃対象を下調べする行為です。コンピュータやWebページ内の<a href="用語/脆弱性" class="internal-link" data-href="用語/脆弱性">脆弱性</a>の有無を調査したり、検索エンジン、公開データベースおよびツールを用いて個人や組織の情報を収集したりするなど、攻撃の足掛かりとなる情報を得る行為が該当します。</p>
</div>
