<p class="ap-question"><strong>問65</strong>　国や地方公共団体などが，環境への配慮を積極的に行っていると評価されている製品・サービスを選んでいる。この取組を何というか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　CSR</div>
<div class="ap-choice-note">これは<a href="用語/CSR" class="internal-link" data-href="用語/CSR">CSR</a>の説明です。企業の社会的責任の考え方であり、公共調達での製品選定の名称ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　エコマーク認定</div>
<div class="ap-choice-note">環境負荷の少ない商品を認定する制度であり、国等が製品・サービスを選んで購入する取組の名称ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　環境アセスメント</div>
<div class="ap-choice-note">大規模開発等の環境への<a href="用語/影響" class="internal-link" data-href="用語/影響">影響</a>を事前評価する手続きであり、調達時の選定方針ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　グリーン購入</div>
<div class="ap-choice-note">正しい。グリーン購入は、環境負荷の少ない製品、環境保護に取組む企業から優先的に購入する<a href="用語/調達" class="internal-link" data-href="用語/調達">調達</a>方針です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>グリーン購入とは、製品や<a href="用語/サービス" class="internal-link" data-href="用語/サービス">サービス</a>を購入する際に、環境を考慮して、<a href="用語/必要性" class="internal-link" data-href="用語/必要性">必要性</a>をよく考え、環境への負荷ができるだけ少ないものを選んで購入することです。我が国においては、2001年(平成13年)にグリーン購入法（国等による環境物品等の<a href="用語/調達" class="internal-link" data-href="用語/調達">調達</a>の推進等に関する法律）が施行されました。この法律は、国等の機関にグリーン購入を義務づけるとともに、地方公共団体や事業者・国民にもグリーン購入に努めることを求めています（環境省HPより引用）。</p>
<p>アの<a href="用語/CSR" class="internal-link" data-href="用語/CSR">CSR</a>は、Corporate Social Responsibilityの略。企業活動において経済的成長だけでなく、環境や社会からの要請に対し、責任を果たすことが企業価値の向上につながるという考え方です。イのエコマーク認定は、生産から廃棄までを通じて環境への負荷が少なく、環境保全に役立つと認められた商品を認定する制度です。認定された商品はエコマークを表示することができます。ウの環境アセスメント(環境<a href="用語/影響" class="internal-link" data-href="用語/影響">影響</a>評価)は、主として大規模開発事業等による環境への<a href="用語/影響" class="internal-link" data-href="用語/影響">影響</a>を事前に調査することによって、予測、評価を行う手続きのことです。設問の「環境への配慮を積極的に行っていると評価されている製品・<a href="用語/サービス" class="internal-link" data-href="用語/サービス">サービス</a>を選んで購入する」取組は、エのグリーン購入が該当します。</p>
</div>
