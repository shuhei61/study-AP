<p class="ap-question"><strong>問55</strong>　プロジェクトメンバーが16人のとき，1対1の総当たりでプロジェクトメンバー相互の顔合わせ会を行うためには，延べ何時間の顔合わせ会が必要か。ここで，顔合わせ会1回の所要時間は0.5時間とする。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　8</div>
<div class="ap-choice-note">16人から2人を選ぶ組合せは120通りであり，8時間では総当たりの回数・所要時間に足りません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　16</div>
<div class="ap-choice-note">メンバー人数と同じ16時間では，全員の1対1の組合せ分（120組）をカバーできません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　30</div>
<div class="ap-choice-note">120組×0.5時間＝60時間となるため，30時間は不足です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　60</div>
<div class="ap-choice-note">正しい。16C2＝120組，120組×0.5時間＝60時間です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>組合せ数を求める公式を利用して答えを導きます。n個の中から異なるr個を取り出す組合せの場合の数 nCr＝n!／((n－r)!×r!)。16人がいるとき、異なる2人の組合せが何通りあるかは 16C2＝(16×15)／(2×1)＝120組です。顔合わせ会1回の所要時間は0.5時間のため、120組×0.5時間＝60時間。よって、顔合わせ会に要する総時間は60時間が適切です。</p>
<p>別の解法として、全ての組合せを列挙していく方法も考えられます。16人をそれぞれA、B、…、Pとすると、AとB〜P（15通り）、BとC〜P（14通り）、…、OとP（1通り）となり、15＋14＋13＋…＋2＋1＝120組、120組×0.5時間＝60時間となります。</p>
