<p class="ap-question"><strong>問5</strong>　A，B，Cの順序で入力されるデータがある。各データについてスタックへの挿入と取出しを1回ずつ行うことができる場合，データの出力順序は何通りあるか。<br><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05.png" alt="問題図" width="164" height="110" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　3</div>
<div class="ap-choice-note"><a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>で実現できる順序を3通りと数えた誤り。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　4</div>
<div class="ap-choice-note">実現可能な順序を1つ見落とした誤り（正解は5通り）。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　5</div>
<div class="ap-choice-note">正しい。3!＝6通りのうち，C，A，Bは<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>の制約で不可能。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　6</div>
<div class="ap-choice-note">3!＝6通りすべて可能とした誤り。C，A，Bは出力できない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>A，B，Cの出力順序としては「3!＝6種類」があります。それぞれが出力可能であるかを検証します。</p>
<p><strong>A，B，C</strong> … push(A) → pop → push(B) → pop → push(C) → pop の順序で出力可能です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05_1.png" alt="解説図" width="406" height="133" ></p>
<p><strong>A，C，B</strong> … push(A) → pop → push(B) → push(C) → pop → pop の順序で出力可能です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05_2.png" alt="解説図" width="406" height="133" ></p>
<p><strong>B，A，C</strong> … push(A) → push(B) → pop → pop → push(C) → pop の順序で出力可能です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05_3.png" alt="解説図" width="406" height="133" ></p>
<p><strong>B，C，A</strong> … push(A) → push(B) → pop → push(C) → pop → pop の順序で出力可能です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05_4.png" alt="解説図" width="406" height="133" ></p>
<p><strong>C，A，B</strong> … push(A) → push(B) → push(C) → pop → pop ❌ ※Cの出力後，Bより先にAを出力することができないため無理な順序です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05_5.png" alt="解説図" width="346" height="133" ></p>
<p><strong>C，B，A</strong> … push(A) → push(B) → push(C) → pop → pop → pop の順序で出力可能です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/5/05_6.png" alt="解説図" width="406" height="133" ></p>
<p>以上より，データの出力順序は5通りになります。したがって「ウ」が正解です。<a href="用語/スタック" class="internal-link" data-href="用語/スタック">スタック</a>の構造上，後からpushされた要素を出力した後，その要素より2つ以上前にpushされた要素を続けて出力することはできません。</p>
