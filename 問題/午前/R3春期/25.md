<p class="ap-question"><strong>問25</strong>　図の論理回路において，S=1，R=1，X=0，Y=1のとき，Sを一旦0にした後，再び1に戻した。この操作を行った後のX，Yの値はどれか。<br><img class="ap-fig" src="../../../assets/問題/午前/R3春期/25/25.png" alt="問題図" width="181" height="90" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　X＝0，Y＝0</div>
<div class="ap-choice-note">Sを0にしたあと安定した状態ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　X＝0，Y＝1</div>
<div class="ap-choice-note">操作前の初期状態です。Sを0→1としたあとの出力ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　X＝1，Y＝0</div>
<div class="ap-choice-note">正しい。Sを0にしたあと再び1に戻し，回路が安定したときの出力です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　X＝1，Y＝1</div>
<div class="ap-choice-note">両出力が1の状態は本回路では安定しません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>図の論理回路は<a href="用語/フリップフロップ" class="internal-link" data-href="用語/フリップフロップ">フリップフロップ</a>と呼ばれ、2つの回路の安定した状態によって1<a href="用語/ビット" class="internal-link" data-href="用語/ビット">ビット</a>の情報を保持する回路です。現在と異なる入力が与えられると、次の入力があるまでその状態を保持しようとします。問題文の初期状態「S=1、R=1、X=0、Y=1」だと、回路は次のようになっています。</p>
<p>次にSを0に変えると、次のような状態になります。他方の出力がもう一方の入力に影響を与えるので、2つの回路が安定するまで何回か入出力を繰り返します。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/25/25_1.png" alt="解説図" width="156" height="82" ></p>
<p>そしてSを1に戻すと次のような状態になります。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/25/25_2.png" alt="解説図" width="390" height="253" ></p>
<p>回路はこの状態で安定するので、出力の値は「X＝1，Y＝0」となります。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/25/25_3.png" alt="解説図" width="156" height="83" ></p>
