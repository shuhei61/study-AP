<p class="ap-question"><strong>問20</strong>　Hadoopの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　JavaEE仕様に準拠したアプリケーションサーバ</div>
<div class="ap-choice-note">WildFly（旧JBoss）などのアプリケーションサーバの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　LinuxやWindowsなどの様々なプラットフォーム上で動作するWebサーバ</div>
<div class="ap-choice-note"><a href="用語/Apache" class="internal-link" data-href="用語/Apache">Apache</a>やNginxなどの<a href="用語/Webサーバ" class="internal-link" data-href="用語/Webサーバ">Webサーバ</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　機能の豊富さが特徴のRDBMS</div>
<div class="ap-choice-note"><a href="用語/PostgreSQL" class="internal-link" data-href="用語/PostgreSQL">PostgreSQL</a>などのRDBMSの説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　大規模なデータセットを分散処理するためのソフトウェアライブラリ</div>
<div class="ap-choice-note">正しい。Hadoopは大規模なデータの<a href="用語/分散処理" class="internal-link" data-href="用語/分散処理">分散処理</a>基盤を提供するソフトウェアライブラリです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>Hadoop（ハドゥープ）は、ペタバイト級の大規模データの蓄積・処理の<a href="用語/分散処理" class="internal-link" data-href="用語/分散処理">分散処理</a>を実現する<a href="用語/ミドルウェア" class="internal-link" data-href="用語/ミドルウェア">ミドルウェア</a>です。Googleが論文として発表した<a href="用語/分散処理" class="internal-link" data-href="用語/分散処理">分散処理</a><a href="用語/フレームワーク" class="internal-link" data-href="用語/フレームワーク">フレームワーク</a>「MapReduce」および分散ファイルシステム「Google File System」を基盤技術に、オープンソースとしてJavaで実装したものです。分散ファイルシステムの<a href="用語/HDFS" class="internal-link" data-href="用語/HDFS">HDFS</a>（Hadoop Distributed File System）と、<a href="用語/分散処理" class="internal-link" data-href="用語/分散処理">分散処理</a>の<a href="用語/フレームワーク" class="internal-link" data-href="用語/フレームワーク">フレームワーク</a>であるHadoop MapReduceの2つから構成されます。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3春期/20/20.png" alt="解説図" width="347" height="83" ></p>
<p>したがって「エ」が適切な説明です。</p>
