<p class="ap-question"><strong>問41</strong>　経済産業省とIPAが策定した"サイバーセキュリティ経営ガイドライン(Ver2.0)"の説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　企業がIT活用を推進していく中で，サイバー攻撃から企業を守る観点で経営者が認識すべき3原則と，情報セキュリティ対策を実施する上での責任者となる担当幹部に，経営者が指示すべき重要10項目をまとめたもの</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/サイバーセキュリティ経営ガイドライン" class="internal-link" data-href="用語/サイバーセキュリティ経営ガイドライン">サイバーセキュリティ経営ガイドライン</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　経営者が情報セキュリティについて方針を示し，マネジメントシステムの要求事項を満たすルールを定め，組織が保有する情報資産をCIAの観点から維持管理し，継続的に見直すためのプロセス及び管理策を体系的に規定したもの</div>
<div class="ap-choice-note">これは<a href="用語/情報セキュリティ方針" class="internal-link" data-href="用語/情報セキュリティ方針">情報セキュリティ方針</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　事業体のITに関する経営者の活動を，大きくITガバナンス(統制)とITマネジメント(管理)に分割し，具体的な目標と工程として40のプロセスを定義したもの</div>
<div class="ap-choice-note">これは<a href="用語/COBIT" class="internal-link" data-href="用語/COBIT">COBIT</a>の説明です。アメリカの情報システムコントロール協会(ISADA)と<a href="用語/ITガバナンス" class="internal-link" data-href="用語/ITガバナンス">ITガバナンス</a>協会(ITGI)が策定しているIT管理についてのフレームワークです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　世界的規模で生じているサイバーセキュリティ上の脅威の深刻化に関して，企業の経営者を支援する施策を総合的かつ効果的に推進するための国の責務を定めたもの</div>
<div class="ap-choice-note">サイバーセキュリティ基本法の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/サイバーセキュリティ経営ガイドライン" class="internal-link" data-href="用語/サイバーセキュリティ経営ガイドライン">サイバーセキュリティ経営ガイドライン</a>は、<a href="用語/サイバー攻撃" class="internal-link" data-href="用語/サイバー攻撃">サイバー攻撃</a>から企業を守る観点で、経営者が認識する必要のある「3原則」、及び経営者が情報セキュリティ対策を実施する上での責任者となる担当幹部(<a href="用語/CISO" class="internal-link" data-href="用語/CISO">CISO</a>等)に指示すべき「重要10項目」をまとめたものです。具体的には、経営者のリーダーシップの下での体制整備と対策の進め方、社会や<a href="用語/ステークホルダ" class="internal-link" data-href="用語/ステークホルダ">ステークホルダ</a>に対する情報開示のあり方などが取りまとめられています。したがって適切な説明は「ア」です。</p>
</div>
