<p class="ap-question"><strong>問42</strong>　JPCERTコーディネーションセンターの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　産業標準化法に基づいて経済産業省に設置されている審議会であり，産業標準化全般に関する調査・審議を行っている。</div>
<div class="ap-choice-note">これは日本産業標準調査会(<a href="用語/JISC" class="internal-link" data-href="用語/JISC">JISC</a>)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　電子政府推奨暗号の安全性を評価・監視し，暗号技術の適切な実装法・運用法を調査・検討するプロジェクトであり，総務省及び経済産業省が共同で運営する暗号技術検討会などで構成される。</div>
<div class="ap-choice-note">これは<a href="用語/CRYPTREC" class="internal-link" data-href="用語/CRYPTREC">CRYPTREC</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　特定の政府機関や企業から独立した組織であり，国内のコンピュータセキュリティインシデントに関する報告の受付，対応の支援，発生状況の把握，手口の分析，再発防止策の検討や助言を行っている。</div>
<div class="ap-choice-note">正しい。<a href="用語/JPCERTコーディネーションセンター" class="internal-link" data-href="用語/JPCERTコーディネーションセンター">JPCERTコーディネーションセンター</a>(JPCERT/CC)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　内閣官房に設置され，我が国をサイバー攻撃から防衛するための司令塔機能を担う組織である。</div>
<div class="ap-choice-note">これは<a href="用語/国家サイバー統括室" class="internal-link" data-href="用語/国家サイバー統括室">国家サイバー統括室</a>(NCO)の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/JPCERTコーディネーションセンター" class="internal-link" data-href="用語/JPCERTコーディネーションセンター">JPCERTコーディネーションセンター</a>(JPCERT/CC)は、インターネットを介して発生する侵入やサービス妨害等のコンピュータセキュリティインシデントについて、日本国内のサイトに関する報告の受け付け、対応の支援、発生状況の把握、手口の分析、再発防止のための対策の検討や助言などを、技術的な立場から行なっています。特定の政府機関や企業からは独立した中立の組織として、日本における<a href="用語/情報セキュリティ" class="internal-link" data-href="用語/情報セキュリティ">情報セキュリティ</a>対策活動の向上に積極的に取り組んでいます。</p>
