<p class="ap-question"><strong>問60</strong>　システム監査人が行う改善提案のフォローアップとして，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　改善提案に対する改善の実施を監査対象部門の長に指示する。</div>
<div class="ap-choice-note">改善の実施指示は監査対象部門が主体で行うべきことであり，監査人の関与は不適切です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　改善提案に対する監査対象部門の改善実施プロジェクトの管理を行う。</div>
<div class="ap-choice-note">改善実施プロジェクトの管理は監査対象部門が主体で行うべきことであり，監査人の関与は不適切です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　改善提案に対する監査対象部門の改善状況をモニタリングする。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/フォローアップ" class="internal-link" data-href="用語/フォローアップ">フォローアップ</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　改善提案の内容を監査対象部門に示した上で改善実施計画を策定する。</div>
<div class="ap-choice-note">改善実施計画の策定は監査対象部門が主体で行うべきことであり，監査人の関与は不適切です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>システム監査人は、監査報告書に<a href="用語/改善提案" class="internal-link" data-href="用語/改善提案">改善提案</a>を記載した場合、その提案に基づいて必要な措置が講じられているかを確認するとともに、改善状況を継続的に<a href="用語/モニタリング" class="internal-link" data-href="用語/モニタリング">モニタリング</a>しなければなりません。この<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>後の活動を「<a href="用語/フォローアップ" class="internal-link" data-href="用語/フォローアップ">フォローアップ</a>」と呼びます。<a href="用語/フォローアップ" class="internal-link" data-href="用語/フォローアップ">フォローアップ</a>は、監査対象部門が主体となって実施される改善を、監査人が事後的に確認するという性質のものです。このため、監査人が改善活動を自ら実施することはなく、また実施に関して責任を負いません。<a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>の策定やその実行への関与は、監査人の独立性と客観性を損なうことに留意しなければなりません。監査人の役割は被監査側が実施する<a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>を見守り、<a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>が適切に遂行されるように助言を行うことです。したがって、監査人が行う<a href="用語/フォローアップ" class="internal-link" data-href="用語/フォローアップ">フォローアップ</a>として適切なのは、「ウ」の"改善状況を<a href="用語/モニタリング" class="internal-link" data-href="用語/モニタリング">モニタリング</a>する"です。</p>
<p>「ア」改善の実施を指示する、「イ」実施状況の管理を行う、「エ」改善実施計画を策定する、はいずれも監査対象部門が主体となって実施すべきことであり、監査人の関与は不適切です。</p>
</div>
