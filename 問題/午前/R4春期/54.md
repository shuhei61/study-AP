<p class="ap-question"><strong>問54</strong>　プロジェクトのコンティンジェンシー計画において決定することとして，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　あらかじめ定義された，ある条件のときにだけ実行する対応策</div>
<div class="ap-choice-note">正しい。コンティンジェンシー計画において決定することです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　活動リストの活動ごとに必要な資源</div>
<div class="ap-choice-note"><a href="用語/資源の見積り" class="internal-link" data-href="用語/資源の見積り">資源の見積り</a>において決定することです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　プロジェクトに適用する品質の要求事項及び規格</div>
<div class="ap-choice-note"><a href="用語/品質の計画" class="internal-link" data-href="用語/品質の計画">品質の計画</a>において決定することです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プロジェクトのステークホルダの情報及びコミュニケーションのニーズ</div>
<div class="ap-choice-note"><a href="用語/コミュニケーションの計画" class="internal-link" data-href="用語/コミュニケーションの計画">コミュニケーションの計画</a>において決定することです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/プロジェクトマネジメント" class="internal-link" data-href="用語/プロジェクトマネジメント">プロジェクトマネジメント</a>におけるコンティンジェンシー計画とは、<a href="用語/プロジェクト" class="internal-link" data-href="用語/プロジェクト">プロジェクト</a>が抱える潜在的な<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>が現実化した際に、<a href="用語/プロジェクト" class="internal-link" data-href="用語/プロジェクト">プロジェクト</a>が被る損害を最小限にとどめ、速やかにその緊急事態の克服を図るための手続きが記述された対応計画のことです。コンティンジェンシー計画は、既知の<a href="用語/リスク" class="internal-link" data-href="用語/リスク">リスク</a>に対する対応計画の一部として策定され、ある事象が起きた場合に限り使用するものなので、発動条件となる事象とそのとき実行する対応策を記述します。</p>
<p>したがって「ア」の記述が適切です。</p>
</div>
