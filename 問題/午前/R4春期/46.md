<p class="ap-question"><strong>問46</strong>　モジュールの独立性の尺度であるモジュール結合度は，低いほど独立性が高くなる。次のうち，モジュールの独立性が最も高い結合はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　外部結合</div>
<div class="ap-choice-note">大域宣言された単一のデータを複数のモジュールが参照する結合であり、<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>より結合度が強い。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　共通結合</div>
<div class="ap-choice-note">大域宣言されたレコードや構造体などのデータ構造を複数のモジュールが参照する結合であり、独立性は<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>より低い。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　スタンプ結合</div>
<div class="ap-choice-note">処理に必要なデータをレコードや構造体などのデータ構造として受け渡す結合であり、<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>より結合度が強い。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　データ結合</div>
<div class="ap-choice-note">正しい。処理に必要なデータだけを単一のパラメータとして受け渡す結合で、結合度が最も低い。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>モジュール結合度とは、モジュール同士の関連性の強さを表し、モジュール結合度が弱いほど関連するモジュールに変更があった場合の影響を受けにくくなるので、<a href="用語/モジュールの独立性" class="internal-link" data-href="用語/モジュールの独立性">モジュールの独立性</a>が高まり<a href="用語/保守性" class="internal-link" data-href="用語/保守性">保守性</a>が向上します。</p>
<p>モジュール結合度の強弱は、モジュール間のデータの受渡し方法などにより6段階のレベルに分類されています。</p>
<ul>
<li><a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>（弱い↑）… 処理に必要なデータだけを単一のパラメータとして受け渡している</li>
<li>スタンプ結合 … 処理に必要なデータだけをレコードや構造体などのデータ構造として受け渡している</li>
<li><a href="用語/制御結合" class="internal-link" data-href="用語/制御結合">制御結合</a> … モジュールの動作を制御するための要素を受け渡している</li>
<li>外部結合 … 大域宣言された単一のデータを、複数のモジュールが参照している</li>
<li>共通結合 … 大域宣言されたレコードや構造体などのデータ構造を、複数のモジュールが参照している</li>
<li>内容結合（強い↓）… モジュールの内部を直接参照・使用している</li>
</ul>
<p><a href="用語/モジュールの独立性" class="internal-link" data-href="用語/モジュールの独立性">モジュールの独立性</a>が最も高くなるのは、結合度が最も低い「<a href="用語/データ結合" class="internal-link" data-href="用語/データ結合">データ結合</a>」になります。したがって正解は「エ」です。</p>
</div>
