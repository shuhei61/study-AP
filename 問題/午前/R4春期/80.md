<p class="ap-question"><strong>問80</strong>　欧州へ電子部品を輸出するには，RoHS指令への対応が必要である。このRoHS指令の目的として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　家電製品から有用な部分や材料をリサイクルし，廃棄物を減量するとともに，資源の有効利用を推進する。</div>
<div class="ap-choice-note">これはWEEE指令の目的です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　機器が発生する電磁妨害が，無線通信機器及びその他の機器が意図する動作を妨げるレベルを超えないようにする。</div>
<div class="ap-choice-note">これはEMC指令の目的です。電気電子機器を欧州に輸出する際に適合させる必要があります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　大量破壊兵器の開発及び拡散，通常兵器の過剰備蓄に関わるおそれがある場合など，国際社会の平和と安全を脅かす輸出行為を防止する。</div>
<div class="ap-choice-note">これは<a href="用語/外国為替及び外国貿易法" class="internal-link" data-href="用語/外国為替及び外国貿易法">外国為替及び外国貿易法</a>に基づく安全保障輸出管理（リスト規制やキャッチオール規制）の目的です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　電気電子製品の生産から処分までの全ての段階で，有害物質が環境及び人の健康に及ぼす危険を最小化する。</div>
<div class="ap-choice-note">正しい。<a href="用語/RoHS指令" class="internal-link" data-href="用語/RoHS指令">RoHS指令</a>の目的です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/RoHS指令" class="internal-link" data-href="用語/RoHS指令">RoHS指令</a>(ろーずしれい)は、EUが発効した電気電子機器の製造段階における特定有害物質の使用制限に関する規則です。特定有害物質とは、鉛、水銀、カドミウム、六価クロムなど合計10物質で、最終的に埋立てや焼却で処分されるときに人体への悪影響がないように、これらの含有率を最大許容濃度以下にすることを定めています。<br>EU加盟国の製造業者は本指令に従った製品を市場に出さなければならず、EU加盟国の輸入業者には、本指令に準拠する電気電子機器であるかどうかを適切に評価するための<a href="用語/適合性評価" class="internal-link" data-href="用語/適合性評価">適合性評価</a>手順を製造業者が実行していることを確認する義務があります。そのため、EUに電子部品を輸出する国内製造業者は<a href="用語/RoHS指令" class="internal-link" data-href="用語/RoHS指令">RoHS指令</a>に対応する必要があります。<br>したがって「エ」の記述が適切です。</p>
