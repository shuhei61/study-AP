<p class="ap-question"><strong>問53</strong>　ソフトウェア開発プロジェクトにおいて，表の全ての作業を完了させるために必要な期間は最短で何日間か。<br><img class="ap-fig" src="../../../assets/問題/午前/R4春期/53/53.png" alt="問題図" width="454" height="200" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　80</div>
<div class="ap-choice-note">並行作業を考慮せず「要件定義→設計→利用者マニュアル作成→利用者教育」のパスだけを合計した誤答です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　95</div>
<div class="ap-choice-note">製造とテストの所要日数を正しく加味できていない誤答です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　100</div>
<div class="ap-choice-note">正しい。<a href="用語/クリティカルパス" class="internal-link" data-href="用語/クリティカルパス">クリティカルパス</a>上の作業日数の合計は30＋20＋25＋15＋10＝100日です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　120</div>
<div class="ap-choice-note">並行して行える作業も直列に足し合わせた誤答です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>作業の先後関係に注意しながら設問の表を<a href="用語/アローダイアグラム" class="internal-link" data-href="用語/アローダイアグラム">アローダイアグラム</a>に変換すると以下のようになります。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R4春期/53/53a.png" alt="解説図" width="469" height="96" ></p>
<p>並行作業となる部分に注目すると、利用者マニュアル作成(20日)よりも製造(25日)＋テスト(15日)の方が日数を要するので、<a href="用語/クリティカルパス" class="internal-link" data-href="用語/クリティカルパス">クリティカルパス</a>は「要件定義→設計→製造→テスト→利用者教育」、最短所要日数は「30＋20＋25＋15＋10＝100日」です。したがって「ウ」が正解となります。</p>
</div>
