<p class="ap-question"><strong>問1</strong>　浮動小数点数を，仮数部が7ビットである表示形式のコンピュータで計算した場合，情報落ちが発生しないものはどれか。ここで，仮数部が7ビットの表示形式とは次のフォーマットであり，( )<sub>2</sub>内は2進数，Yは指数である。また，{ }内を先に計算するものとする。<br>(1.X<sub>1</sub>X<sub>2</sub>X<sub>3</sub>X<sub>4</sub>X<sub>5</sub>X<sub>6</sub>X<sub>7</sub>)<sub>2</sub>×2<sup>Y</sup></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　{(1.1)<sub>2</sub>×2<sup>-3</sup>＋(1.0)<sub>2</sub>×2<sup>-4</sup>}＋(1.0)<sub>2</sub>×2<sup>5</sup></div>
<div class="ap-choice-note">正しい。{ }内を指数-3に揃えて加算すると(1.0)<sub>2</sub>×2<sup>-2</sup>となり，続けて(1.0)<sub>2</sub>×2<sup>5</sup>と加算しても<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>7ビット内に収まり，<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>は発生しない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　{(1.1)<sub>2</sub>×2<sup>-3</sup>－(1.0)<sub>2</sub>×2<sup>-4</sup>}＋(1.0)<sub>2</sub>×2<sup>5</sup></div>
<div class="ap-choice-note">{ }内の減算後は(1.0)<sub>2</sub>×2<sup>-3</sup>。これに(1.0)<sub>2</sub>×2<sup>5</sup>を加算すると指数5に揃えたとき小さい方の仮数が落ち，必要な<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>が8ビットとなり<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>が発生する。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　{(1.0)<sub>2</sub>×2<sup>5</sup>＋(1.1)<sub>2</sub>×2<sup>-3</sup>}＋(1.0)<sub>2</sub>×2<sup>-4</sup></div>
<div class="ap-choice-note">大きい(1.0)<sub>2</sub>×2<sup>5</sup>に小さい項を足しても結果は(1.0)<sub>2</sub>×2<sup>5</sup>のままとなり，必要な<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>が9ビットとなり<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>が発生する。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　{(1.0)<sub>2</sub>×2<sup>5</sup>－(1.0)<sub>2</sub>×2<sup>-4</sup>}＋(1.1)<sub>2</sub>×2<sup>-3</sup></div>
<div class="ap-choice-note">(1.0)<sub>2</sub>×2<sup>5</sup>から小さい(1.0)<sub>2</sub>×2<sup>-4</sup>を引いても結果は(1.0)<sub>2</sub>×2<sup>5</sup>のままとなり，必要な<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>が9ビットとなり<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>が発生する。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>は、浮動小数点演算において、絶対値の大きな数と絶対値の小さな数の加減算を行ったとき、絶対値の小さな数の有効けたの一部または全部が結果に反映されないことをいいます。一般的に数多くの数値の加算を行う場合には、絶対値の小さなものから順番に計算すると<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>を抑制することができます。</p>
<p>浮動小数点数同士の計算では、指数が小さい方の<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>を指数が大きい方に揃え、<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>同士を演算します。実際に計算していくと以下のようになります。</p>
<p>ア：(1.1)<sub>2</sub>×2<sup>-3</sup>＋(1.0)<sub>2</sub>×2<sup>-4</sup>＝(1.1)<sub>2</sub>×2<sup>-3</sup>＋(0.1)<sub>2</sub>×2<sup>-3</sup>（<a href="用語/指数部" class="internal-link" data-href="用語/指数部">指数部</a>を-3に揃える）＝(10.0)<sub>2</sub>×2<sup>-3</sup>＝(1.0)<sub>2</sub>×2<sup>-2</sup>。(1.0)<sub>2</sub>×2<sup>-2</sup>＋(1.0)<sub>2</sub>×2<sup>5</sup>＝(0.0000001)<sub>2</sub>×2<sup>5</sup>＋(1.0)<sub>2</sub>×2<sup>5</sup>（<a href="用語/指数部" class="internal-link" data-href="用語/指数部">指数部</a>を5に揃える）＝(1.0000001)<sub>2</sub>×2<sup>5</sup>。<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>は発生しない。</p>
<p>イ：(1.1)<sub>2</sub>×2<sup>-3</sup>－(1.0)<sub>2</sub>×2<sup>-4</sup>＝(1.0)<sub>2</sub>×2<sup>-3</sup>のあと，(1.0)<sub>2</sub>×2<sup>-3</sup>＋(1.0)<sub>2</sub>×2<sup>5</sup>＝(0.00000001)<sub>2</sub>×2<sup>5</sup>＋(1.0)<sub>2</sub>×2<sup>5</sup>→(1.0)<sub>2</sub>×2<sup>5</sup>（<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>が発生）。必要な<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>が8ビットとなり、7ビットの<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>からあふれた部分が計算結果に反映されない。</p>
<p>ウ：(1.0)<sub>2</sub>×2<sup>5</sup>＋(1.1)<sub>2</sub>×2<sup>-3</sup>＝(1.0)<sub>2</sub>×2<sup>5</sup>＋(0.000000011)<sub>2</sub>×2<sup>5</sup>→(1.0)<sub>2</sub>×2<sup>5</sup>（<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>が発生）。必要な<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>が9ビットとなる。</p>
<p>エ：(1.0)<sub>2</sub>×2<sup>5</sup>－(1.0)<sub>2</sub>×2<sup>-4</sup>＝(1.0)<sub>2</sub>×2<sup>5</sup>－(0.000000001)<sub>2</sub>×2<sup>5</sup>→(1.0)<sub>2</sub>×2<sup>5</sup>（<a href="用語/情報落ち" class="internal-link" data-href="用語/情報落ち">情報落ち</a>が発生）。必要な<a href="用語/仮数部" class="internal-link" data-href="用語/仮数部">仮数部</a>が9ビットとなる。</p>
</div>
