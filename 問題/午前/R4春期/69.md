<p class="ap-question"><strong>問69</strong>　バイラルマーケティングの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　顧客の好みや欲求の多様化に対応するために，画一的なマーケティングを行うのではなく，顧客一人ひとりの興味関心に合わせてマーケティングを行う手法</div>
<div class="ap-choice-note">これは<a href="用語/ワントゥワンマーケティング" class="internal-link" data-href="用語/ワントゥワンマーケティング">ワントゥワンマーケティング</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　市場全体をセグメント化せずに一つとして捉え，一つの製品を全ての購買者に対し，画一的なマーケティングを行う手法</div>
<div class="ap-choice-note">これは<a href="用語/マスマーケティング" class="internal-link" data-href="用語/マスマーケティング">マスマーケティング</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　実店舗での商品販売，ECサイトなどのバーチャル店舗販売など複数のチャネルを連携させ，顧客がチャネルを意識せず購入できる利便性を実現する手法</div>
<div class="ap-choice-note">これは<a href="用語/オムニチャネル" class="internal-link" data-href="用語/オムニチャネル">オムニチャネル</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　人から人へ，プラスの評価が口コミで爆発的に広まりやすいインターネットの特長を生かす手法</div>
<div class="ap-choice-note">正しい。<a href="用語/バイラルマーケティング" class="internal-link" data-href="用語/バイラルマーケティング">バイラルマーケティング</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/バイラルマーケティング" class="internal-link" data-href="用語/バイラルマーケティング">バイラルマーケティング</a>は、インターネット上では人から人への口コミが連鎖して爆発的に広がっていく性質を利用し、<a href="用語/SNS" class="internal-link" data-href="用語/SNS">SNS</a>やメール等での拡散を狙って話題になりやすい記事や企画を仕掛けることで、商品やサービスへの認知や興味を高めようとするマーケティング手法です。口コミを活用するので少ない広告宣伝費用で大きなプロモーション効果が望めますが、利用者に自発的なアクションを促す仕掛け作りが難しい、拡散の度合いがコントロールしにくいという面もあります。バイラル(viral)は、「ウイルス性の」や「感染的な」という意味を持つ言葉です。</p>
