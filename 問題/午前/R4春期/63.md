<p class="ap-question"><strong>問63</strong>　BPOの説明はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　災害や事故で被害を受けても，重要事業を中断させない，又は可能な限り中断期間を短くする仕組みを構築すること</div>
<div class="ap-choice-note">これは<a href="用語/BCP" class="internal-link" data-href="用語/BCP">BCP</a>(Business Continuity Plan：<a href="用語/事業継続計画" class="internal-link" data-href="用語/事業継続計画">事業継続計画</a>)の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　社内業務のうちコアビジネスでない事業に関わる業務の一部又は全部を，外部の専門的な企業に委託すること</div>
<div class="ap-choice-note">正しい。<a href="用語/BPO" class="internal-link" data-href="用語/BPO">BPO</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　製品の基準生産計画，部品表及び在庫情報を基に，資材の所要量と必要な時期を求め，これを基準に資材の手配，納入の管理を支援する生産管理手法のこと</div>
<div class="ap-choice-note">これは<a href="用語/MRP" class="internal-link" data-href="用語/MRP">MRP</a>(Material Requirements Planning)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プロジェクトを，戦略との適合性や費用対効果，リスクといった観点から評価を行い，情報化投資のバランスを管理し，最適化を図ること</div>
<div class="ap-choice-note">これは<a href="用語/IT投資マネジメント" class="internal-link" data-href="用語/IT投資マネジメント">IT投資マネジメント</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/BPO" class="internal-link" data-href="用語/BPO">BPO</a>(Business Process Outsourcing)は、経営資源の集中、業務効率の向上、業務コストの削減などを目的に、業務プロセスを対象として<a href="用語/アウトソーシング" class="internal-link" data-href="用語/アウトソーシング">アウトソーシング</a>を実施することです。企業内部の間接部門で行われていた労務管理、人事、経理、給与計算、データの入出力や処理などのバックオフィス業務の一部や、<a href="用語/コールセンター" class="internal-link" data-href="用語/コールセンター">コールセンター</a>の業務全体などを外部の専門業者に委託するケースがこれに該当します。</p>
</div>
</div>
