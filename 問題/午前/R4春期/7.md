<p class="ap-question"><strong>問7</strong>　プログラム言語のうち，ブロックの範囲を指定する方法として特定の記号や予約語を用いず，等しい文字数の字下げを用いるという特徴をもつものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　C</div>
<div class="ap-choice-note">Cは「{」と「}」で囲むことでコードブロックを指定します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　Java</div>
<div class="ap-choice-note">Javaは「{」と「}」で囲むことでコードブロックを指定します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　PHP</div>
<div class="ap-choice-note">PHPは「{」と「}」で囲むことでコードブロックを指定します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　Python</div>
<div class="ap-choice-note">正しい。<a href="用語/Python" class="internal-link" data-href="用語/Python">Python</a>はインデントの深さでコードブロックを指定する<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/Python" class="internal-link" data-href="用語/Python">Python</a>(パイソン)は、1991年にグイド・ヴァンロッサム氏によって開発された汎用の高水準<a href="用語/プログラミング" class="internal-link" data-href="用語/プログラミング">プログラミング</a>言語です。コードブロックのインデントが構文規則となっていることがソースコード上の特徴です。小さなプログラムから大規模なシステムまで、そしてデスクトップアプリケーションからWebアプリケーションの開発まで様々な場面で使用されています("YouTube"や"Dropbox"などが有名)。簡潔な文法と使いやすさ、対応するプラットフォームの多さ、優れた<a href="用語/ライブラリ" class="internal-link" data-href="用語/ライブラリ">ライブラリ</a>の存在等により、AI開発に適した言語としても人気が過熱しています。</p>
<p>C、Java、および選択肢のPHPを含む一般的な<a href="用語/プログラミング" class="internal-link" data-href="用語/プログラミング">プログラミング</a>言語では、コードブロックを「{」と「}」で囲って示します。コードブロック内の字下げ(インデント)は、<a href="用語/コーディング" class="internal-link" data-href="用語/コーディング">コーディング</a>規約上の指定であり言語仕様上は任意です。また、コードブロック内の処理が1つのときには「{ }」を省略可能となっています。</p>
<p>一方、<a href="用語/Python" class="internal-link" data-href="用語/Python">Python</a>ではコードブロックをインデントの深さによって示す言語仕様になっており、「{ }」で囲む必要がありません。また条件式全体を「( )」で囲むこともありません。インデントの強制により誰が記述してもある程度整形されたソースコードになるため、可読性に優れていると言えます。したがって「エ」が正解です。</p>
