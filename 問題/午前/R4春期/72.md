<p class="ap-question"><strong>問72</strong>　"かんばん方式"を説明したものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　各作業の効率を向上させるために，仕様が統一された部品，半製品を調達する。</div>
<div class="ap-choice-note">仕様統一による部品・半製品の調達は、かんばん方式の「後工程が必要分だけ前工程から引き取る」説明ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　効率よく部品調達を行うために，関連会社から部品を調達する。</div>
<div class="ap-choice-note">関連会社からの調達は供給経路の話であり、中間在庫を減らすための後工程引き取りの説明ではありません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　中間在庫を極力減らすために，生産ラインにおいて，後工程が必要とする部品を自工程で生産できるように，必要な部品だけを前工程から調達する。</div>
<div class="ap-choice-note">正しい。後工程の部品数が一定以下になった時点で必要分を前工程から引き取り、前工程は引き取られた分だけを補充する後工程引き取り方式の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　より品質の高い部品を調達するために，部品の納入指定業者を複数定め，競争入札で部品を調達する。</div>
<div class="ap-choice-note">複数業者への競争入札による調達は、調達の競争原理の話であり、かんばん方式の説明ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>かんばん方式は、工程間の中間在庫の最少化を目的として"かんばん"と呼ばれる生産指示票を使う生産システムで、ジャストインタイム<a href="用語/生産方式" class="internal-link" data-href="用語/生産方式">生産方式</a>を実現するために重要な役割を果たす要素です。前工程がムダに部品を多く造り、後工程に溜めてしまうという、それまでの非効率な生産を改善するために、かんばん方式では後工程の部品数が一定数以下になった時点で後工程が前工程から必要分を引き取り、前工程は引き取られた分だけを補充するという「後工程引き取り方式」で生産を行います。このとき後工程から前工程に渡される生産指示票を"かんばん"と呼ぶことから、このような<a href="用語/生産方式" class="internal-link" data-href="用語/生産方式">生産方式</a>を"かんばん方式"といいます。(現在ではITを利用した"eかんばん"に進化しています)したがって「ウ」が正解です。</p>
</div>
