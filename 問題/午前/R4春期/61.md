<p class="ap-question"><strong>問61</strong>　システム管理基準(平成30年)によれば，ITシステムの運用・利用におけるログ管理に関する記述のうち，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　取得したログは，不正なアクセスから保護し，内容が改ざんされないように保管する。</div>
<div class="ap-choice-note">正しい。取得したログを、認可されていないアクセスから保護し、内容が<a href="用語/改ざん" class="internal-link" data-href="用語/改ざん">改ざん</a>されないように保管することとされています。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　通常の運用範囲を超えたアクセスや違反行為に関するログを除外し，運用の作業ログ，利用部門の活動ログを記録し，保管する。</div>
<div class="ap-choice-note">通常の<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>範囲を超えたアクセスや違反行為を含めて、<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>の作業ログ、<a href="用語/利用部門" class="internal-link" data-href="用語/利用部門">利用部門</a>の活動ログを記録し、保管することとされています。通常の<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>範囲を超えたアクセスや違反行為に関するログを除外してはいけません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　特権的アクセスのログは，あまり重要ではないので，分析対象から除外する。</div>
<div class="ap-choice-note">特権的アクセスのログは、その重要性から、多頻度で厳密に分析するなど特別に厳格な管理をすることとされています。特権的アクセスのログも分析対象とすべきです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　保管したログは，情報セキュリティインシデントが発生した場合にだけ分析し，分析結果に応じて必要な対策を講じる。</div>
<div class="ap-choice-note">保管したログを定期的に分析し、分析結果に応じて必要な対策を講じることとされています。ログの分析には、セキュリティインシデントの原因究明だけでなく予兆や痕跡を速やかに取得する目的もあるため、セキュリティインシデント発生時だけなく定期的に行う必要があります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/システム管理基準" class="internal-link" data-href="用語/システム管理基準">システム管理基準</a>(平成30年)では、ITシステムの<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>・利用における<a href="用語/ログ管理" class="internal-link" data-href="用語/ログ管理">ログ管理</a>について、取得したログを認可されていないアクセスから保護し、内容が<a href="用語/改ざん" class="internal-link" data-href="用語/改ざん">改ざん</a>されないように保管することとされています。「ア」は適切です。</p>
<p>通常の<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>範囲を超えたアクセスや違反行為を含めて、<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>の作業ログ、<a href="用語/利用部門" class="internal-link" data-href="用語/利用部門">利用部門</a>の活動ログを記録し、保管することとされています。通常の<a href="用語/運用" class="internal-link" data-href="用語/運用">運用</a>範囲を超えたアクセスや違反行為に関するログを除外してはいけません。「イ」は適切ではありません。</p>
<p>特権的アクセスのログは、その重要性から、多頻度で厳密に分析するなど特別に厳格な管理をすることとされています。特権的アクセスのログも分析対象とすべきです。「ウ」は適切ではありません。</p>
<p>保管したログを定期的に分析し、分析結果に応じて必要な対策を講じることとされています。ログの分析には、セキュリティインシデントの原因究明だけでなく予兆や痕跡を速やかに取得する目的もあるため、セキュリティインシデント発生時だけなく定期的に行う必要があります。「エ」は適切ではありません。</p>
<p>正解は「ア」です。</p>
