<p class="ap-question"><strong>問37</strong>　サイバーキルチェーンの偵察段階に関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　攻撃対象企業の公開Webサイトの脆弱性を悪用してネットワークに侵入を試みる。</div>
<div class="ap-choice-note">"配送"に関する記述です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　攻撃対象企業の社員に標的型攻撃メールを送ってPCをマルウェアに感染させ，PC内の個人情報を入手する。</div>
<div class="ap-choice-note">"配送"～"目的実行"に関する記述です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　攻撃対象企業の社員のSNS上の経歴，肩書などを足がかりに，関連する組織や人物の情報を洗い出す。</div>
<div class="ap-choice-note">正しい。"偵察"に関する記述です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　サイバーキルチェーンの2番目の段階をいい，攻撃対象に特化したPDFやドキュメントファイルにマルウェアを仕込む。</div>
<div class="ap-choice-note">"武器化"に関する記述です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/サイバーキルチェーン" class="internal-link" data-href="用語/サイバーキルチェーン">サイバーキルチェーン</a>は、<a href="用語/サイバー攻撃" class="internal-link" data-href="用語/サイバー攻撃">サイバー攻撃</a>の手順を攻撃者の視点からいくつかの段階に分けモデル化したものです。攻撃の段階としては、偵察、武器化、配送、エクスプロイト、インストール、遠隔操作、目的の実行という7段階が一般的です。攻撃者の視点から<a href="用語/サイバー攻撃" class="internal-link" data-href="用語/サイバー攻撃">サイバー攻撃</a>を見ることで、各段階における防御策の立案に役立てることができます。1. 偵察：標的に関する情報を収集して調査する。2. 武器化：攻撃のためのマルウェアを用意する。3. 配送（デリバリ）：メールやWebサイトを使って標的にマルウェアを送付／配信する、または標的のシステムに侵入する。4. エクスプロイト（攻撃）：標的にマルウェアを実行させる。5. インストール：標的のPCがマルウェアに感染する。6. 遠隔操作（C&amp;C）：マルウェアとC&amp;Cサーバを通信させて標的のサーバを遠隔操作する。7. 目的の実行：秘密情報を得る、サービス停止、改ざんなど攻撃者の目的を達成する。したがって「ウ」が正解です。</p>
