<p class="ap-question"><strong>問2</strong>　3台の機械A，B，Cが良品を製造する確率は，それぞれ60%，70%，80%である。機械A，B，Cが製品を一つずつ製造したとき，いずれか二つの製品が良品で残り一つが不良品になる確率は何%か。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　22.4</div>
<div class="ap-choice-note">Aが不良品・B・Cが良品の確率(0.224)のみであり、3通りの合計ではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　36.8</div>
<div class="ap-choice-note">設問の3通りのケースを合計した確率(45.2%)とは一致しません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　45.2</div>
<div class="ap-choice-note">正しい。不良品が1台になる3通りの確率を合計すると45.2%になります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　78.8</div>
<div class="ap-choice-note">「二つが良品で一つが不良品」以外の組合せを含めた値ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>各々が独立した事象Aと事象Bがあり、起こる確率がそれぞれP(A)，P(B)であるとき、事象A，Bが同時に起こる確率は次の式で求められます(確率の<a href="用語/乗法定理" class="internal-link" data-href="用語/乗法定理">乗法定理</a>)。<br>P(A∩B)＝P(A)×P(B)</p>
<p>また、事象Aが起こらない確率P(A)は「1－P(A)」で表します。</p>
<p>設問にある二つの製品が良品で残り一つが不良品になるケースには以下の3通りがあります。</p>
<p>[Aが不良品、B・Cが良品の確率]<br>(1－0.6)×0.7×0.8＝0.224</p>
<p>[Bが不良品、A・Cが良品の確率]<br>0.6×(1－0.7)×0.8＝0.144</p>
<p>[Cが不良品、A・Bが良品の確率]<br>0.6×0.7×(1－0.8)＝0.084</p>
<p>それぞれの事象は排反事象であるため、二つの製品が良品で残り一つが不良品になる確率は、上記を合計して、<br>0.224＋0.144＋0.084＝0.452→45.2%</p>
<p>したがって正解は「ウ」です。</p>
</div>
