<p class="ap-question"><strong>問64</strong>　クラウドサービスの利用手順を，"利用計画の策定"，"クラウド事業者の選定"，"クラウド事業者との契約締結"，"クラウド事業者の管理"，"サービスの利用終了"としたときに，"利用計画の策定"において，利用者が実施すべき事項はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　クラウドサービスの利用目的，利用範囲，利用による期待効果を検討し，クラウドサービスに求める要件やクラウド事業者に求めるコントロール水準を定める。</div>
<div class="ap-choice-note">正しい。<a href="用語/クラウドサービス" class="internal-link" data-href="用語/クラウドサービス">クラウドサービス</a>をどのように利用するかを決めているので，"利用計画の策定"で実施すべき事項です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　クラウド事業者がSLAなどを適切に遵守しているかモニタリングし，また，自社で構築しているコントロールの有効性を確認し，改善の必要性を検討する。</div>
<div class="ap-choice-note"><a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>の締結後，クラウド事業者を<a href="用語/モニタリング" class="internal-link" data-href="用語/モニタリング">モニタリング</a>しているので，"クラウド事業者の管理"で実施すべき事項です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　クラウド事業者との間で調整不可となる諸事項については，自社による代替策を用意した上で，クラウド事業者との間でコントロール水準をSLAなどで合意する。</div>
<div class="ap-choice-note"><a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>を締結する段階なので，"クラウド事業者との契約締結"で実施すべき事項です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　複数あるクラウド事業者のサービス内容を比較検討し，自社が求める要件及びコントロール水準が充足できるかどうかを判定する。</div>
<div class="ap-choice-note">複数のクラウド事業者のサービス内容を比較検討しているので，"クラウド事業者の選定"で実施すべき事項です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/クラウドサービス" class="internal-link" data-href="用語/クラウドサービス">クラウドサービス</a>の利用手順は，(1)利用計画の策定，(2)クラウド事業者の選定，(3)クラウド事業者との契約締結，(4)クラウド事業者の管理，(5)サービスの利用終了，の順に進めます。</p>
<p>「ア」は利用目的・利用範囲・期待効果の検討と，サービス・事業者に求める要件や<a href="用語/コントロール" class="internal-link" data-href="用語/コントロール">コントロール</a>水準の定めであり，<a href="用語/クラウドサービス" class="internal-link" data-href="用語/クラウドサービス">クラウドサービス</a>をどのように利用するかを決める"利用計画の策定"の段階で実施すべき事項です。</p>
<p>「イ」は<a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>締結後の<a href="用語/モニタリング" class="internal-link" data-href="用語/モニタリング">モニタリング</a>であり，"クラウド事業者の管理"の段階です。「ウ」は<a href="用語/SLA" class="internal-link" data-href="用語/SLA">SLA</a>などでの合意であり，"クラウド事業者との契約締結"の段階です。「エ」は複数事業者の比較検討であり，"クラウド事業者の選定"の段階です。</p>
<p>正解は「ア」です。</p>
</div>
