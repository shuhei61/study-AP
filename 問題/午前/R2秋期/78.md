<p class="ap-question"><strong>問78</strong>　情報流通プラットフォーム対処法が定める特定電気通信役務提供者が行う送信防止措置に関する記述として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　明らかに不当な権利侵害がなされている場合でも，情報の発信者から事前に承諾を得ていなければ，特定電気通信役務提供者は送信防止措置の結果として生じた損害の賠償責任を負う。</div>
<div class="ap-choice-note">情報の流通によって他人の権利が不当に侵害されていると信じるに足りる相当の理由があるときには、特定電気通信役務提供者（プロバイダ等）は、送信防止措置によって発信者に生じた損害について免責されます。相当の理由の例として、発信者への確認その他の必要な調査により十分な確認を行った場合などがあります。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　権利侵害を防ぐための送信防止措置の結果，情報の発信者に損害が生じた場合でも，一定の条件を満たしていれば，特定電気通信役務提供者は賠償責任を負わない。</div>
<div class="ap-choice-note">正しい。特定電気通信役務提供者は、送信防止措置によって発信者に生じた損害について、当該措置が送信を防止するために必要な限度で行われたものであり、かつ、情報の流通によって他人の権利が不当に侵害されていると信じるに足りる相当の理由があるときなど、一定の要件を満たす場合に免責されます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　情報発信者に対して表現の自由を保障し，通信の秘密を確保するため，特定電気通信役務提供者は，裁判所の決定を受けなければ送信防止措置を実施することができない。</div>
<div class="ap-choice-note">特定電気通信役務提供者は、他人の権利が不当に侵害されていると信じるに足りる相当の理由があるときは自主的に、そうでないときには発信者へ照会した上で送信防止措置を講じることができます。裁判所に対して送信防止措置を命じる仮処分・判決を求めることも可能ですが、裁判所の決定がなければ実施できないわけではありません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　特定電気通信による情報の流通によって権利を侵害された者が，個人情報保護委員会に苦情を申し立て，被害が認定された際に特定電気通信役務提供者に命令される措置である。</div>
<div class="ap-choice-note">特定電気通信による情報の流通によって権利を侵害された者は、<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>の侵害であれば<a href="用語/著作権" class="internal-link" data-href="用語/著作権">著作権</a>関係信頼性確認団体、<a href="用語/商標権" class="internal-link" data-href="用語/商標権">商標権</a>の侵害であれば<a href="用語/商標権" class="internal-link" data-href="用語/商標権">商標権</a>関係信頼性確認団体を経由して送信防止措置を申し立てることができます。<a href="用語/個人情報保護委員会" class="internal-link" data-href="用語/個人情報保護委員会">個人情報保護委員会</a>ではありません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>正解は「イ」です。</p>
<p><a href="用語/情報流通プラットフォーム対処法" class="internal-link" data-href="用語/情報流通プラットフォーム対処法">情報流通プラットフォーム対処法</a>（旧称：プロバイダ責任制限法）は、インターネット上のWebページ、<a href="用語/SNS" class="internal-link" data-href="用語/SNS">SNS</a>、<a href="用語/電子掲示板" class="internal-link" data-href="用語/電子掲示板">電子掲示板</a>のように不特定多数の者が閲覧する通信について、その情報を媒介する事業者に対し、情報流通で権利を侵害された者への対応を義務付ける法律です。対象となる事業者はプロバイダ、ホスティング事業者、<a href="用語/電子掲示板" class="internal-link" data-href="用語/電子掲示板">電子掲示板</a>の運営者、大規模プラットフォーム事業者などです。</p>
<p>通常は明らかにされることのない私人のプライバシー情報（住所、電話番号等）について当事者本人から連絡があった場合で、当該者の本人性が確認できている場合、権利を侵害された者から送信防止措置の申立てを受け、発信者に同意するかどうかを照会した場合に、照会した日から7日が経過しても発信者から同意しない申出がなかったときなど、一定の条件の下で送信防止措置が行われます。</p>
</div>
