<p class="ap-question"><strong>問68</strong>　現在の動向から未来を予測したり，システム分析に使用したりする手法であり，専門的知識や経験を有する複数の人にアンケート調査を行い，その結果を互いに参照した上で調査を繰り返して，集団としての意見を収束させる手法はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　因果関係分析法</div>
<div class="ap-choice-note">複数の対象データ間の因果関係を分析する手法です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　クロスセクション法</div>
<div class="ap-choice-note">時間の経過につれて変動していく現象を、ある一時点で横断的に取ったデータを分析する方法です。<a href="用語/時系列分析" class="internal-link" data-href="用語/時系列分析">時系列分析</a>とは対比的な分析概念です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　時系列回帰分析法</div>
<div class="ap-choice-note"><a href="用語/時系列データ" class="internal-link" data-href="用語/時系列データ">時系列データ</a>を分析することで、それらの関係を表す「回帰モデル」と呼ばれる数式を明らかにし、将来の売り上げ予測などに用いる手法です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　デルファイ法</div>
<div class="ap-choice-note">正しい。<a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>は、複数の専門家から個別に意見の収集を行い、得られた意見の集約をフィードバックするということを繰り返して、最終的に意見の収束をしていく手法です。技術革新や社会変動などに関する未来予測においてよく用いられます。</p>
<p><a href="用語/デルファイ法" class="internal-link" data-href="用語/デルファイ法">デルファイ法</a>は次の手順に従って行われます。</p>
<ul>
<li>複数の専門家を回答者として選定し、個別に質問を行う</li>
<li>質問に対する回答結果をフィードバックし、再度質問を行う</li>
<li>回答結果を統計的に処理し，確率分布とともに回答結果を示す</li>
</ul>
<p>したがって「エ」が正解です。</p>
