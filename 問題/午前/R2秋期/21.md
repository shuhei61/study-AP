<p class="ap-question"><strong>問21</strong>　FPGAなどに実装するデジタル回路を記述して，直接論理合成するために使用されるものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　DDL</div>
<div class="ap-choice-note">これはData Definition Languageの説明です。<a href="用語/SQL" class="internal-link" data-href="用語/SQL">SQL</a>言語のうち<a href="用語/データベース" class="internal-link" data-href="用語/データベース">データベース</a>の構造や構成を定義する命令群である。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　HDL</div>
<div class="ap-choice-note">正しい。<a href="用語/HDL" class="internal-link" data-href="用語/HDL">HDL</a>は、デジタル回路の設計や構成をテキストベースで記述できるハードウェア記述言語である。論理合成とは、<a href="用語/HDL" class="internal-link" data-href="用語/HDL">HDL</a>のソースコードを論理合成ツールに与えてネットリストを生成する工程を指す。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　UML</div>
<div class="ap-choice-note">これはUnified Modeling Languageの説明です。オブジェクト指向開発で用いられる標準表記法群である。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　XML</div>
<div class="ap-choice-note">これはeXtended Markup Languageの説明です。利用者がタグや属性の名前を自由に定義できる<a href="用語/マークアップ言語" class="internal-link" data-href="用語/マークアップ言語">マークアップ言語</a>である。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/HDL" class="internal-link" data-href="用語/HDL">HDL</a>(Hardware Description Language)は、デジタル回路の設計や構成をテキストベースで記述できるハードウェア記述言語の総称である。VHDL、Verilog、SystemCなどの種類がある。</p>
<p>ゲートや<a href="用語/フリップフロップ" class="internal-link" data-href="用語/フリップフロップ">フリップフロップ</a>といった論理素子の接続構造や、クロック信号に応じた動作をテキストで記述できることで、論理<a href="用語/回路設計" class="internal-link" data-href="用語/回路設計">回路設計</a>が抽象化されソフトウェアのように扱うことができる。これにより、回路を実装する前の<a href="用語/シミュレーション" class="internal-link" data-href="用語/シミュレーション">シミュレーション</a>検証が可能になる、モジュール化による<a href="用語/保守性" class="internal-link" data-href="用語/保守性">保守性</a>・<a href="用語/再利用性" class="internal-link" data-href="用語/再利用性">再利用性</a>の向上などのメリットがある。</p>
<p><a href="用語/FPGA" class="internal-link" data-href="用語/FPGA">FPGA</a>や<a href="用語/ASIC" class="internal-link" data-href="用語/ASIC">ASIC</a>などのカスタムハードウェアを構築する際には、<a href="用語/HDL" class="internal-link" data-href="用語/HDL">HDL</a>を使った設計が標準的な手法となっている。</p>
</div>
