<p class="ap-question"><strong>問3</strong>　式A＋B×Cの逆ポーランド表記法による表現として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　＋×CBA</div>
<div class="ap-choice-note">演算子が先頭に来ており、A＋B×C の計算順序（B×C を先に求めてから A と加算）に対応しません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　×＋ABC</div>
<div class="ap-choice-note">A と B の加算を先に行う順序になっており、通常の式では × が ＋ より優先される点と一致しません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　ABC×＋</div>
<div class="ap-choice-note">正しい。B×C を BC× に変換したうえで A との加算を行うと ABC×＋ になります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　CBA＋×</div>
<div class="ap-choice-note">オペランドの並びと演算の順序が A＋B×C の変換結果と一致しません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>通常の式を、<a href="用語/逆ポーランド表記法" class="internal-link" data-href="用語/逆ポーランド表記法">逆ポーランド表記法</a>（後置表記法）で表現するための基本は、「A＋B」を「AB＋」で表すことです。1回変換した部分は1つの項とみなすことに注意して、普通に計算式を解くのと同じ順番で変換を行っていくことで<a href="用語/逆ポーランド表記法" class="internal-link" data-href="用語/逆ポーランド表記法">逆ポーランド表記法</a>の式になります。なお、変換を行う優先順位は通常の計算式と同様に、括弧付き「()」→積商算（×÷）→和差算（＋－）の順番です。</p>
<p>最初に「B×C」の部分を変換します。<br>A＋B×C → A＋BC×</p>
<p>次に「BC×」を1つの項とみなして A との＋演算部分を変換します。<br>A＋BC× → ABC×＋</p>
<p>これで式 A＋B×C の<a href="用語/逆ポーランド表記法" class="internal-link" data-href="用語/逆ポーランド表記法">逆ポーランド表記法</a>への変換が完了です。よって正解は「ウ」です。</p>
</div>
