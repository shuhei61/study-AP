<p class="ap-question"><strong>問65</strong>　次に示すグリーン購入基本原則の"製品・サービスのライフサイクルの考慮"に該当する購入例はどれか。<br>〔グリーン購入基本原則〕<br>必要性の考慮<br>購入する前に必要性を十分に考える。<br>製品・サービスのライフサイクルの考慮<br>資源採取から廃棄までの製品ライフサイクルにおける多様な環境負荷を考慮して購入する。<br>事業者の取組の考慮<br>環境負荷の低減に努める事業者から製品・サービスを優先して購入する。<br>環境情報の入手・活用<br>製品・サービスや事業者に関する環境情報を積極的に入手・活用して購入する。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　環境マネジメントシステムを導入し，環境方針，環境対応の責任体制などを定め，環境改善に取り組んでいる企業を，重要な購入先として指定する。</div>
<div class="ap-choice-note">"事業者の取組の考慮"に該当します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　環境や人の健康に悪影響を与えるような物質の使用や排出が削減されており，リユースやリサイクルが可能な製品を選定する。</div>
<div class="ap-choice-note">正しい。"製品・サービスのライフサイクルの考慮"に該当します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　製品の購入に当たっては，遊休資産となっている製品や使用頻度が少ない製品の活用などの代替策を検討した上で判断をする。</div>
<div class="ap-choice-note">"<a href="用語/必要性" class="internal-link" data-href="用語/必要性">必要性</a>の考慮"に該当します。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　複数の製品を環境配慮や環境保全効果などの視点で比較するために，製品紹介のWebページ，カタログなどに示されている環境表示を参考にする。</div>
<div class="ap-choice-note">"環境情報の入手・活用"に該当します。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>グリーン購入基本原則は，グリーン購入を自主的かつ積極的に進めようとする個人や組織の役に立つよう，さまざまな製品やサービスのグリーン購入に共通する基本的な考え方をまとめたものです。グリーン購入とは，購入の<a href="用語/必要性" class="internal-link" data-href="用語/必要性">必要性</a>を十分に考慮し，品質や価格だけでなく環境や社会への影響を考え，環境負荷ができるだけ小さく，かつ社会面に配慮した製品やサービスを，環境負荷の低減や社会的責任の遂行に努める事業者から優先して購入することと言います（グリーン購入基本原則本文より引用）。</p>
<p>各選択肢の記述がそれぞれの原則に対応しているので，問題文に照らして考えれば容易にわかると思います。参考URL: グリーン購入基本原則<br>https://www.gpn.jp/about/rule/</p>
