<p class="ap-question"><strong>問20</strong>　SRAMと比較した場合のDRAMの特徴はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　主にキャッシュメモリとして使用される。</div>
<div class="ap-choice-note"><a href="用語/DRAM" class="internal-link" data-href="用語/DRAM">DRAM</a>は<a href="用語/主記憶" class="internal-link" data-href="用語/主記憶">主記憶</a>として使用される。<a href="用語/キャッシュメモリ" class="internal-link" data-href="用語/キャッシュメモリ">キャッシュメモリ</a>にはアクセス速度がより速い<a href="用語/SRAM" class="internal-link" data-href="用語/SRAM">SRAM</a>が使用される。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　データを保持するためのリフレッシュ又はアクセス動作が不要である。<span class="ap-mistake">DRAMはリフレッシュやアクセス時の再書き込みが必要</span></div>
<div class="ap-choice-note"><a href="用語/DRAM" class="internal-link" data-href="用語/DRAM">DRAM</a>は<a href="用語/コンデンサ" class="internal-link" data-href="用語/コンデンサ">コンデンサ</a>の電荷で記憶するため、時間経過で電荷が失われ、<a href="用語/リフレッシュ" class="internal-link" data-href="用語/リフレッシュ">リフレッシュ</a>やアクセスによる再書き込みが必要である。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　メモリセル構成が単純なので，ビット当たりの単価が安くなる。</div>
<div class="ap-choice-note">正しい。<a href="用語/DRAM" class="internal-link" data-href="用語/DRAM">DRAM</a>は1ビットあたり<a href="用語/トランジスタ" class="internal-link" data-href="用語/トランジスタ">トランジスタ</a>1個と<a href="用語/コンデンサ" class="internal-link" data-href="用語/コンデンサ">コンデンサ</a>1個で構成され、<a href="用語/SRAM" class="internal-link" data-href="用語/SRAM">SRAM</a>より面積が小さいためビット当たりの単価が安い。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　メモリセルにフリップフロップを用いてデータを保存する。</div>
<div class="ap-choice-note">これは<a href="用語/SRAM" class="internal-link" data-href="用語/SRAM">SRAM</a>の説明です。<a href="用語/フリップフロップ" class="internal-link" data-href="用語/フリップフロップ">フリップフロップ</a>を用いるのは<a href="用語/SRAM" class="internal-link" data-href="用語/SRAM">SRAM</a>である。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/DRAM" class="internal-link" data-href="用語/DRAM">DRAM</a>(Dynamic Random Access Memory)は、<a href="用語/コンデンサ" class="internal-link" data-href="用語/コンデンサ">コンデンサ</a>に電荷を蓄えることにより情報を記憶し、電源供給が無くなると記憶情報も失われる揮発性メモリである。集積度を上げることが比較的簡単にできるためコンピュータの<a href="用語/主記憶" class="internal-link" data-href="用語/主記憶">主記憶</a>装置として使用されている。</p>
</div>
