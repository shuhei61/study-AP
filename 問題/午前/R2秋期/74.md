<p class="ap-question"><strong>問74</strong>　不良品の個数を製品別に集計すると表のようになった。ABC分析を行って，まずA群の製品に対策を講じることにした。A群の製品は何種類か。ここで，A群は70%以上とする。<br><img class="ap-fig" src="../../../assets/問題/午前/R2秋期/74/74.png" alt="問題図" width="462" height="53" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　3</div>
<div class="ap-choice-note">上位3製品の累計個数では、A群のしきい値（全体の70%以上）に達しません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　4</div>
<div class="ap-choice-note">上位4製品の累計個数では、A群のしきい値に達しません。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　5</div>
<div class="ap-choice-note">正しい。個数の多い順に累計し、70%以上となる製品は5種類です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　6</div>
<div class="ap-choice-note">6製品まで含めると、A群の範囲（70%以上）を超えてしまいます。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ABC分析" class="internal-link" data-href="用語/ABC分析">ABC分析</a>は、重要度や優先度の高い要素・項目を明らかにするために行われる分析手法です。<a href="用語/パレート図" class="internal-link" data-href="用語/パレート図">パレート図</a>を使って分析する要素・項目群を大きい順に並べ、上位70%を占める要素群をA、70%～90%の要素群をB、それ以外の要素群をCとしてグルーピングすることで重点的に管理すべきグループがどれであるかを明らかにできます。</p>
<p>表の個数合計は875個なので、Aグループ(70%以上)のしきい値となる個数は、875×0.7＝612.5。小数点を切り上げると613個以上になります。表は左から項目の大きい順に整列済みですので、613個以上となるまで足し合わせていけばよいだけです。</p>
<p>182＋136＝318<br>182＋136＋120＝438<br>182＋136＋120＋98＝536<br>182＋136＋120＋98＋91＝627</p>
<p>項目数の大きい製品から5つ足し合わせたところでグループの個数が613個(全体の70％)以上となるので、A群に含まれる製品は5種類です。正解は「ウ」です。</p>
