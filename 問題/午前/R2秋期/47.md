<p class="ap-question"><strong>問47</strong>　次の流れ図において，①→②→③→⑤→②→③→④→②→⑥の順に実行させるために，①においてmとnに与えるべき初期値aとbの関係はどれか。ここで，a，bはともに正の整数とする。<br><img class="ap-fig" src="../../../assets/問題/午前/R2秋期/47/47.png" alt="問題図" width="406" height="284" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　a＝2b</div>
<div class="ap-choice-note">a:b＝2:1 となり、初期の m:n＝2:1 では m＜n から始まらない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　2a＝b</div>
<div class="ap-choice-note">a:b＝1:2 となり、初期の m:n＝1:2 では必要な 2:3 にならない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　2a＝3b</div>
<div class="ap-choice-note">a:b＝3:2 となり、初期の m:n＝3:2 では必要な 2:3 にならない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　3a＝2b</div>
<div class="ap-choice-note">正しい。初期値 m:n＝a:b＝2:3 のとき、指定どおりに⑤→④を経て m＝n となる。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>指定の順序通りに進むと次のように処理されていきます。</p>
<p>m ← a，n ← b<br>m＜n で⑤の処理に進む<br>n ← n－m<br>m＞n で④の処理に進む<br>m ← m－n<br>m＝nとなり処理終了</p>
<p>⑤でnを1回更新、④でmを1回更新して、⑥の段階でm＝nとなります。</p>
<p>まず直前の④の処理について考えます。「m ← m－n」という処理をしているので、"更新前のm"の値は、"更新後のm"と"n"の和です。④の処理後は m＝n になっていますから、"更新前のm"の値は 2n で表すことができます。</p>
<p>次に⑤の処理について考えます。「n ← n－m」という処理をしているので、"更新前のn"の値は、"更新後のn"と"m"の和です。m＝2n ですから、"更新前のn"の値は 3n で表すことができます。</p>
<p>これより前にはm及びnを更新する処理はないので、処理開始時に m：n＝2：3 であれば最終的に m＝n となることがわかります。mはa、nはbが<a href="用語/代入" class="internal-link" data-href="用語/代入">代入</a>される変数なので初期値aとbの関係は a：b＝2：3、これを式に変換した「3a＝2b」が正解です。</p>
<p>【別解】<br>① m←a、n←b<br>②→③→⑤ m&lt;n(⇔a&lt;b)より n←(n-m)=b-a<br>この時点で m=a、n=b-a<br>②→③→④ m&gt;n(⇔a&gt;b-a)より m←m-n=a-(b-a)=2a-b<br>この時点で m=2a-b、n=b-a<br>②→⑥ m=n⇔2a-b=b-a⇔3a=2b</p>
