<p class="ap-question"><strong>問51</strong>　プロジェクトの期間を短縮する方法のうち，クラッシングに該当するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　作業の前後関係を組み直し，実施する順番を変える。</div>
<div class="ap-choice-note">これは<a href="用語/ファストトラッキング" class="internal-link" data-href="用語/ファストトラッキング">ファストトラッキング</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　作業を分析し，同時に実施できる部分を複数の作業に分割し，並行して実施する。</div>
<div class="ap-choice-note">これは<a href="用語/ファストトラッキング" class="internal-link" data-href="用語/ファストトラッキング">ファストトラッキング</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　先行作業の一部の成果物が完成した時点で，後続作業を開始する。</div>
<div class="ap-choice-note">これは<a href="用語/ファストトラッキング" class="internal-link" data-href="用語/ファストトラッキング">ファストトラッキング</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　プロジェクトの外部から要員を調達し，クリティカルパス上の作業に投入する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/クラッシング" class="internal-link" data-href="用語/クラッシング">クラッシング</a></div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>プロジェクト期間を短縮する方法は、<a href="用語/クラッシング" class="internal-link" data-href="用語/クラッシング">クラッシング</a>と<a href="用語/ファストトラッキング" class="internal-link" data-href="用語/ファストトラッキング">ファストトラッキング</a>に大別されます。</p>
<p><a href="用語/クラッシング" class="internal-link" data-href="用語/クラッシング">クラッシング</a>は、アクティビティに追加資源を投入し、所要期間を短縮することでスケジュールの短縮を図る方法です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/H30秋期/51/51_1.png" alt="解説図 1" width="430" height="234" ></p>
<p><a href="用語/ファストトラッキング" class="internal-link" data-href="用語/ファストトラッキング">ファストトラッキング</a>は、開始当初の計画では直列に並んでいた作業を同時並行的に行ったり、作業の前後を入れ替えたりすることで期間短縮を図る方法です。</p><p><img class="ap-fig" src="../../../assets/問題/午前/H30秋期/51/51_2.png" alt="解説図 2" width="430" height="260" ></p>
<p>したがって「エ」が適切です。</p>
