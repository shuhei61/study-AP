<p class="ap-question"><strong>問59</strong>　システム監査において，ペネトレーションテストが最も適合するチェックポイントはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　オフィスへの入退室に，不正防止及び機密保護の物理的な対策が講じられているか。</div>
<div class="ap-choice-note">物理的な入退室対策の確認であり、ネットワークへの<a href="用語/侵入" class="internal-link" data-href="用語/侵入">侵入</a>を試みる<a href="用語/ペネトレーションテスト" class="internal-link" data-href="用語/ペネトレーションテスト">ペネトレーションテスト</a>の対象ではない</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　データ入力が漏れなく，重複なく正確に行われているか。</div>
<div class="ap-choice-note">データ入力の漏れ・重複・誤りの確認であり、<a href="用語/脆弱性" class="internal-link" data-href="用語/脆弱性">脆弱性</a>への<a href="用語/侵入" class="internal-link" data-href="用語/侵入">侵入</a>試行による検査ではない</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ネットワークの負荷状況の推移が記録，分析されているか。</div>
<div class="ap-choice-note">ネットワーク負荷の記録・分析の確認であり、<a href="用語/アクセス制御" class="internal-link" data-href="用語/アクセス制御">アクセス制御</a>が機能しているかの検証ではない</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　ネットワークへのアクセスコントロールが有効に機能しているか。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/ペネトレーションテスト" class="internal-link" data-href="用語/ペネトレーションテスト">ペネトレーションテスト</a></div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ペネトレーションテスト" class="internal-link" data-href="用語/ペネトレーションテスト">ペネトレーションテスト</a>(<a href="用語/侵入" class="internal-link" data-href="用語/侵入">侵入</a>テスト)は、ネットワークに接続されているシステムに対して、実際に様々な方法で<a href="用語/侵入" class="internal-link" data-href="用語/侵入">侵入</a>を試みることで<a href="用語/脆弱性" class="internal-link" data-href="用語/脆弱性">脆弱性</a>の有無を検査するテストです。</p>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>で<a href="用語/機密性" class="internal-link" data-href="用語/機密性">機密性</a>を検証する際に<a href="用語/ペネトレーションテスト" class="internal-link" data-href="用語/ペネトレーションテスト">ペネトレーションテスト</a>を実施することで、ネットワークに適切な<a href="用語/アクセス制御" class="internal-link" data-href="用語/アクセス制御">アクセス制御</a>が施され、権限をもたないものがアクセスできないようになっているかどうかを確認できます。</p>
<p>したがって「エ」が適切です。</p>
