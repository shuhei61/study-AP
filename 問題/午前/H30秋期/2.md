<p class="ap-question"><strong>問2</strong>　コンピュータによる伝票処理システムがある。このシステムは，伝票データをためる待ち行列をもち，M/M/1の待ち行列モデルが適用できるものとする。平均待ち時間がT秒以上となるのは，処理装置の利用率が少なくとも何%以上となったときか。ここで，伝票データをためる待ち行列の特徴は次のとおりである。<br>・伝票データは，ポアソン分布に従って到着する。<br>・伝票データをためる数に制限はない。<br>・1件の伝票データの処理時間は，平均T秒の指数分布に従う。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　33</div>
<div class="ap-choice-note">利用率0.33では利用率÷(1－利用率)＜1となり、平均待ち時間は平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a>より短い。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　50</div>
<div class="ap-choice-note">正しい。利用率0.5で利用率÷(1－利用率)＝1となり、平均待ち時間が平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a>T秒以上となる閾値である。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　67</div>
<div class="ap-choice-note">0.67ではすでに1を超えているが、平均待ち時間がT秒以上となる最小の利用率ではない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　80</div>
<div class="ap-choice-note">0.8ではすでに1を超えているが、平均待ち時間がT秒以上となる最小の利用率ではない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>M/M/1の詳しい説明はここでは省きますが、<a href="用語/待ち行列モデル" class="internal-link" data-href="用語/待ち行列モデル">待ち行列モデル</a>における平均待ち時間とは、サービス（処理）の要求が発生してから実際にそのサービス（処理）が開始されるまでの時間のことです。<br><br>平均待ち時間を求める公式は次のとおりです。<br><br>　平均待ち時間＝<span class="frac"><span>利用率</span><span>1－利用率</span></span>×平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a><br><br>設問には「1件の伝票データの処理時間は，平均T秒の<a href="用語/指数分布" class="internal-link" data-href="用語/指数分布">指数分布</a>に従う」とあるため、上記式の平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a>はT秒が入ります。平均待ち時間がT秒以上となる利用率を求めたいので、以下の式で解がT秒以上となる利用率を求めればよいわけです。<br><br>　<span class="frac"><span>利用率</span><span>1－利用率</span></span>×T<br><br>式を確認すると、平均<a href="用語/サービス時間" class="internal-link" data-href="用語/サービス時間">サービス時間</a>はT秒以上になるのは、<span class="frac"><span>利用率</span><span>1－利用率</span></span>の部分が1以上であるときです。この関係式を解くことで、求めるべき利用率ρ（ロー）を得られます。<br><br>　<span class="frac"><span>ρ</span><span>1－ρ</span></span>≧1<br>　ρ≧1－ρ<br>　2ρ≧1<br>　ρ≧0.5 ⇒ 50%<br><br>以上より、平均待ち時間がT秒以上となるのは、利用率が50%以上となったときとわかります。したがって「イ」が正解です。</p>
