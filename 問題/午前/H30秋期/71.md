<p class="ap-question"><strong>問71</strong>　IoTがもたらす効果を"監視"，"制御"，"最適化"，"自律化"の4段階に分類すると，IoTによって工場の機械の監視や制御などを行っているシステムにおいて，"自律化"の段階に達している例はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　機械に対して，保守員が遠隔地の保守センターからインターネットを経由して，機器の電源のオン・オフなどの操作命令を送信する。</div>
<div class="ap-choice-note">制御の段階です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　機械の温度や振動データをセンサーで集めて，インターネットを経由してクラウドシステム上のサーバに蓄積する。</div>
<div class="ap-choice-note">監視の段階です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　クラウドサービスを介して，機械同土が互いの状態を常時監視・分析し，人手を介すことなく目標に合わせた協調動作を自動で行う。</div>
<div class="ap-choice-note">正しい。自律化の段階です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　クラウドシステム上に常時収集されている機械の稼働情報を基に，機械の故障検知時に，保守員が故障部位を分析して特定する。</div>
<div class="ap-choice-note">最適化の段階です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>アメリカの経営学者であるマイケル・E・ポーター氏らは、<a href="用語/IoTがもたらす効果" class="internal-link" data-href="用語/IoTがもたらす効果">IoTがもたらす効果</a>が4段階に進化すると述べています。</p>
<p><strong>監視</strong> … 離れた場所にあるモノの状態を遠隔からモニタリングできる状態</p>
<p><strong>制御</strong> … 通信を介した外部からの指示でモノを操作できる、または、一定の状態を観測したときに決められた動作を自動で行うようになっている状態</p>
<p><strong>最適化</strong> … 蓄積されたデータと観測したデータをもとに、人間が与えた基準に基づく望ましい状態を自ら見つけ、モノの様態を最適に保つことのできる状態</p>
<p><strong>自律化</strong> … 目標値などの最低限の指示のみ与えれば、人間による細かな条件や基準がなくても、自律的に（あたかも人間のように）最適な状態を判断し、モノ同士が協調して動作できる状態</p>
<p>これを踏まえると各事例の段階は次のようになります。</p>
<p>アは制御の段階、イは監視の段階、ウは自律化の段階、エは最適化の段階です。したがって「ウ」が正解です。</p>
</div>
