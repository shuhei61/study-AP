<p class="ap-question"><strong>問70</strong>　MPEG4などに存在するパテントプールの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　国際機関及び標準化団体による公的な標準ではなく，市場の実勢によって事実上の標準とみなされるようになった規格及び製品</div>
<div class="ap-choice-note">これは<a href="用語/デファクトスタンダード" class="internal-link" data-href="用語/デファクトスタンダード">デファクトスタンダード</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　著作権表示を保持することによって，ソフトウェアの使用，複製，改変，及び再頒布が認められる仕組み</div>
<div class="ap-choice-note">これはオープンソースソフトウェアの説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　特許料が無償でライセンスされている技術</div>
<div class="ap-choice-note">これはオープンライセンスの説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　複数の企業が自社の特許権をもち寄り，特許権を一括して管理する仕組み</div>
<div class="ap-choice-note">正しい。<a href="用語/パテントプール" class="internal-link" data-href="用語/パテントプール">パテントプール</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/パテントプール" class="internal-link" data-href="用語/パテントプール">パテントプール</a>(Patent Pool)とは、標準規格等に関して、複数の特許権者が保有する特許を持ち寄り、一括で効率よくライセンスする仕組みです。特許権者が各々管理する場合と比較して、①複数の特許についてまとめてライセンス交渉できる、②<a href="用語/パテントプール" class="internal-link" data-href="用語/パテントプール">パテントプール</a>で一定の条件が定められるため市場価格の参考になり、③<a href="用語/パテントプール" class="internal-link" data-href="用語/パテントプール">パテントプール</a>で定められた条件は市場価格に影響するなどの強みがあります。</p>
<p><a href="用語/パテントプール" class="internal-link" data-href="用語/パテントプール">パテントプール</a>の例としては、MPEG2、MPEG4、CODEX、WCDMAなどの規格があります。</p>
</div>
