<p class="ap-question"><strong>問62</strong>　業務プロセスを可視化する手法としてUMLを採用した場合の活用シーンはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　対象をエンティティとその属性及びエンティティ間の関連で捉え，データ中心アプローチの表現によって図に示す。</div>
<div class="ap-choice-note">これは<a href="用語/E-R図" class="internal-link" data-href="用語/E-R図">E-R図</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　データの流れによってプロセスを表現するために，データ送出し，データ受取り，データ格納域，データに施す処理を，データの流れを示す矢印でつないで表現する。</div>
<div class="ap-choice-note">これは<a href="用語/DFD" class="internal-link" data-href="用語/DFD">DFD</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　複数の観点でプロセスを表現するために，目的に応じたモデル図法を使用し，オブジェクトモデリングのために標準化された記述ルールで表現する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/UML" class="internal-link" data-href="用語/UML">UML</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　プロセスの機能を網羅的に表現するために，一つの要件に対し発生する事象を条件分岐の形式で記述する。</div>
<div class="ap-choice-note">これはBPDの説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/UML" class="internal-link" data-href="用語/UML">UML</a>(Unified Modeling Language)は、オブジェクト指向開発で用いられる標準表記法群で、オブジェクト間の関連や動作・構成を表現する13種の図表から構成されています。UML2.0の13の図法は表現する対象によって構造図，振る舞い図に分類することができます。</p>
<p>構造図：システムの静的な構造をモデルで表現する。<a href="用語/クラス図" class="internal-link" data-href="用語/クラス図">クラス図</a>，<a href="用語/コンポジット構造図" class="internal-link" data-href="用語/コンポジット構造図">コンポジット構造図</a>，<a href="用語/コンポーネント図" class="internal-link" data-href="用語/コンポーネント図">コンポーネント図</a>，<a href="用語/配置図" class="internal-link" data-href="用語/配置図">配置図</a>，<a href="用語/オブジェクト図" class="internal-link" data-href="用語/オブジェクト図">オブジェクト図</a>，<a href="用語/パッケージ図" class="internal-link" data-href="用語/パッケージ図">パッケージ図</a></p>
<p>振る舞い図：システムの振る舞いをモデルで表現する。<a href="用語/アクティビティ図" class="internal-link" data-href="用語/アクティビティ図">アクティビティ図</a>，<a href="用語/ユースケース図" class="internal-link" data-href="用語/ユースケース図">ユースケース図</a>，<a href="用語/シーケンス図" class="internal-link" data-href="用語/シーケンス図">シーケンス図</a>，インタラクション概念図，コラボレーション図，<a href="用語/タイミング図" class="internal-link" data-href="用語/タイミング図">タイミング図</a>，<a href="用語/ステートマシン図" class="internal-link" data-href="用語/ステートマシン図">ステートマシン図</a></p>
