<p class="ap-question"><strong>問53</strong>　あるシステムの開発工数を見積もると120人月であった。このシステムの開発を12か月で終えるように表に示す計画を立てる。プログラム作成工程には，何名の要員を確保しておく必要があるか。ここで，工程内での要員の増減はないものとする。<br><img class="ap-fig" src="../../../assets/問題/午前/H30秋期/53/53.png" alt="問題図" width="296" height="98" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　7</div>
<div class="ap-choice-note">工数比や期間比の掛け違いで得られる値</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　8</div>
<div class="ap-choice-note">工数比や期間比の掛け違いで得られる値</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　10</div>
<div class="ap-choice-note">工数比や期間比の掛け違いで得られる値</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　18</div>
<div class="ap-choice-note">正しい。プログラム作成工程の工数54人月を期間3か月で割ると18人</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>全体の工数が120人月ですので、プログラム作成工程の工数は以下のように計算できます。</p>
<p>120人月×0.45＝54人月</p>
<p>また、開発期間全体が12カ月ですので、プログラム作成工程の所定期間は、</p>
<p>12か月×0.25＝3か月</p>
<p>54人月の作業を3カ月で完了させなくてならないため、必要な<a href="用語/要員" class="internal-link" data-href="用語/要員">要員</a>数は、</p>
<p>54人月÷3カ月＝18人</p>
<p>したがって「エ」が正解です。</p>
