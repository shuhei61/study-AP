<p class="ap-question"><strong>問4</strong>　次に示す記述は，BNFで表現されたあるプログラム言語の構文の一部である。&lt;パラメータ指定&gt;として，適切なものはどれか。<br>&lt;パラメータ指定&gt;::＝&lt;パラメータ&gt;｜(&lt;パラメータ指定&gt;，&lt;パラメータ&gt;)<br>&lt;パラメータ&gt;::＝&lt;英字&gt;｜&lt;パラメータ&gt;&lt;英字&gt;<br>&lt;英字&gt;::＝a｜b｜c｜d｜e｜f｜g｜h｜i</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　((abc，def)，ghi)</div>
<div class="ap-choice-note">正しい。展開すると最終的に&lt;パラメータ指定&gt;に帰結する。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　((abc，def))</div>
<div class="ap-choice-note">外側の括弧を外すことができず、&lt;パラメータ指定&gt;にならない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　(abc，(def))</div>
<div class="ap-choice-note">これ以上変形できず、&lt;パラメータ指定&gt;にならない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　(abc)</div>
<div class="ap-choice-note">括弧を外すことができず、&lt;パラメータ指定&gt;にならない。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/BNF" class="internal-link" data-href="用語/BNF">BNF</a>で定義された構文について、各肢を展開していき&lt;パラメータ指定&gt;に帰結するか否かを調べていきます。※英字の並びは&lt;パラメータ&gt;で表されます。また、上記とは逆に&lt;パラメータ指定&gt;を再帰的に展開していき、答えを導くこともできます。</p>
<p>&lt;パラメータ指定&gt;<br>→　(&lt;パラメータ指定&gt;，&lt;パラメータ&gt;)<br>→　((&lt;パラメータ指定&gt;，&lt;パラメータ&gt;)，&lt;パラメータ&gt;)<br>→　((&lt;パラメータ&gt;，&lt;パラメータ&gt;)，&lt;パラメータ&gt;)</p>
<p>括弧の位置と英字列の数から考えて適切な構文となっているのは「ア」とわかります。</p>
<p>アは ((abc，def)，ghi) で、((&lt;パラメータ&gt;，&lt;パラメータ&gt;)，&lt;パラメータ&gt;) → ((&lt;パラメータ指定&gt;，&lt;パラメータ&gt;)，&lt;パラメータ&gt;) → (&lt;パラメータ指定&gt;，&lt;パラメータ&gt;) → &lt;パラメータ指定&gt; と帰結する。</p>
<p>イは ((abc，def)) で外側の括弧を外せず、ウは (abc，(def)) でこれ以上変形できず、エは (abc) で括弧を外せないため、いずれも不適切である。</p>
