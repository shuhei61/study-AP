<p class="ap-question"><strong>問41</strong>　クロスサイトスクリプティング対策に該当するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　WebサーバでSNMPエージェントを常時稼働させることによって，攻撃を検知する。</div>
<div class="ap-choice-note"><a href="用語/不正アクセス対策" class="internal-link" data-href="用語/不正アクセス対策">不正アクセス対策</a>に該当する。XSSは正常なHTTP(S)パケットとして届けられるため、SNMP監視では検知できない。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　WebサーバのOSにセキュリティパッチを適用する。</div>
<div class="ap-choice-note">OSの<a href="用語/セキュリティホール" class="internal-link" data-href="用語/セキュリティホール">セキュリティホール</a>を突く攻撃への対策。XSSはWebアプリケーションの実装に起因するため、OSのアップデートでは防げない。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　Webページに入力されたデータの出力データが，HTMLタグとして解釈されないように処理する。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/クロスサイトスクリプティング" class="internal-link" data-href="用語/クロスサイトスクリプティング">クロスサイトスクリプティング</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　許容量を超えた大きさのデータをWebページに入力することを禁止する。</div>
<div class="ap-choice-note">これは<a href="用語/バッファオーバーフロー対策" class="internal-link" data-href="用語/バッファオーバーフロー対策">バッファオーバーフロー対策</a>の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/クロスサイトスクリプティング" class="internal-link" data-href="用語/クロスサイトスクリプティング">クロスサイトスクリプティング</a>(XSS)は、動的にWebページを生成するアプリケーションのセキュリティ上の不備を意図的に利用して、悪意のあるスクリプトを混入させることで、攻撃者が仕込んだ操作を実行させたり、別のサイトを横断してユーザーのクッキーや個人情報を盗んだりする攻撃手法です。</p>
<p><a href="用語/クロスサイトスクリプティング" class="internal-link" data-href="用語/クロスサイトスクリプティング">クロスサイトスクリプティング</a>は、攻撃者が送り込んでくる文字列がHTMLドキュメントの一部となるように合成されて、その部分がブラウザによってスクリプトとして解釈される構文となることによって成立します。よって、プログラム中のデータをHTMLとして出力する際に、HTMLで特別な意味をもつ文字(&lt;,&gt;,&quot;,'&amp;など)を実体参照に置き換えて無害化したり、HTMLタグを削除したりすることによりスクリプトとして解釈されないようにすることが重要です。</p>
