<p class="ap-question"><strong>問42</strong>　ブルートフォース攻撃に該当するものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　WebブラウザとWebサーバの間の通信で，認証が成功してセッションが開始されているときに，Cookieなどのセッション情報を盗む。</div>
<div class="ap-choice-note">これは<a href="用語/セッションハイジャック" class="internal-link" data-href="用語/セッションハイジャック">セッションハイジャック</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　コンピュータへのキー入力を全て記録して外部に送信する。</div>
<div class="ap-choice-note">これは<a href="用語/キーロガー" class="internal-link" data-href="用語/キーロガー">キーロガー</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　使用可能な文字のあらゆる組合せをそれぞれパスワードとして，繰り返しログインを試みる。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/総当たり攻撃" class="internal-link" data-href="用語/総当たり攻撃">総当たり攻撃</a></div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　正当な利用者のログインシーケンスを盗聴者が記録してサーバに送信する。</div>
<div class="ap-choice-note">これは<a href="用語/リプレイ攻撃" class="internal-link" data-href="用語/リプレイ攻撃">リプレイ攻撃</a>の説明です</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>ブルートフォース攻撃は、パスワードクラックや暗号鍵の解読に用いられる手法の1つで、特定の文字数および文字種で設定される可能性のあるすべての組合せを試すことで不正ログインを試みる攻撃手法です。<a href="用語/総当たり攻撃" class="internal-link" data-href="用語/総当たり攻撃">総当たり攻撃</a>とも呼ばれます。</p>
<p>パスワード長が短く、使用可能な文字種が少ない場合には、この手法によって<a href="用語/認証" class="internal-link" data-href="用語/認証">認証</a>を不正に突破されてしまう危険性が高くなります。</p>
</div>
