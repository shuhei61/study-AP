<p class="ap-question"><strong>問80</strong>　ユーザーから請負うソフトウェア開発を中小受託事業者に委託する場合，中小受託取引適正化法で禁止されている行為はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　交通費などの経費については金額を明記せず，実費負担とする旨を発注書面に記載する。</div>
<div class="ap-choice-note">発注書面には支払うべき具体的な金額の記載が原則だが、困難なやむを得ない事情がある場合は算定方法の記載で代えることも認められる。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　中小受託事業者に委託する業務内容は決まっているが，ユーザーとの契約代金が未定なので，製造委託等代金の取決めはユーザーとの契約決定後とする。</div>
<div class="ap-choice-note">正しい。委託時点で制作物の内容を確定できない場合の後日取決めは認められるが、業務内容がすでに確定しているのに代金を定めない正当な理由はないため禁止される。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　発注書面を交付する代わりに，中小受託事業者の承諾を得て，必要な事項を記載した電子メールで発注を行う。</div>
<div class="ap-choice-note">発注書面の代わりに、書面記載事項を電磁的方法（電子メールや記録媒体の交付）で提供することが認められる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　ユーザーの事情で委託予定の業務内容の一部が未定なので，その部分及び製造委託等代金は別途取り決める。</div>
<div class="ap-choice-note">必要記載事項のうち内容が定められない正当な理由があるものについては、記載を省略し、内容確定後に書面または電磁的方法で明示してよい。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/中小受託取引適正化法" class="internal-link" data-href="用語/中小受託取引適正化法">中小受託取引適正化法</a>（旧称：下請法）は、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>（発注側）と<a href="用語/中小受託事業者" class="internal-link" data-href="用語/中小受託事業者">中小受託事業者</a>（受注側）の公平な取引関係を確保し、受託側の利益保護と取引の公正化を図るための法律です。一定規模以上の<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>を「優越的地位にある」ものと位置付け、受託取引における<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>の不当な行為を規制することを狙いとしています。</p>
<p>具体的な規制内容として、<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>に対し、発注内容等の明示、書類等の作成・保存、代金の支払期日の設定、遅延利息の支払といった義務を課すとともに、受領拒否、返品、代金減額、支払遅延など11類型の禁止行為を定めています。<a href="用語/委託事業者" class="internal-link" data-href="用語/委託事業者">委託事業者</a>の4つの義務は次のとおりです。</p>
<ul>
<li>発注内容等を明示する義務：発注に当たって、発注内容（給付の内容、代金の額、支払期日、支払方法）等を書面または電子メールなどの電磁的方法により明示する</li>
<li>書類等を作成・保存する義務：取引が完了した場合、給付内容、代金の額など、取引に関する記録を書類または電磁的記録として作成し、2年間保存する</li>
<li>支払期日を定める義務：検査をするかどうかを問わず、発注した物品等を受領した日から起算して60日以内のできる限り短い期間内で支払期日を定める</li>
<li>遅延利息を支払う義務：支払遅延や減額等を行った場合、遅延した日数や減じた額に応じ、遅延利息（年率14.6%）を支払う</li>
</ul>
</div>
