#問題 #マネジメント #サービスマネジメント #サービスの運用

<p class="ap-question"><strong>問57</strong>　ITIL 2011 editionに示されるサービスデスク組織の構造とその特徴のうち，"フォロー・ザ・サン"の説明として，最も適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　サービスデスクを1拠点又は少数の場所に集中することによって，サービス要員を効率的に配置したり，大量のコールに対応したりすることができる。</div>
<div class="ap-choice-note">これは<a href="用語/中央サービスデスク" class="internal-link" data-href="用語/中央サービスデスク">中央サービスデスク</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　サービスデスクを利用者の近くに配置することによって，言語や文化の異なる利用者への対応，専門要員によるVIP対応などができる。</div>
<div class="ap-choice-note">これは<a href="用語/ローカルサービスデスク" class="internal-link" data-href="用語/ローカルサービスデスク">ローカルサービスデスク</a>の説明です</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　サービス要員が複数の地域や部門に分散していても，通信技術を利用することによって，単一のサービスデスクがあるようにサービスを提供することができる。</div>
<div class="ap-choice-note">これは<a href="用語/バーチャルサービスデスク" class="internal-link" data-href="用語/バーチャルサービスデスク">バーチャルサービスデスク</a>の説明です</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　時差がある分散拠点にサービスデスクを配置し，各サービスデスクが連携してサービスを提供することによって，24時間対応のサービスが提供できる。</div>
<div class="ap-choice-note">正しい。詳細：<a href="用語/フォロー・ザ・サン" class="internal-link" data-href="用語/フォロー・ザ・サン">フォロー・ザ・サン</a></div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/サービスデスク" class="internal-link" data-href="用語/サービスデスク">サービスデスク</a>の組織構造には次のような形態があります。</p>
<p><a href="用語/ローカルサービスデスク" class="internal-link" data-href="用語/ローカルサービスデスク">ローカルサービスデスク</a>：ユーザーのローカルサイト内、若しくは地理的に近い場所に設置された<a href="用語/サービスデスク" class="internal-link" data-href="用語/サービスデスク">サービスデスク</a>。担当者の直接派遣が容易であり、ユーザーの問題や改善点を把握しやすい。</p>
<p><a href="用語/中央サービスデスク" class="internal-link" data-href="用語/中央サービスデスク">中央サービスデスク</a>：一箇所の拠点にてすべてのユーザーからの問い合わせに対応する体制の<a href="用語/サービスデスク" class="internal-link" data-href="用語/サービスデスク">サービスデスク</a>。運営コストを低く抑えることができ情報の管理がしやすい。</p>
<p><a href="用語/バーチャルサービスデスク" class="internal-link" data-href="用語/バーチャルサービスデスク">バーチャルサービスデスク</a>：実際には分散しているが連携することで擬似的に一つの組織として機能を提供する<a href="用語/サービスデスク" class="internal-link" data-href="用語/サービスデスク">サービスデスク</a>。</p>
<p><a href="用語/フォロー・ザ・サン" class="internal-link" data-href="用語/フォロー・ザ・サン">フォロー・ザ・サン</a>：2つ以上の異なる(大陸の)拠点に配置され、中央での統括管理によって24時間365日のサービスを提供する<a href="用語/サービスデスク" class="internal-link" data-href="用語/サービスデスク">サービスデスク</a>。</p>
