<p class="ap-question"><strong>問32</strong>　TCP/IPネットワークにおけるARPの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　IPアドレスからMACアドレスを得るプロトコルである。</div>
<div class="ap-choice-note">正しい。<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　IPネットワークにおける誤り制御のためのプロトコルである。</div>
<div class="ap-choice-note">これは<a href="用語/TCP" class="internal-link" data-href="用語/TCP">TCP</a>(Transmission Control Protocol)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　ゲートウェイ間のホップ数によって経路を制御するプロトコルである。</div>
<div class="ap-choice-note">これはRIP(Routing Information Protocol)の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　端末に対して動的にIPアドレスを割り当てるためのプロトコルである。</div>
<div class="ap-choice-note">これは<a href="用語/DHCP" class="internal-link" data-href="用語/DHCP">DHCP</a>(Dynamic Host Configuration Protocol)の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>(Address Resolution Protocol)は、<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>から対応する機器の<a href="用語/MAC" class="internal-link" data-href="用語/MAC">MAC</a>アドレスを取得するプロトコルです。<a href="用語/ARP" class="internal-link" data-href="用語/ARP">ARP</a>が、<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>から<a href="用語/MAC" class="internal-link" data-href="用語/MAC">MAC</a>アドレスを取得する手順は以下のとおりです。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3秋期/32/32.png" alt="解説図" width="404" height="179" ></p>
<p>1. 要求<a href="用語/パケット" class="internal-link" data-href="用語/パケット">パケット</a>に、<a href="用語/MAC" class="internal-link" data-href="用語/MAC">MAC</a>アドレスを得たい<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>と、送信元の<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>・<a href="用語/MAC" class="internal-link" data-href="用語/MAC">MAC</a>アドレスを格納して、同一セグメント内に<a href="用語/ブロードキャスト" class="internal-link" data-href="用語/ブロードキャスト">ブロードキャスト</a>する。<br>2. 要求<a href="用語/パケット" class="internal-link" data-href="用語/パケット">パケット</a>を受け取った各ノードは、解決目的の<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>を確認し、自分の<a href="用語/IPアドレス" class="internal-link" data-href="用語/IPアドレス">IPアドレス</a>であれば、自分の<a href="用語/MAC" class="internal-link" data-href="用語/MAC">MAC</a>アドレスを送信元に伝える。</p>
<p>したがって正解は「ア」です。</p>
