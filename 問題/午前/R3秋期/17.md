<p class="ap-question"><strong>問17</strong>　主記憶へのアクセスを1命令当たり平均2回行い，ページフォールトが発生すると1回当たり40ミリ秒のオーバーヘッドを伴うシステムがある。ページフォールトによる命令実行の遅れを1命令当たり平均0.4マイクロ秒以下にするために許容できるページフォールト発生率は最大幾らか。ここで，ほかのオーバーヘッドは考慮しないものとする。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　5×10<sup>-6</sup></div>
<div class="ap-choice-note">正しい。条件を満たす発生率の上限は5×10<sup>-6</sup>以下となる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　1×10<sup>-5</sup></div>
<div class="ap-choice-note">許容上限5×10<sup>-6</sup>より大きい。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　5×10<sup>-5</sup></div>
<div class="ap-choice-note">許容上限5×10<sup>-6</sup>より大きい。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　1×10<sup>-4</sup></div>
<div class="ap-choice-note">許容上限5×10<sup>-6</sup>より大きい。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>仮に、1命令における2回の<a href="用語/主記憶" class="internal-link" data-href="用語/主記憶">主記憶</a>へのアクセスの両方で<a href="用語/ページフォールト" class="internal-link" data-href="用語/ページフォールト">ページフォールト</a>が発生した（発生率100%だった）とすると、命令実行の遅れは80ミリ秒になります。この時間を0.4マイクロ秒以下にしたいので、次のような式を立てて条件を満たす発生率(n)を計算します。</p>
<p>80ミリ秒×n≦0.4マイクロ秒<br>80×10<sup>-3</sup>×n≦0.4×10<sup>-6</sup><br>n≦0.4×10<sup>-6</sup>÷(80×10<sup>-3</sup>)<br>n≦0.005×10<sup>-3</sup>＝5×10<sup>-6</sup></p>
<p>上記の計算から、<a href="用語/ページフォールト" class="internal-link" data-href="用語/ページフォールト">ページフォールト</a>発生率(n)は5×10<sup>-6</sup>以下にすればよいとわかります。したがって「ア」が正解です。</p>
</div>
