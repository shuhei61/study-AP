<p class="ap-question"><strong>問78</strong>　企業が業務で使用しているコンピュータに，記憶媒体を介してマルウェアを侵入させ，そのコンピュータの記憶内容を消去した者を処罰の対象とする法律はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　刑法</div>
<div class="ap-choice-note">正しい。<a href="用語/電子計算機損壊等業務妨害罪" class="internal-link" data-href="用語/電子計算機損壊等業務妨害罪">電子計算機損壊等業務妨害罪</a>等の対象となります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　製造物責任法</div>
<div class="ap-choice-note">製造物の安全性上の欠陥による被害に関する法律です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　不正アクセス禁止法</div>
<div class="ap-choice-note">不正アクセスとそれを助長する行為を規制する法律です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　情報流通プラットフォーム対処法</div>
<div class="ap-choice-note">情報流通で権利を侵害された者への対応を義務付ける法律です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>電磁的記録を用いて業務で使用するコンピュータやデータを破壊する、または偽の情報や不正な指令を与えてコンピュータを不正に制御することで業務を妨害した場合、<a href="用語/刑法" class="internal-link" data-href="用語/刑法">刑法</a>の<a href="用語/電子計算機損壊等業務妨害罪" class="internal-link" data-href="用語/電子計算機損壊等業務妨害罪">電子計算機損壊等業務妨害罪</a>に該当します（5年以下の懲役または100万円以下の罰金）。マルウェアを実行しているため、<a href="用語/不正指令電磁的記録に関する罪" class="internal-link" data-href="用語/不正指令電磁的記録に関する罪">不正指令電磁的記録に関する罪</a>（通称：ウイルス作成罪）にも該当することになります。したがって正解は「ア」の<a href="用語/刑法" class="internal-link" data-href="用語/刑法">刑法</a>です。</p>
</div>
