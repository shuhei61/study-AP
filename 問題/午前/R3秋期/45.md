<p class="ap-question"><strong>問45</strong>　化学製品を製造する化学プラントに，情報ネットワークと制御ネットワークがある。この二つのネットワークを接続し，その境界に，制御ネットワークのセキュリティを高めるためにDMZを構築し，制御ネットワーク内の機器のうち，情報ネットワークとの通信が必要なものをこのDMZに移した。DMZに移した機器はどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　温度，流量，圧力などを計測するセンサー</div>
<div class="ap-choice-note">センサーは制御ネットワーク内で測定値を送る機器であり，情報ネットワークとの通信は不要です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　コントローラーからの測定値を監視し，設定値(目標値)を入力する操作端末</div>
<div class="ap-choice-note">操作端末は制御ネットワーク内の監視・設定用であり，情報ネットワークからの通信を前提としていません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　センサーからの測定値が設定値に一致するように調整するコントローラー</div>
<div class="ap-choice-note">コントローラーは制御ネットワーク内で制御を行う機器であり，情報ネットワークとの通信は不要です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">エ　定期的にソフトウェアをアップデートする機器に対して，情報ネットワークから入手したアップデートソフトウェアを提供するパッチ管理サーバ</div>
<div class="ap-choice-note">正しい。アップデートソフトウェアを情報ネットワークから入手するため，情報ネットワークとの通信が必要な機器です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>近年の制御システムのオープン化、ネットワーク化に伴い、制御システムのセキュリティ脅威は増大しています。情報ネットワークで発生したマルウェア被害の影響を制御ネットワークに与えないために、情報ネットワークと制御ネットワークはできるだけ分離しておき、必要最小限の接続に留めることが大切です。DMZに移すのは「情報ネットワークとの通信が必要なもの」です。選択肢のうち「エ」のパッチ管理サーバは、アップデートソフトウェアを情報ネットワークから入手するとありますから、これがDMZに移す機器であると判断できます。</p>
</div>
