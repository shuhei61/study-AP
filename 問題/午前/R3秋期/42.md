<p class="ap-question"><strong>問42</strong>　盗まれたクレジットカードの不正利用を防ぐ仕組みのうち，オンラインショッピングサイトでの不正利用の防止に有効なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　3Dセキュアによって本人確認する。</div>
<div class="ap-choice-note">正しい。3Dセキュアは、オンライン決済時に本人のみが知る情報（パスワードや属性情報等）を入力させることで、利用者本人が取引を行っていることを確認する仕組みです。カードを盗まれたとしても、パスワードは本人しか知らないのでオンラインでの不正利用を防止できます。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　クレジットカード内に保持されたPINとの照合によって本人確認する。</div>
<div class="ap-choice-note">PIN（暗証番号）は、オフライン決済時にIC対応カードを決済端末に挿入し、ICチップからカード情報を読み込んで照合する方法です。ICチップ内の情報を読み出すことが前提となっているため、オンライン決済の認証には使えません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　クレジットカードの有効期限を確認する。</div>
<div class="ap-choice-note">有効期限はカード券面に記載されているため、盗まれたカードの不正利用対策にはなりません。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　セキュリティコードの入力によって券面認証する。</div>
<div class="ap-choice-note">セキュリティコードは、カード券面の「セキュリティコード（数字3～4桁）」を入力し、カードが真正であることを確認する方法です。セキュリティコードはカード券面に記載されているため、盗まれたカードの不正利用対策にはなりません。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>3Dセキュアは、オンライン決済時に本人のみが知る情報（パスワードや属性情報等）を入力させることで、利用者本人が取引を行っていることを確認する仕組みです。カードを盗まれたとしても、パスワードは本人しか知らないのでオンラインでの不正利用を防止できます。</p>
<p>PIN（暗証番号）は、オフライン決済時にIC対応カードを決済端末に挿入し、ICチップからカード情報を読み込んで照合する方法です。ICチップ内の情報を読み出すことが前提となっているため、オンライン決済の認証には使えません。</p>
<p>有効期限はカード券面に記載されているため、盗まれたカードの不正利用対策にはなりません。</p>
<p>セキュリティコードは、カード券面の「セキュリティコード（数字3～4桁）」を入力し、カードが真正であることを確認する方法です。セキュリティコードはカード券面に記載されているため、盗まれたカードの不正利用対策にはなりません。</p>
</div>
