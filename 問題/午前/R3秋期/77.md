<p class="ap-question"><strong>問77</strong>　A社とB社の比較表から分かる，A社の特徴はどれか。<br><img class="ap-fig" src="../../../assets/問題/午前/R3秋期/77/77.png" alt="問題図" width="173" height="143" ></p>
<ul class="ap-choices">
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ア　売上高の増加が大きな利益に結びつきやすい。</div>
<div class="ap-choice-note">正しい。A社は限界利益率が高いので、売上高の増加が大きな利益に結びつきやすいです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　限界利益率が低い。</div>
<div class="ap-choice-note">限界利益率が低いのはB社です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　損益分岐点が低い。</div>
<div class="ap-choice-note"><a href="用語/損益分岐点" class="internal-link" data-href="用語/損益分岐点">損益分岐点</a>売上高が低いのはB社です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　不況時にも，売上高の減少が大きな損失に結びつかず不況抵抗力は強い。</div>
<div class="ap-choice-note">A社は固定費が高いので、売上の減少が利益の減少に結びつきやすいです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>限界利益率は「(売上高－変動費)÷売上高」で計算します。A社は(1,000－500)÷1,000＝0.5、B社は(1,000－800)÷1,000＝0.2です。<a href="用語/損益分岐点" class="internal-link" data-href="用語/損益分岐点">損益分岐点</a>売上高は固定費÷(1－変動費率)で、A社は400÷(1－0.5)＝800、B社は100÷(1－0.8)＝500です。売上高が半分に減少した場合、A社は固定費が高いため売上の減少が利益の減少に結びつきやすいと言えます。</p><p><img class="ap-fig" src="../../../assets/問題/午前/R3秋期/77/77a.png" alt="解説図" width="197" height="151" ></p>
</div>
</div>
