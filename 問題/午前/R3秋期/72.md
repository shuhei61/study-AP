<p class="ap-question"><strong>問72</strong>　個人が，インターネットを介して提示された単発の仕事を受託する働き方や，それによって形成される経済形態を表すものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　APIエコノミー</div>
<div class="ap-choice-note">他社のWebAPIを連携・活用してサービスや価値を創出する経済の仕組みです。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">イ　ギグエコノミー</div>
<div class="ap-choice-note">正しい。インターネットを通じて単発の仕事を請け負う働き方です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ウ　シャドーエコノミー</div>
<div class="ap-choice-note">政府が公表している統計に現れない経済活動のことです。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　トークンエコノミー</div>
<div class="ap-choice-note">代用貨幣のやり取りによって生まれる経済圏のことです。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/ギグエコノミー" class="internal-link" data-href="用語/ギグエコノミー">ギグエコノミー</a>は、企業に雇用されるのではなく、インターネットを通じてマッチングした単発の仕事（ギグ）でお金を稼ぐ働き方や、その働き方によって成り立っている経済圏のことをいいます。このような形態で働く人々は「ギグワーカー」と呼ばれます。出前・宅配サービスのUber Eatsや、民泊仲介サービス、インターネットで依頼者と受注者を結びつける<a href="用語/クラウドソーシング" class="internal-link" data-href="用語/クラウドソーシング">クラウドソーシング</a>などが<a href="用語/ギグエコノミー" class="internal-link" data-href="用語/ギグエコノミー">ギグエコノミー</a>で成長している分野と言えます。労働者にとっては空いた時間で自分のスキルを活用して収入を得られること、発注者にとっては雇用や育成に係るコストを削減できるというメリットがありますが、労働者保護の仕組みに欠ける点や、人材の見定めが難しくトラブルが起きやすいといった課題もあります。</p>
