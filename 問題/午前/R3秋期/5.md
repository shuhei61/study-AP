#問題 #テクノロジ #アルゴリズムとプログラミング #アルゴリズム

<p class="ap-question"><strong>問5</strong>　バブルソートの説明として，適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ある間隔おきに取り出した要素から成る部分列をそれぞれ整列し，更に間隔を詰めて同様の操作を行い，間隔が1になるまでこれを繰り返す。</div>
<div class="ap-choice-note"><a href="用語/シェルソート" class="internal-link" data-href="用語/シェルソート">シェルソート</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　中間的な基準値を決めて，それよりも大きな値を集めた区分と，小さな値を集めた区分に要素を振り分ける。次に，それぞれの区分の中で同様の操作を繰り返す。</div>
<div class="ap-choice-note"><a href="用語/クイックソート" class="internal-link" data-href="用語/クイックソート">クイックソート</a>の説明です。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　隣り合う要素を比較して，大小の順が逆であれば，それらの要素を入れ替えるという操作を繰り返す。</div>
<div class="ap-choice-note">正しい。<a href="用語/バブルソート" class="internal-link" data-href="用語/バブルソート">バブルソート</a>（単純交換法、隣接交換法）は、隣り合う要素同士との比較・交換を繰り返すことで整列を行う<a href="用語/アルゴリズム" class="internal-link" data-href="用語/アルゴリズム">アルゴリズム</a>です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　未整列の部分を順序木にし，そこから最小値を取り出して整列済の部分に移す。この操作を繰り返して，未整列の部分を縮めていく。</div>
<div class="ap-choice-note"><a href="用語/ヒープソート" class="internal-link" data-href="用語/ヒープソート">ヒープソート</a>の説明です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>各選択肢は、いずれかの整列<a href="用語/アルゴリズム" class="internal-link" data-href="用語/アルゴリズム">アルゴリズム</a>の説明になっています。アは<a href="用語/シェルソート" class="internal-link" data-href="用語/シェルソート">シェルソート</a>、イは<a href="用語/クイックソート" class="internal-link" data-href="用語/クイックソート">クイックソート</a>、エは<a href="用語/ヒープソート" class="internal-link" data-href="用語/ヒープソート">ヒープソート</a>の説明です。ウは<a href="用語/バブルソート" class="internal-link" data-href="用語/バブルソート">バブルソート</a>（単純交換法、隣接交換法）の説明であり、隣り合う要素を比較して大小の順が逆なら入れ替える操作を繰り返すことで整列します。したがって正解は「ウ」です。</p>
