<p class="ap-question"><strong>問60</strong>　システム監査基準(平成30年)に基づいて，監査報告書に記載された指摘事項に対応する際に，不適切なものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　監査対象部門が，経営者の指摘事項に対するリスク受容を理由に改善を行わないこととする。</div>
<div class="ap-choice-note">指摘されたリスクを受容し、追加的措置を実施しない場合もあります。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　監査対象部門が，自発的な取組によって指摘事項に対する改善に着手する。</div>
<div class="ap-choice-note"><a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>は、監査対象部門が主体となって実施します。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　システム監査人が，監査対象部門の改善計画を作成する。</div>
<div class="ap-choice-note">正しい（不適切な対応）。<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人として不適切な対応です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　システム監査人が，監査対象部門の改善実施状況を確認する。</div>
<div class="ap-choice-note"><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人は、監査対象部門による改善活動が適切であるかどうかを<a href="用語/フォローアップ" class="internal-link" data-href="用語/フォローアップ">フォローアップ</a>する必要があります。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p><a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>基準(平成30年)には、「<a href="用語/フォローアップ" class="internal-link" data-href="用語/フォローアップ">フォローアップ</a>は、監査対象部門の責任において実施される改善を<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人が事後的に確認するという性質のものであり、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人による<a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>の策定及びその実行への関与は、独立性と客観性を損なうことに留意すべきである」という記載があります。<a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>は、監査対象部門や改善責任部門が主体となって実施するため、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人は、改善の実施そのものに責任をもつことはありません。あくまで、独立性かつ客観的な視点で改善の実施が適切であるかどうかを監視し、指導する立場になります。したがって、<a href="用語/システム監査" class="internal-link" data-href="用語/システム監査">システム監査</a>人自らが<a href="用語/改善計画" class="internal-link" data-href="用語/改善計画">改善計画</a>を作成している「ウ」が不適切な対応です。</p>
</div>
