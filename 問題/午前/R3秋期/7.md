<p class="ap-question"><strong>問7</strong>　静的型付けを行うプログラム言語では，コンパイル時に変数名の誤り，誤った値の代入などが発見できる。Webプログラミングで用いられるスクリプト言語のうち，変数の静的型付けができるものはどれか。</p>
<ul class="ap-choices">
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">ア　ECMAScript</div>
<div class="ap-choice-note">これは<a href="用語/ECMAScript" class="internal-link" data-href="用語/ECMAScript">ECMAScript</a>の説明です。JavaScriptを標準化した言語仕様であり，JavaScriptと同じく動的型付けの<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>です。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">イ　JavaScript</div>
<div class="ap-choice-note">動的型付けの<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>です。変数の型があらかじめ決まっていないため，コンパイル時ではなく実行時にエラーが起こりやすい。</div>
</li>
<li class="ap-choice-item ap-correct">
<div class="ap-choice-text">ウ　TypeScript</div>
<div class="ap-choice-note">正しい。TypeScriptは，JavaScriptに静的型付けなどを追加した上位互換の<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>であり，コンパイル時に型チェックができる。</div>
</li>
<li class="ap-choice-item ap-wrong">
<div class="ap-choice-text">エ　VBScript</div>
<div class="ap-choice-note">Visual Basicを元に開発した<a href="用語/スクリプト言語" class="internal-link" data-href="用語/スクリプト言語">スクリプト言語</a>です。Visual Basicは静的型付けですが，VBScriptは動的型付けの<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>です。</div>
</li>
</ul>
<div class="ap-explanation">
<h4>解説</h4>
<p>TypeScriptは、JavaScriptに省略可能な静的型付け、クラス定義、ジェネリックなどを追加した上位互換の<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>です。マイクロソフトにより開発され、オープンソースとしてメンテナンスされています。JavaScriptは動的型付けの<a href="用語/プログラム言語" class="internal-link" data-href="用語/プログラム言語">プログラム言語</a>であり、変数の型があらかじめ決まっていないため、誤った変数名、変数への誤った代入などがあると実行時エラーが起こります。またデータ型の指定がないので、別の人が書いたソースコードをみるときコードから変数の型を読み取らなくてはなりません。これらの問題は開発に複数の人が関係する大規模開発でより顕著となってきます。TypeScriptは、JavaScriptのこのような問題を解消するため大規模開発のために必要な機能が追加されています。その一つが静的型付けで、ソースコードでデータ型を明示できる機能を追加し、コンパイル時にチェックすることで、大人数がかかわる大規模開発でもエラーを未然に防ぎやすいようになっています。</p>
